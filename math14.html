정리 1

라그랑주 정리

① 정리 설명
유한군 \(G\)의 부분군 \(H\)에 대해 \[|H|\mid |G|\] 이다.
② 증명 핵심 아이디어
  1. 군 \(G\)를 부분군 \(H\)의 왼쪽 잉여류 \(aH\)들로 나눈다.
  2. 서로 다른 잉여류들은 겹치지 않고, 모든 원소는 어떤 잉여류에 속한다.
  3. 각 잉여류 \(aH\)의 원소 개수는 \(H\)의 원소 개수와 같다.
  4. 따라서 \(G\)는 같은 크기의 잉여류 여러 개로 분할된다.
  5. 그러므로 \(|G|=[G:H]|H|\)이고, \(|H|\)는 \(|G|\)를 나눈다.
③ 활용 예제
부분군의 가능한 크기 판정, 군의 구조 분석, 순환군·대칭군 학습의 기본 도구이다.
④ 파이썬 코드
# Z_12의 부분군 <4>의 크기 확인
n = 12
generator = 4

subgroup = set()
x = 0

while x not in subgroup:
    subgroup.add(x)
    x = (x + generator) % n

print("부분군:", sorted(subgroup))
print("|G|:", n)
print("|H|:", len(subgroup))
print("|H|가 |G|를 나누는가?", n % len(subgroup) == 0)
⑤ 코드 상세 설명
  1. `n = 12` : 군 \(Z_{12}\)의 원소 개수이다.
  2. `generator = 4` : 부분군을 생성할 원소이다.
  3. `subgroup = set()` : 중복 없이 부분군 원소를 저장한다.
  4. `x = (x + generator) % n` : \(Z_{12}\)에서 덧셈을 반복한다.
  5. `while x not in subgroup` : 이미 나온 원소가 다시 나오면 순환이 끝난다.
  6. `len(subgroup)` : 부분군의 크기를 구한다.
  7. `n % len(subgroup) == 0` : 라그랑주 정리처럼 부분군의 크기가 군의 크기를 나누는지 확인한다.
정리 2

원소의 위수 정리

① 정리 설명
유한군 \(G\)의 원소 \(a\)의 위수 \(\operatorname{ord}(a)\)는 \(|G|\)를 나눈다.
② 증명 핵심 아이디어
  1. 원소 \(a\)가 생성하는 부분군을 \(\langle a\rangle\)라고 하자.
  2. \(\langle a\rangle=\{e,a,a^2,\dots\}\)이고 그 크기가 \(\operatorname{ord}(a)\)이다.
  3. \(\langle a\rangle\)는 \(G\)의 부분군이다.
  4. 라그랑주 정리에 의해 \(|\langle a\rangle|\mid |G|\)이다.
  5. 따라서 \(\operatorname{ord}(a)\mid |G|\)이다.
③ 활용 예제
원소의 가능한 주기 계산, 순환군 구조 분석, 암호학의 모듈러 거듭제곱 주기 분석에 쓰인다.
④ 파이썬 코드
# Z_15에서 원소 6의 위수 구하기
n = 15
a = 6

x = 0
order = 0

while True:
    order += 1
    x = (x + a) % n
    if x == 0:
        break

print("원소의 위수:", order)
print("군의 크기를 나누는가?", n % order == 0)
⑤ 코드 상세 설명
  1. `n = 15` : 덧셈군 \(Z_{15}\)의 크기이다.
  2. `a = 6` : 위수를 구할 원소이다.
  3. `x = (x + a) % n` : 원소 \(a\)를 반복해서 더한다.
  4. `order += 1` : 몇 번 더했는지 센다.
  5. `if x == 0` : 항등원 0으로 돌아오면 반복을 멈춘다.
  6. `n % order == 0` : 원소의 위수가 군의 크기를 나누는지 확인한다.
정리 3

군 동형 정리

① 정리 설명
두 군 \(G,H\) 사이에 연산을 보존하는 전단사 함수 \(\varphi:G\to H\)가 존재하면 \(G\cong H\)이다.
② 증명 핵심 아이디어
  1. 동형사상 \(\varphi\)는 \(\varphi(ab)=\varphi(a)\varphi(b)\)를 만족한다.
  2. 전단사이므로 \(G\)의 모든 원소와 \(H\)의 모든 원소가 정확히 하나씩 대응된다.
  3. 연산표의 구조가 \(\varphi\)를 통해 그대로 보존된다.
  4. 따라서 두 군은 이름만 다르고 대수적 구조는 같다.
  5. 이를 \(G\cong H\)라고 쓴다.
③ 활용 예제
서로 다르게 보이는 구조가 실제로 같은지 판별하고, 복잡한 군을 더 단순한 군으로 바꾸어 분석할 때 사용된다.
④ 파이썬 코드
# Z_4와 {1, i, -1, -i}의 곱셈 구조 대응
Z4 = [0, 1, 2, 3]
complex_group = [1, 1j, -1, -1j]

def phi(k):
    return complex_group[k]

ok = True
for a in Z4:
    for b in Z4:
        left = phi((a + b) % 4)
        right = phi(a) * phi(b)
        if abs(left - right) > 1e-9:
            ok = False

print("연산 보존:", ok)
⑤ 코드 상세 설명
  1. `Z4` : 덧셈군 \(Z_4\)의 원소이다.
  2. `complex_group` : 네 개의 복소수 \(1,i,-1,-i\)로 이루어진 곱셈군이다.
  3. `phi(k)` : \(k\)를 \(i^k\)에 대응시키는 함수이다.
  4. `phi((a + b) % 4)` : \(Z_4\)에서 먼저 더한 뒤 대응시킨 값이다.
  5. `phi(a) * phi(b)` : 대응시킨 뒤 복소수 곱셈을 한 값이다.
  6. 두 결과가 항상 같으면 연산을 보존하므로 동형의 핵심 조건을 확인한 것이다.
정리 4

제1 동형정리

① 정리 설명
군 준동형 \(\varphi:G\to H\)에 대해 \[G/\ker(\varphi)\cong \operatorname{Im}(\varphi)\] 이다.
② 증명 핵심 아이디어
  1. \(G/\ker(\varphi)\)의 원소는 잉여류 \(g\ker(\varphi)\)이다.
  2. 함수 \(\psi(g\ker(\varphi))=\varphi(g)\)를 정의한다.
  3. 같은 잉여류의 대표원을 골라도 \(\varphi\)값이 같으므로 잘 정의된다.
  4. \(\psi\)는 연산을 보존한다.
  5. \(\psi\)의 상은 \(\operatorname{Im}(\varphi)\)이고 핵은 자명하다.
  6. 따라서 \(G/\ker(\varphi)\)와 \(\operatorname{Im}(\varphi)\)는 동형이다.
③ 활용 예제
몫군을 실제 이미지 구조와 연결해 복잡한 군을 단순화하는 핵심 정리이다.
④ 파이썬 코드
# Z_12 -> Z_3, phi(x)=x mod 3
G = list(range(12))

def phi(x):
    return x % 3

kernel = [x for x in G if phi(x) == 0]
image = sorted(set(phi(x) for x in G))

print("kernel:", kernel)
print("image:", image)
print("|G| / |kernel|:", len(G) // len(kernel))
print("|image|:", len(image))
⑤ 코드 상세 설명
  1. `G = list(range(12))` : \(Z_{12}\)의 원소를 만든다.
  2. `phi(x) = x % 3` : \(Z_{12}\)에서 \(Z_3\)로 가는 준동형 예시이다.
  3. `kernel` : \(arphi(x)=0\)이 되는 원소들의 집합이다.
  4. `image` : 실제로 나오는 함수값들의 집합이다.
  5. `len(G) // len(kernel)` : 몫군의 크기를 계산한다.
  6. `len(image)` : 상의 크기이다.
  7. 두 크기가 같으면 제1 동형정리의 크기 관계를 확인할 수 있다.
정리 5

정규부분군과 몫군 정리

① 정리 설명
부분군 \(N\)이 정규부분군이면, 잉여류들의 집합 \(G/N\)은 자연스럽게 군이 된다.
② 증명 핵심 아이디어
  1. 몫집합 \(G/N\)의 원소는 \(gN\) 꼴의 잉여류이다.
  2. 연산을 \((aN)(bN)=(ab)N\)으로 정의한다.
  3. 이 연산이 대표원 선택과 무관하려면 \(N\)이 정규부분군이어야 한다.
  4. 정규성 \(gN=Ng\) 때문에 연산이 잘 정의된다.
  5. 항등원은 \(N\), 역원은 \(a^{-1}N\)이 된다.
  6. 따라서 \(G/N\)은 군이다.
③ 활용 예제
대칭을 축약하여 새로운 구조를 만들 때 사용되며, 동형정리와 군 분류 이론의 핵심이다.
④ 파이썬 코드
# Z_12 / <4>의 잉여류 만들기
n = 12
N = {0, 4, 8}

cosets = []
seen = set()

for a in range(n):
    coset = frozenset((a + x) % n for x in N)
    if coset not in seen:
        seen.add(coset)
        cosets.append(sorted(coset))

print("몫군의 잉여류:")
for c in cosets:
    print(c)

print("몫군의 원소 개수:", len(cosets))
⑤ 코드 상세 설명
  1. `N = {0, 4, 8}` : \(Z_{12}\)의 부분군이다.
  2. `coset` : \(a+N\) 형태의 잉여류이다.
  3. `(a + x) % n` : \(Z_{12}\)에서 덧셈을 계산한다.
  4. `frozenset` : 집합을 다시 집합의 원소처럼 저장하기 위해 사용한다.
  5. `seen` : 이미 나온 잉여류를 중복 저장하지 않기 위한 집합이다.
  6. `len(cosets)` : 몫군의 원소 개수를 의미한다.
정리 6

케일리 정리

① 정리 설명
모든 군 \(G\)는 어떤 대칭군 \(S_G\)의 부분군과 동형이다.
② 증명 핵심 아이디어
  1. 각 \(g\in G\)에 대해 왼쪽 곱셈 함수 \(L_g(x)=gx\)를 생각한다.
  2. \(L_g\)는 \(G\)에서 \(G\)로 가는 전단사 함수이므로 순열이다.
  3. \(g\mapsto L_g\)라는 함수를 정의한다.
  4. \(L_{gh}=L_g\circ L_h\)이므로 연산을 보존한다.
  5. \(L_g\)가 항등순열이면 \(g=e\)이므로 이 함수는 단사이다.
  6. 따라서 \(G\)는 순열군의 부분군과 동형이다.
③ 활용 예제
추상적인 군을 순열로 구체화하여 계산 가능하게 만드는 강력한 정리이다.
④ 파이썬 코드
# Z_3의 원소를 순열로 표현
G = [0, 1, 2]

def left_action(g):
    return [(g + x) % 3 for x in G]

for g in G:
    print(g, "->", left_action(g))
⑤ 코드 상세 설명
  1. `G = [0, 1, 2]` : 덧셈군 \(Z_3\)의 원소이다.
  2. `left_action(g)` : 원소 g가 모든 x를 \(g+x\)로 보내는 순열을 만든다.
  3. `(g + x) % 3` : \(Z_3\)에서의 덧셈이다.
  4. 출력된 리스트는 각 원소가 어디로 이동하는지를 나타낸다.
  5. 각 g는 하나의 순열에 대응되므로 케일리 정리의 아이디어를 확인할 수 있다.
정리 7

실로우 제1정리

① 정리 설명
유한군 \(G\)의 크기가 \(|G|=p^k m\), \(p\nmid m\)이면, \(G\)에는 크기 \(p^k\)인 부분군이 존재한다.
② 증명 핵심 아이디어
  1. 군 \(G\)가 어떤 집합 위에 작용하는 상황을 만든다.
  2. 궤도-안정자 정리와 조합적 계산을 이용한다.
  3. \(p\)-거듭제곱 크기를 갖는 부분구조가 반드시 존재함을 보인다.
  4. 그 부분구조가 \(G\)의 부분군이 된다.
  5. 이 부분군을 실로우 \(p\)-부분군이라고 한다.
③ 활용 예제
유한군의 구조 분류, 부분군 존재성 판정, 군론 고급 문제의 핵심 도구이다.
④ 파이썬 코드
# |G| = 12일 때 실로우 p-부분군의 가능한 크기
group_order = 12

def highest_power_of_p(n, p):
    power = 1
    while n % p == 0:
        power *= p
        n //= p
    return power

for p in [2, 3]:
    print("p =", p, "실로우 p-부분군 크기:", highest_power_of_p(group_order, p))
⑤ 코드 상세 설명
  1. `group_order = 12` : 군의 크기이다.
  2. `highest_power_of_p(n, p)` : n을 나누는 가장 큰 \(p\)의 거듭제곱을 찾는다.
  3. `while n % p == 0` : p로 계속 나누어떨어지는 동안 반복한다.
  4. `power *= p` : \(p\)-거듭제곱 크기를 키운다.
  5. `n //= p` : n에서 p 인수를 제거한다.
  6. `p = 2`일 때 크기 4, `p = 3`일 때 크기 3의 실로우 부분군이 존재해야 한다.
정리 8

아이디얼과 몫환 정리

① 정리 설명
환 \(R\)의 부분집합 \(I\)가 아이디얼이면, 몫집합 \(R/I\)는 자연스럽게 환이 된다.
② 증명 핵심 아이디어
  1. 몫집합 \(R/I\)의 원소는 \(a+I\) 꼴의 잉여류이다.
  2. 덧셈을 \((a+I)+(b+I)=(a+b)+I\)로 정의한다.
  3. 곱셈을 \((a+I)(b+I)=ab+I\)로 정의한다.
  4. 아이디얼 조건 때문에 대표원을 바꾸어도 결과가 같다.
  5. 따라서 연산이 잘 정의되고, 환의 공리들이 \(R\)로부터 내려온다.
③ 활용 예제
모듈러 산술, 다항식 몫환, 대수기하, 코딩이론, 암호학에서 널리 사용된다.
④ 파이썬 코드
# 정수환 Z에서 5Z로 나눈 몫환 Z/5Z
mod = 5

elements = list(range(mod))

add_table = [[(a + b) % mod for b in elements] for a in elements]
mul_table = [[(a * b) % mod for b in elements] for a in elements]

print("덧셈표")
for row in add_table:
    print(row)

print("곱셈표")
for row in mul_table:
    print(row)
⑤ 코드 상세 설명
  1. `mod = 5` : 몫환 \(\mathbb Z/5\mathbb Z\)를 만든다.
  2. `elements = list(range(mod))` : 대표원 0,1,2,3,4를 사용한다.
  3. `(a + b) % mod` : 몫환에서의 덧셈이다.
  4. `(a * b) % mod` : 몫환에서의 곱셈이다.
  5. `add_table` : 덧셈표를 만든다.
  6. `mul_table` : 곱셈표를 만든다.
  7. 몫환에서는 같은 나머지를 갖는 정수들을 하나의 원소처럼 다룬다.
정리 9

체의 곱셈군 정리

① 정리 설명
체 \(F\)에서 \(0\)이 아닌 원소들의 집합 \(F^\times\)는 곱셈에 대해 군을 이룬다.
② 증명 핵심 아이디어
  1. 체의 정의에 의해 \(0\)이 아닌 두 원소의 곱은 다시 \(0\)이 아니다.
  2. 곱셈 결합법칙은 체의 곱셈에서 이미 성립한다.
  3. 곱셈 항등원 \(1\)은 \(F^\times\)에 속한다.
  4. 체에서는 \(0\)이 아닌 모든 원소가 곱셈 역원을 가진다.
  5. 따라서 \(F^\times\)는 군이다.
③ 활용 예제
유한체 암호학, 타원곡선 암호, 모듈러 역원, 다항식 계산에 사용된다.
④ 파이썬 코드
# Z_7에서 0이 아닌 원소들의 곱셈 역원 찾기
p = 7
elements = range(1, p)

for a in elements:
    for b in elements:
        if (a * b) % p == 1:
            print(a, "의 역원:", b)
            break
⑤ 코드 상세 설명
  1. `p = 7` : 소수 7에 대한 체 \(\mathbb Z_7\)을 생각한다.
  2. `range(1, p)` : 0이 아닌 원소 1부터 6까지 검사한다.
  3. `(a * b) % p == 1` : b가 a의 곱셈 역원인지 확인한다.
  4. `print()` : 각 원소의 역원을 출력한다.
  5. 소수 mod에서는 0이 아닌 모든 원소가 곱셈 역원을 가진다.
정리 10

중국인의 나머지 정리의 환 버전

① 정리 설명
\(\gcd(m,n)=1\)이면 \[\mathbb Z/(mn)\mathbb Z\cong \mathbb Z/m\mathbb Z\times \mathbb Z/n\mathbb Z\] 이다.
② 증명 핵심 아이디어
  1. 함수 \(\varphi:\mathbb Z/(mn)\mathbb Z\to \mathbb Z/m\mathbb Z\times\mathbb Z/n\mathbb Z\)를 \(\varphi(x)=(x\bmod m,x\bmod n)\)으로 정의한다.
  2. 이 함수는 덧셈과 곱셈을 모두 보존한다.
  3. \(\gcd(m,n)=1\)이면 중국인의 나머지 정리에 의해 모든 쌍 \((a,b)\)에 대해 해 \(x\)가 존재한다.
  4. 또한 그 해는 \(mn\)을 법으로 유일하다.
  5. 따라서 \(\varphi\)는 전단사 환 준동형이고 동형이다.
③ 활용 예제
큰 모듈러 계산을 작은 모듈러 계산으로 분해하는 데 사용되며, 암호학과 계산대수에 중요하다.
④ 파이썬 코드
# Z_15와 Z_3 x Z_5의 대응 확인
m = 3
n = 5

pairs = {}

for x in range(m * n):
    pairs[x] = (x % m, x % n)

for x, pair in pairs.items():
    print(x, "->", pair)
⑤ 코드 상세 설명
  1. `m = 3`, `n = 5` : 서로소인 두 법이다.
  2. `range(m * n)` : \(Z_{15}\)의 대표원 0부터 14까지 검사한다.
  3. `(x % m, x % n)` : \(x\)를 두 작은 나머지 정보로 보낸다.
  4. `pairs[x]` : \(Z_{15}\)의 원소와 \(Z_3 imes Z_5\)의 원소를 대응시킨다.
  5. 모든 쌍이 한 번씩 나오면 두 구조가 동형임을 확인할 수 있다.

마무리 핵심 정리

현대대수학의 매력은 계산 대상이 숫자에서 구조로 확장된다는 점이다. 군은 대칭을, 환은 덧셈과 곱셈의 결합 구조를, 체는 나눗셈까지 가능한 세계를 설명한다. 동형정리는 서로 달라 보이는 구조가 사실상 같은 구조임을 보여 주는 현대대수학의 핵심 언어이다.
