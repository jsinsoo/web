1. 극한값 근사

x가 어떤 값에 가까워질 때 함수값이 어디에 가까워지는지 살펴보는 개념입니다.

# 함수 f(x) = (x^2 - 1) / (x - 1)의 x -> 1 극한 근사
a = 1  # x가 가까워질 목표값
h = 0.1  # 처음 접근 간격

for i in range(6):  # h를 점점 작게 하며 6번 반복
    x = a + h  # 1보다 조금 큰 값에서 접근
    y = (x ** 2 - 1) / (x - 1)  # 함수값 계산
    print("h =", h, "x =", x, "f(x) =", y)  # 과정 출력
    h /= 10  # h를 10분의 1로 줄임
한 줄씩 상세 설명
1. a = 1: x가 가까워질 목표값을 정합니다.
2. h = 0.1: 처음에는 1에서 0.1만큼 떨어진 곳에서 시작합니다.
3. for i in range(6): h를 점점 줄이면서 반복합니다.
4. x = a + h: 1보다 조금 큰 x값을 만듭니다.
5. y = ...: 함수값을 계산합니다.
6. h /= 10: x가 1에 더 가까워지도록 h를 줄입니다.

2. 도함수 근사

도함수는 순간변화율이며, 아주 작은 h를 이용해 기울기를 근사할 수 있습니다.

# f(x) = x^2의 x = 3에서 도함수 근사
x = 3  # 미분계수를 구할 위치
h = 0.0001  # 매우 작은 변화량

fx = x ** 2  # f(x)
fxh = (x + h) ** 2  # f(x+h)

derivative = (fxh - fx) / h  # 평균변화율로 순간변화율 근사

print("근사 도함수 =", derivative)
한 줄씩 상세 설명
1. x = 3: 기울기를 구할 위치입니다.
2. h = 0.0001: 매우 작은 변화량입니다.
3. fx = x ** 2: f(3)을 계산합니다.
4. fxh = (x + h) ** 2: f(3+h)를 계산합니다.
5. derivative = ...: 변화량의 비로 도함수를 근사합니다.

3. 정적분 근사

정적분은 함수 그래프 아래 넓이를 의미하며 직사각형 합으로 근사할 수 있습니다.

# f(x) = x^2을 0부터 1까지 적분 근사
a = 0  # 시작점
b = 1  # 끝점
n = 1000  # 직사각형 개수
dx = (b - a) / n  # 직사각형의 폭
total = 0  # 넓이 합

for i in range(n):  # n개의 직사각형을 더함
    x = a + i * dx  # 각 직사각형의 왼쪽 x좌표
    height = x ** 2  # 직사각형의 높이
    total += height * dx  # 넓이 = 높이 × 폭

print("정적분 근사값 =", total)
한 줄씩 상세 설명
1. a, b: 적분 구간의 시작과 끝입니다.
2. n: 직사각형을 몇 개로 나눌지 정합니다.
3. dx: 각 직사각형의 폭입니다.
4. total: 넓이의 합을 저장합니다.
5. height = x ** 2: 함수값을 높이로 사용합니다.
6. total += height * dx: 작은 넓이를 계속 더합니다.

4. 뉴턴 방법

방정식의 근을 반복적으로 더 정확하게 찾는 수치해석 방법입니다.

# x^2 - 2 = 0의 근, 즉 sqrt(2) 근사
x = 1.0  # 처음 추측값

for i in range(10):  # 10번 반복하여 근을 개선
    f = x ** 2 - 2  # 함수값
    df = 2 * x  # 도함수값
    x = x - f / df  # 뉴턴 방법 공식

print("sqrt(2) 근사값 =", x)
한 줄씩 상세 설명
1. x = 1.0: 처음 추측값입니다.
2. f = x ** 2 - 2: 현재 x에서의 함수값입니다.
3. df = 2 * x: 도함수값입니다.
4. x = x - f / df: 접선을 이용해 근에 더 가까이 갑니다.

5. 이분법

부호가 바뀌는 구간을 반으로 나누며 방정식의 근을 찾는 방법입니다.

# x^2 - 2 = 0의 근을 이분법으로 찾기
left = 1  # 왼쪽 끝
right = 2  # 오른쪽 끝

for i in range(20):  # 20번 반복
    mid = (left + right) / 2  # 중간값
    if mid ** 2 - 2 > 0:  # 중간값에서 함수값이 양수이면
        right = mid  # 오른쪽 끝을 중간값으로 이동
    else:
        left = mid  # 왼쪽 끝을 중간값으로 이동

print("근사 근 =", (left + right) / 2)
한 줄씩 상세 설명
1. left, right: 근이 들어 있는 구간입니다.
2. mid: 구간의 중간값입니다.
3. mid ** 2 - 2: 함수값의 부호를 확인합니다.
4. right = mid 또는 left = mid: 근이 있는 구간을 좁힙니다.

6. 벡터 내적

두 벡터의 방향 관계와 크기 관계를 나타내는 연산입니다.

# 두 벡터의 내적
a = [1, 2, 3]  # 첫 번째 벡터
b = [4, 5, 6]  # 두 번째 벡터
dot = 0  # 내적 결과

for i in range(len(a)):  # 각 성분 위치를 반복
    dot += a[i] * b[i]  # 같은 위치 성분끼리 곱해 더함

print("내적 =", dot)
한 줄씩 상세 설명
1. a, b: 두 벡터입니다.
2. dot = 0: 결과를 저장할 변수입니다.
3. for i in range(len(a)): 각 성분을 순서대로 봅니다.
4. dot += a[i] * b[i]: 같은 위치 성분끼리 곱해 더합니다.

7. 벡터 크기

벡터의 길이는 각 성분의 제곱합의 제곱근입니다.

# 벡터의 크기 구하기
v = [3, 4, 12]  # 벡터
total = 0  # 제곱합

for x in v:  # 각 성분을 하나씩 꺼냄
    total += x ** 2  # 성분의 제곱을 더함

length = total ** 0.5  # 제곱근을 구함

print("벡터의 크기 =", length)
한 줄씩 상세 설명
1. v: 벡터입니다.
2. total: 성분 제곱합입니다.
3. for x in v: 성분을 하나씩 꺼냅니다.
4. total += x ** 2: 제곱을 더합니다.
5. length = total ** 0.5: 제곱근으로 길이를 구합니다.

8. 2차원 행렬 덧셈

같은 크기의 두 행렬은 같은 위치의 원소끼리 더할 수 있습니다.

# 2x2 행렬 덧셈
A = [[1, 2], [3, 4]]  # 첫 번째 행렬
B = [[5, 6], [7, 8]]  # 두 번째 행렬
C = []  # 결과 행렬

for i in range(2):  # 행 반복
    row = []  # 한 행을 저장
    for j in range(2):  # 열 반복
        row.append(A[i][j] + B[i][j])  # 같은 위치 원소끼리 더함
    C.append(row)  # 완성된 행 추가

print(C)
한 줄씩 상세 설명
1. A, B: 더할 두 행렬입니다.
2. C: 결과 행렬입니다.
3. i: 행 번호입니다.
4. j: 열 번호입니다.
5. A[i][j] + B[i][j]: 같은 위치 원소를 더합니다.

9. 2차원 행렬 곱셈

행렬 곱셈은 앞 행렬의 행과 뒤 행렬의 열을 곱해 더합니다.

# 2x2 행렬 곱셈
A = [[1, 2], [3, 4]]
B = [[5, 6], [7, 8]]
C = [[0, 0], [0, 0]]

for i in range(2):  # A의 행
    for j in range(2):  # B의 열
        for k in range(2):  # 곱해서 더할 위치
            C[i][j] += A[i][k] * B[k][j]

print(C)
한 줄씩 상세 설명
1. C를 0으로 초기화합니다.
2. i는 결과 행 번호입니다.
3. j는 결과 열 번호입니다.
4. k는 곱해서 더할 중간 인덱스입니다.
5. C[i][j]에 A의 행과 B의 열의 곱을 더합니다.

10. 행렬식

2×2 행렬의 행렬식은 평면 변환의 넓이 배율과 관련됩니다.

# 2x2 행렬식 det = ad - bc
A = [[3, 2], [1, 4]]  # 2x2 행렬

det = A[0][0] * A[1][1] - A[0][1] * A[1][0]  # ad - bc

print("행렬식 =", det)
한 줄씩 상세 설명
1. A는 2x2 행렬입니다.
2. A[0][0]과 A[1][1]을 곱합니다.
3. A[0][1]과 A[1][0]을 곱합니다.
4. 두 값을 빼면 행렬식입니다.

11. 일차회귀 기울기

데이터의 경향을 가장 잘 나타내는 직선의 기울기를 구합니다.

# 간단한 일차회귀 기울기 구하기
x = [1, 2, 3, 4, 5]  # x 데이터
y = [2, 4, 5, 4, 6]  # y 데이터
n = len(x)  # 데이터 개수

mean_x = sum(x) / n  # x 평균
mean_y = sum(y) / n  # y 평균

top = 0  # 분자
bottom = 0  # 분모

for i in range(n):
    top += (x[i] - mean_x) * (y[i] - mean_y)
    bottom += (x[i] - mean_x) ** 2

slope = top / bottom  # 회귀직선의 기울기

print("기울기 =", slope)
한 줄씩 상세 설명
1. x, y: 데이터입니다.
2. mean_x, mean_y: 평균입니다.
3. top: 공분산에 해당하는 값입니다.
4. bottom: x의 분산에 해당하는 값입니다.
5. slope = top / bottom: 회귀직선 기울기입니다.

12. 분산과 표준편차

자료가 평균에서 얼마나 흩어져 있는지 나타냅니다.

# 분산과 표준편차
data = [2, 4, 4, 4, 5, 5, 7, 9]
mean = sum(data) / len(data)  # 평균

total = 0
for x in data:
    total += (x - mean) ** 2  # 평균과의 차이를 제곱해 더함

variance = total / len(data)  # 분산
std = variance ** 0.5  # 표준편차

print("분산 =", variance)
print("표준편차 =", std)
한 줄씩 상세 설명
1. mean: 평균입니다.
2. x - mean: 평균과의 차이입니다.
3. 제곱을 하는 이유는 음수와 양수를 모두 양수 거리로 보기 위해서입니다.
4. variance: 제곱 차이의 평균입니다.
5. std: 분산의 제곱근입니다.

13. 복리 성장

원금이 이자를 포함해 반복적으로 증가하는 구조입니다.

# 복리 계산
money = 1000000  # 원금
rate = 0.05  # 연 이율 5%
years = 10  # 기간

for year in range(1, years + 1):
    money = money * (1 + rate)  # 이자가 붙은 금액
    print(year, "년 후:", int(money), "원")
한 줄씩 상세 설명
1. money: 원금입니다.
2. rate: 이율입니다.
3. years: 기간입니다.
4. money * (1 + rate): 이자가 붙은 다음 금액입니다.
5. 반복할수록 이자가 다시 원금에 포함됩니다.

14. 등비급수 부분합

등비수열의 항을 계속 더한 값을 부분합이라고 합니다.

# 등비급수 부분합
a = 1  # 첫째항
r = 0.5  # 공비
n = 10  # 더할 항의 개수
total = 0

for k in range(n):
    term = a * (r ** k)  # k번째 항
    total += term  # 항을 더함
    print(k + 1, "번째까지 합 =", total)

print("부분합 =", total)
한 줄씩 상세 설명
1. a: 첫째항입니다.
2. r: 공비입니다.
3. n: 더할 항의 개수입니다.
4. term = a * r^k: 각 항입니다.
5. total += term: 항을 누적합니다.

16. 팩토리얼과 조합

조합은 서로 다른 n개 중 r개를 순서 없이 고르는 경우의 수입니다.

# nCr 조합 계산
n = 6
r = 2

top = 1  # 분자
bottom = 1  # 분모

for i in range(n, n - r, -1):
    top *= i  # n부터 r개 곱함

for i in range(1, r + 1):
    bottom *= i  # r! 계산

comb = top // bottom

print("조합 =", comb)
한 줄씩 상세 설명
1. n, r: n개 중 r개를 고릅니다.
2. top: n부터 r개를 곱한 값입니다.
3. bottom: r!입니다.
4. comb = top // bottom: 조합 값을 구합니다.

17. 유클리드 거리

다차원 공간에서 두 점 사이의 직선 거리를 구합니다.

# 3차원 두 점 사이 거리
p = [1, 2, 3]
q = [4, 6, 8]
total = 0

for i in range(len(p)):
    diff = q[i] - p[i]  # 각 좌표의 차
    total += diff ** 2  # 차이의 제곱을 더함

distance = total ** 0.5

print("거리 =", distance)
한 줄씩 상세 설명
1. p, q: 두 점입니다.
2. diff: 각 좌표의 차입니다.
3. total += diff ** 2: 차이 제곱을 누적합니다.
4. distance: 제곱합의 제곱근입니다.

18. 맨해튼 거리

격자 위에서 가로·세로 이동 거리의 합을 구합니다.

# 2차원 맨해튼 거리
p = [1, 2]
q = [5, 7]

distance = 0

for i in range(len(p)):
    diff = q[i] - p[i]
    if diff < 0:
        diff = -diff  # 절댓값 처리
    distance += diff

print("맨해튼 거리 =", distance)
한 줄씩 상세 설명
1. p, q: 두 점입니다.
2. diff: 좌표 차이입니다.
3. diff < 0이면 부호를 바꿉니다.
4. distance += diff: 절댓값 차이를 더합니다.

19. n각형 넓이

좌표평면의 다각형 넓이는 신발끈 공식으로 구할 수 있습니다.

# 신발끈 공식으로 다각형 넓이 구하기
points = [(0, 0), (4, 0), (4, 3), (0, 3)]  # 사각형 좌표
n = len(points)
sum1 = 0
sum2 = 0

for i in range(n):
    x1, y1 = points[i]  # 현재 점
    x2, y2 = points[(i + 1) % n]  # 다음 점, 마지막 다음은 첫 점
    sum1 += x1 * y2
    sum2 += y1 * x2

area = abs(sum1 - sum2) / 2

print("다각형 넓이 =", area)
한 줄씩 상세 설명
1. points: 다각형의 꼭짓점입니다.
2. (i + 1) % n: 마지막 점 다음에 첫 점으로 돌아갑니다.
3. sum1: 앞쪽 곱의 합입니다.
4. sum2: 뒤쪽 곱의 합입니다.
5. area: 두 합의 차이의 절반입니다.

20. 피타고라스 정리 검증

직각삼각형에서 두 직각변의 제곱합은 빗변의 제곱과 같습니다.

# 세 변이 직각삼각형인지 확인
a = 3
b = 4
c = 5

if a ** 2 + b ** 2 == c ** 2:
    print("직각삼각형")
else:
    print("직각삼각형 아님")
한 줄씩 상세 설명
1. a, b: 두 직각변 후보입니다.
2. c: 빗변 후보입니다.
3. a ** 2 + b ** 2: 두 변의 제곱합입니다.
4. c ** 2: 빗변의 제곱입니다.
5. 두 값이 같으면 직각삼각형입니다.

2. OX 개념 퀴즈 10개

1. 도함수는 순간변화율을 나타낸다.
정답: O
