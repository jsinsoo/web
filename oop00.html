6. 은행 계좌 객체

개념 요약
📌 개념 상세 설명 은행 계좌는 주인과 잔액 정보를 가지고 있습니다. 입금하면 돈이 늘고, 출금하면 돈이 줄어듭니다. 조건문을 사용해 잔액 부족도 확인할 수 있습니다.

💻 제시 코드

class BankAccount:
    def __init__(self, owner, money):
        self.owner = owner
        self.money = money

    def deposit(self, amount):
        self.money += amount
        return f"{amount}원을 입금했습니다. 잔액: {self.money}원"

    def withdraw(self, amount):
        if amount > self.money:
            return "잔액이 부족합니다."
        self.money -= amount
        return f"{amount}원을 출금했습니다. 잔액: {self.money}원"

account = BankAccount("지우", 10000)

print(account.deposit(5000))
print(account.withdraw(3000))
print(account.withdraw(20000))
문항 11OX 개념 퀴즈

은행 계좌 객체는 입금과 출금 같은 행동을 메서드로 표현할 수 있다.

7. 도서관 책 대출 객체

개념 요약
📌 개념 상세 설명 책은 대출되었는지 아닌지 상태를 가질 수 있습니다. OOP에서는 이런 상태를 속성으로 저장하고, 대출과 반납 같은 행동을 메서드로 만듭니다.

💻 제시 코드

class Book:
    def __init__(self, title):
        self.title = title
        self.is_borrowed = False

    def borrow(self):
        if self.is_borrowed:
            return f"{self.title}은 이미 대출 중입니다."
        self.is_borrowed = True
        return f"{self.title}을 대출했습니다."

    def return_book(self):
        self.is_borrowed = False
        return f"{self.title}을 반납했습니다."

book = Book("어린 왕자")

print(book.borrow())
print(book.borrow())
print(book.return_book())
문항 13OX 개념 퀴즈

is_borrowed는 책이 대출 중인지 상태를 저장하는 속성이다.

9. 점수 관리 객체

개념 요약
📌 개념 상세 설명 여러 점수를 하나의 객체가 관리할 수 있습니다. 평균과 최고점 계산도 객체의 행동으로 만들 수 있습니다.

💻 제시 코드

class ScoreManager:
    def __init__(self, scores):
        self.scores = scores

    def average(self):
        return sum(self.scores) / len(self.scores)

    def highest(self):
        return max(self.scores)

scores = ScoreManager([90, 80, 100, 70])

print("평균:", scores.average())
print("최고점:", scores.highest())

13. 급식 메뉴 추천 객체

💻 제시 코드

class LunchRecommender:
    def __init__(self, menu):
        self.menu = menu

    def recommend(self, keyword):
        result = []
        for food in self.menu:
            if keyword in food:
                result.append(food)
        return result

lunch = LunchRecommender(["김치볶음밥", "치킨샐러드", "된장국", "김밥"])

14. 운동 기록 관리 객체

개념 요약
📌 개념 상세 설명 운동 앱은 매일 운동한 시간을 저장하고 총 운동 시간을 알려줍니다. 객체 안에 리스트를 두면 기록을 차곡차곡 저장할 수 있습니다.

💻 제시 코드

class ExerciseRecord:
    def __init__(self, name):
        self.name = name
        self.minutes = []

    def add_record(self, minute):
        self.minutes.append(minute)

    def total_time(self):
        return sum(self.minutes)

    def report(self):
        return f"{self.name}의 총 운동 시간은 {self.total_time()}분입니다."

record = ExerciseRecord("수아")

record.add_record(30)
record.add_record(20)
record.add_record(40)

print(record.report())
문항 27OX 개념 퀴즈

개념 요약
📌 개념 상세 설명 선생님은 학생들의 출석 상태를 기록합니다. 딕셔너리를 사용하면 학생 이름과 출석 상태를 짝으로 저장할 수 있습니다.

💻 제시 코드

class Attendance:
    def __init__(self):
        self.students = {}

    def check(self, name, status):
        self.students[name] = status

    def absent_students(self):
        return [name for name, status in self.students.items() if status == "결석"]

attendance = Attendance()

attendance.check("민수", "출석")
attendance.check("지우", "결석")
attendance.check("하린", "출석")
attendance.check("도윤", "결석")

print(attendance.absent_students())

딕셔너리의 items()는 키와 값을 함께 꺼낼 때 사용할 수 있다.
