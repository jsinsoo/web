정리 1

코시-리만 방정식

① 정리 설명
복소함수 \(f(z)=u(x,y)+iv(x,y)\)가 복소미분 가능하면, 적절한 조건에서 \[u_x=v_y,\qquad u_y=-v_x\] 를 만족한다.
② 증명 핵심 아이디어
  1. 복소미분 가능하다는 것은 극한 \(\lim_{h\to0}\frac{f(z+h)-f(z)}{h}\)가 \(h\)의 접근 방향과 무관하게 존재한다는 뜻이다.
  2. 먼저 \(h\)를 실수 방향으로 보내면 미분값은 \(u_x+iv_x\)가 된다.
  3. 다음 \(h\)를 순허수 방향으로 보내면 미분값은 \(v_y-iu_y\)가 된다.
  4. 두 값이 같아야 하므로 실수부와 허수부를 비교한다.
  5. 따라서 \(u_x=v_y\), \(u_y=-v_x\)가 성립한다.
③ 활용 예제
복소미분 가능성 판정, 해석함수 확인, 유체역학의 포텐셜 흐름, 전기장 해석에 사용된다.
④ 파이썬 코드
# f(z)=z^2의 코시-리만 방정식 확인
# z=x+iy, f(z)=z^2=(x^2-y^2)+i(2xy)

x = 2
y = 3

u_x = 2*x
u_y = -2*y
v_x = 2*y
v_y = 2*x

print("u_x =", u_x, "v_y =", v_y)
print("u_y =", u_y, "-v_x =", -v_x)
print("코시-리만 만족:", u_x == v_y and u_y == -v_x)
⑤ 코드 상세 설명
  1. `f(z)=z^2`를 \(z=x+iy\)로 전개하면 실수부는 \(u=x^2-y^2\), 허수부는 \(v=2xy\)이다.
  2. `u_x = 2*x` : \(u\)를 \(x\)로 편미분한 값이다.
  3. `u_y = -2*y` : \(u\)를 \(y\)로 편미분한 값이다.
  4. `v_x = 2*y` : \(v\)를 \(x\)로 편미분한 값이다.
  5. `v_y = 2*x` : \(v\)를 \(y\)로 편미분한 값이다.
  6. `u_x == v_y and u_y == -v_x` : 코시-리만 방정식을 만족하는지 확인한다.
정리 2

코시 적분정리

① 정리 설명
단순연결 영역 \(D\)에서 \(f\)가 해석적이면, \(D\) 안의 닫힌곡선 \(\gamma\)에 대해 \[\int_\gamma f(z)\,dz=0\] 이다.
② 증명 핵심 아이디어
  1. 해석함수는 코시-리만 방정식을 만족한다.
  2. 복소선적분을 실수부와 허수부의 선적분으로 나눈다.
  3. 그린 정리를 적용하면 닫힌곡선 적분이 영역 내부의 편도함수 적분으로 바뀐다.
  4. 코시-리만 방정식 때문에 내부 적분항들이 서로 상쇄된다.
  5. 따라서 닫힌곡선 위의 복소적분은 \(0\)이 된다.
③ 활용 예제
복소적분 계산 단순화, 경로 독립성, 코시 적분공식과 유수정리의 출발점이다.
④ 파이썬 코드
# f(z)=z 를 단위원 위에서 수치 적분
import cmath
import math

N = 10000
total = 0j

for k in range(N):
    t1 = 2*math.pi*k/N
    t2 = 2*math.pi*(k+1)/N
    z1 = cmath.exp(1j*t1)
    z2 = cmath.exp(1j*t2)
    dz = z2 - z1
    total += z1 * dz

print(total)
print("0에 가까운가?", abs(total) < 1e-3)
⑤ 코드 상세 설명
  1. `cmath.exp(1j*t)` : 단위원 위의 점 \(e^{it}\)를 만든다.
  2. `z1`, `z2` : 곡선 위의 이웃한 두 점이다.
  3. `dz = z2 - z1` : 작은 경로 조각이다.
  4. `total += z1 * dz` : \(\int z\,dz\)를 리만합처럼 근사한다.
  5. `abs(total)` : 적분값의 크기를 확인한다.
  6. 코시 적분정리에 의해 해석함수 \(f(z)=z\)의 닫힌곡선 적분은 0이어야 한다.
정리 3

코시 적분공식

① 정리 설명
\(f\)가 단순닫힌곡선 \(\gamma\) 내부와 위에서 해석적이고 \(a\)가 내부점이면 \[f(a)=\frac{1}{2\pi i}\int_\gamma \frac{f(z)}{z-a}\,dz\] 이다.
② 증명 핵심 아이디어
  1. \(\frac{f(z)-f(a)}{z-a}\)는 \(a\)에서 제거가능한 특이점을 갖고, 전체 내부에서 해석적으로 확장된다.
  2. 코시 적분정리에 의해 그 함수의 닫힌곡선 적분은 \(0\)이다.
  3. 따라서 \(\int_\gamma\frac{f(z)}{z-a}\,dz=f(a)\int_\gamma\frac{1}{z-a}\,dz\)가 된다.
  4. 원점 중심의 작은 원으로 변형하여 계산하면 \(\int_\gamma\frac{1}{z-a}\,dz=2\pi i\)이다.
  5. 정리하면 코시 적분공식이 나온다.
③ 활용 예제
함수값을 경계값으로 표현, 복소해석학의 강력한 계산법, 테일러 전개와 유수정리의 기초이다.
④ 파이썬 코드
# 코시 적분공식 예시: f(z)=z^2, a=0, 단위원
import cmath
import math

N = 20000
total = 0j
a = 0j

for k in range(N):
    t1 = 2*math.pi*k/N
    t2 = 2*math.pi*(k+1)/N
    z1 = cmath.exp(1j*t1)
    z2 = cmath.exp(1j*t2)
    dz = z2 - z1
    fz = z1**2
    total += fz / (z1 - a) * dz

value = total / (2*math.pi*1j)

print("적분공식 결과:", value)
print("f(0):", a**2)
⑤ 코드 상세 설명
  1. `f(z)=z**2` : 사용할 해석함수이다.
  2. `a = 0j` : 함수값을 구할 내부점이다.
  3. `fz / (z1 - a)` : 코시 적분공식의 integrand \(rac{f(z)}{z-a}\)이다.
  4. `total` : 단위원 위 적분의 수치 근사이다.
  5. `total / (2*math.pi*1j)` : \(rac{1}{2\pi i}\)를 곱한 값이다.
  6. 결과는 \(f(0)=0\)에 가까워야 한다.
정리 4

코시 미분공식

① 정리 설명
\(f\)가 \(\gamma\) 내부와 위에서 해석적이면 \[f^{(n)}(a)=\frac{n!}{2\pi i}\int_\gamma\frac{f(z)}{(z-a)^{n+1}}\,dz\] 이다.
② 증명 핵심 아이디어
  1. 코시 적분공식은 \(f(a)\)를 경계 적분으로 표현한다.
  2. 이 공식에서 \(a\)에 대해 미분할 수 있다.
  3. 적분 안의 함수가 충분히 좋으므로 미분과 적분을 교환할 수 있다.
  4. 한 번 미분하면 \((z-a)^{-2}\)가 나타난다.
  5. 이를 \(n\)번 반복하면 계수 \(n!\)와 \((z-a)^{-(n+1)}\)가 생긴다.
  6. 따라서 코시 미분공식이 얻어진다.
③ 활용 예제
해석함수가 무한 번 미분 가능함을 보여 주며, 테일러 급수와 계수 추정에 사용된다.
④ 파이썬 코드
# f(z)=z^3에서 f'(0)=0 확인
import cmath
import math

N = 20000
a = 0j
total = 0j

for k in range(N):
    t1 = 2*math.pi*k/N
    t2 = 2*math.pi*(k+1)/N
    z1 = cmath.exp(1j*t1)
    z2 = cmath.exp(1j*t2)
    dz = z2 - z1
    total += (z1**3) / ((z1 - a)**2) * dz

derivative = total / (2*math.pi*1j)

print("f'(0) 근사:", derivative)
⑤ 코드 상세 설명
  1. `f(z)=z**3` : 사용할 함수이다.
  2. `((z1 - a)**2)` : \(n=1\)인 미분공식의 분모 \((z-a)^2\)이다.
  3. `total` : 선적분을 수치적으로 더한 값이다.
  4. `derivative = total / (2*pi*i)` : \(f'(a)\)를 계산한다.
  5. \(f'(z)=3z^2\)이므로 \(f'(0)=0\)이다.
정리 5

리우빌 정리

① 정리 설명
전해석함수 \(f:\mathbb C\to\mathbb C\)가 유계이면 \(f\)는 상수함수이다.
② 증명 핵심 아이디어
  1. 코시 미분공식을 사용하면 \(f'(a)\)를 원 \(C_R\) 위의 적분으로 표현할 수 있다.
  2. \(|f(z)|\le M\)이라고 하자.
  3. 코시 추정에 의해 \(|f'(a)|\le \frac{M}{R}\)가 된다.
  4. \(R\)은 임의로 크게 잡을 수 있으므로 \(R\to\infty\)를 보내면 \(f'(a)=0\)이다.
  5. 모든 \(a\)에서 \(f'(a)=0\)이므로 \(f\)는 상수함수이다.
③ 활용 예제
대수학의 기본정리 증명, 전해석함수의 성장성 분석, 복소함수 이론의 핵심 도구이다.
④ 파이썬 코드
# 유계처럼 보이는 함수와 유계가 아닌 함수 비교 샘플
import cmath

def f(z):
    return cmath.exp(z)

samples = [complex(x, 0) for x in range(0, 6)]
values = [abs(f(z)) for z in samples]

print(values)
print("값이 커지는가?", values[-1] > values[0])
⑤ 코드 상세 설명
  1. `cmath.exp(z)` : 복소 지수함수 \(e^z\)이다.
  2. `samples` : 실수축 위의 표본점이다.
  3. `abs(f(z))` : 복소수 함수값의 절댓값이다.
  4. `values[-1] > values[0]` : 값이 커지는지 확인한다.
  5. 리우빌 정리에 따르면 상수가 아닌 전해석함수는 전체 복소평면에서 유계일 수 없다.
  6. \(e^z\)는 전해석이지만 유계가 아니다.
정리 6

최대절댓값 원리

① 정리 설명
영역 \(D\)에서 해석적인 비상수 함수 \(f\)는 내부점에서 \(|f|\)의 최대값을 가질 수 없다.
② 증명 핵심 아이디어
  1. 만약 내부점 \(a\)에서 \(|f|\)가 최대라고 하자.
  2. 코시 적분공식은 \(f(a)\)가 작은 원 위 함수값들의 평균으로 표현됨을 보여 준다.
  3. 평균의 절댓값이 원 위 모든 값의 최대와 같으려면 원 위 값들이 같은 방향과 같은 크기를 가져야 한다.
  4. 따라서 작은 원 위에서 함수가 상수처럼 행동한다.
  5. 해석함수의 항등정리에 의해 함수는 영역 전체에서 상수이다.
  6. 비상수라는 가정과 모순이므로 내부 최대는 불가능하다.
③ 활용 예제
복소함수의 크기 제어, 조화함수, 편미분방정식, 안정성 분석에 사용된다.
④ 파이썬 코드
# f(z)=z의 절댓값은 단위원판 내부보다 경계에서 크다
points = [complex(x/10, y/10) for x in range(-9, 10) for y in range(-9, 10)
          if (x/10)**2 + (y/10)**2 <= 1]

max_abs = max(abs(z) for z in points)

print("샘플 내부 최대 절댓값:", max_abs)
print("경계 최대값 1에 가까운가?", abs(max_abs - 1) < 0.2)
⑤ 코드 상세 설명
  1. `points` : 단위원판 안의 격자점 샘플이다.
  2. `abs(z)` : 복소수 \(z\)의 절댓값이다.
  3. `max(abs(z) for z in points)` : 샘플 중 가장 큰 절댓값을 찾는다.
  4. \(f(z)=z\)는 해석함수이고 \(|f(z)|=|z|\)이다.
  5. 최대절댓값은 내부가 아니라 경계 \(|z|=1\)에서 나타난다.
정리 7

항등정리

① 정리 설명
영역 \(D\)에서 해석적인 두 함수 \(f,g\)가 \(D\) 안의 축적점을 갖는 집합에서 같으면, \(D\) 전체에서 \(f=g\)이다.
② 증명 핵심 아이디어
  1. 함수 \(h=f-g\)를 생각한다.
  2. \(h\)는 해석함수이고, 축적점을 갖는 무한한 영점들을 가진다.
  3. 해석함수의 영점은 함수가 영함수가 아닌 한 고립되어야 한다.
  4. 그런데 영점들이 축적점을 가지므로 고립되어 있지 않다.
  5. 따라서 \(h\equiv0\)이고 \(f=g\)이다.
③ 활용 예제
해석함수의 강한 결정성, 급수 전개, 복소함수 방정식, 해석적 연속에 사용된다.
④ 파이썬 코드
# 두 다항식이 많은 점에서 같으면 같은지 확인하는 직관
def f(x):
    return x**2 + 2*x + 1

def g(x):
    return (x + 1)**2

points = range(-5, 6)
same_all = all(f(x) == g(x) for x in points)

print("샘플 점들에서 모두 같은가?", same_all)
⑤ 코드 상세 설명
  1. `f(x)`와 `g(x)` : 서로 다르게 쓰였지만 같은 다항식이다.
  2. `points` : 여러 검사점이다.
  3. `all(...)` : 모든 점에서 같은지 확인한다.
  4. 복소해석학의 항등정리는 해석함수가 축적점을 갖는 무한집합에서 같으면 전체에서 같다고 말한다.
  5. 코드는 그 직관을 다항식 예시로 보여 준다.
정리 8

로랑 전개 정리

① 정리 설명
함수 \(f\)가 환형영역 \(r\lt |z-a|\lt R\)에서 해석적이면 \[f(z)=\sum_{n=-\infty}^{\infty}c_n(z-a)^n\] 으로 전개된다.
② 증명 핵심 아이디어
  1. 환형영역 안에서 점 \(z\)를 둘러싸는 두 원을 잡는다.
  2. 코시 적분공식을 안쪽 경계와 바깥쪽 경계에 적용한다.
  3. 각 적분핵 \(\frac{1}{\zeta-z}\)를 적절한 기하급수로 전개한다.
  4. 바깥쪽 경계에서는 양의 거듭제곱 항들이 생기고, 안쪽 경계에서는 음의 거듭제곱 항들이 생긴다.
  5. 두 부분을 합치면 로랑 급수가 된다.
③ 활용 예제
특이점 분석, 유수 계산, 복소적분, 물리학의 장론과 신호 해석에 사용된다.
④ 파이썬 코드
# 1/(1-z)의 테일러 전개 일부 확인: |z|<1
z = 0.5
partial_sum = 0

for n in range(10):
    partial_sum += z**n

exact = 1 / (1 - z)

print("부분합:", partial_sum)
print("정확값:", exact)
⑤ 코드 상세 설명
  1. `z = 0.5` : \(|z|<1\)인 값을 고른다.
  2. `partial_sum += z**n` : 기하급수 \(1+z+z^2+\cdots\)의 일부를 더한다.
  3. `exact = 1 / (1 - z)` : 원래 함수값이다.
  4. 로랑 전개는 테일러 전개를 음의 거듭제곱까지 확장한 것이다.
  5. 이 예시는 양의 거듭제곱만 있는 간단한 경우이다.
정리 9

유수정리

① 정리 설명
닫힌곡선 \(\gamma\) 내부의 고립특이점들이 \(a_1,\dots,a_n\)이면 \[\int_\gamma f(z)\,dz=2\pi i\sum_{k=1}^{n}\operatorname{Res}(f,a_k)\] 이다.
② 증명 핵심 아이디어
  1. 각 특이점 주변에서 \(f\)를 로랑 전개한다.
  2. 로랑 전개에서 \((z-a)^{-1}\) 항의 계수를 유수라고 한다.
  3. 닫힌곡선 적분에서 \((z-a)^m\) 항은 \(m\ne -1\)일 때 적분이 \(0\)이다.
  4. \((z-a)^{-1}\) 항만 적분값 \(2\pi i\)를 만든다.
  5. 여러 특이점의 기여를 모두 더하면 유수정리가 된다.
③ 활용 예제
복잡한 실적분 계산, 물리학의 푸리에 적분, 라플라스 변환, 안정성 해석에 사용된다.
④ 파이썬 코드
# f(z)=1/z 를 단위원 위에서 적분하면 2*pi*i
import cmath
import math

N = 20000
total = 0j

for k in range(N):
    t1 = 2*math.pi*k/N
    t2 = 2*math.pi*(k+1)/N
    z1 = cmath.exp(1j*t1)
    z2 = cmath.exp(1j*t2)
    dz = z2 - z1
    total += (1/z1) * dz

print("적분 근사:", total)
print("2*pi*i:", 2*math.pi*1j)
⑤ 코드 상세 설명
  1. `f(z)=1/z` : 원점에 단순극을 갖는 함수이다.
  2. `cmath.exp(1j*t)` : 단위원 위의 점을 만든다.
  3. `dz` : 작은 경로 조각이다.
  4. `total += (1/z1) * dz` : 선적분을 수치적으로 근사한다.
  5. 원점에서의 유수는 1이므로 유수정리에 의해 적분값은 \(2\pi i\)이다.
정리 10

루셰 정리

① 정리 설명
단순닫힌곡선 \(\gamma\) 위에서 \(|g(z)|\lt |f(z)|\)이면, \(f\)와 \(f+g\)는 \(\gamma\) 내부에서 같은 개수의 영점을 가진다.
② 증명 핵심 아이디어
  1. 곡선 위에서 \(f\)가 \(g\)보다 크므로 \(f+t g\)는 \(0\le t\le1\) 동안 경계에서 영점을 갖지 않는다.
  2. 따라서 \(f+t g\)의 경계 위 winding number는 변하지 않는다.
  3. 논변원리에 의해 이 winding number는 내부 영점의 개수와 같다.
  4. 그러므로 \(t=0\)인 \(f\)와 \(t=1\)인 \(f+g\)는 내부 영점 개수가 같다.
③ 활용 예제
다항식의 영점 개수 판정, 대수학의 기본정리, 안정성 해석에 사용된다.
④ 파이썬 코드
# 루셰 정리 직관: z^3 + 작은 상수는 단위원 내부에 영점 3개
# 여기서는 반지름과 계수 크기 조건만 확인
r = 1
small_constant = 0.2

# |g(z)| = 0.2, |f(z)| = |z^3| = 1 on |z|=1
condition = small_constant < r**3

print("경계에서 |g| < |f| 조건:", condition)
print("따라서 z^3과 z^3+0.2는 내부 영점 개수가 같다.")
⑤ 코드 상세 설명
  1. `r = 1` : 단위원 경계를 생각한다.
  2. `small_constant = 0.2` : 작은 교란항 \(g(z)=0.2\)이다.
  3. `r**3` : 경계 \(|z|=1\)에서 \(|z^3|=1\)이다.
  4. `small_constant < r**3` : 루셰 정리 조건 \(|g|<|f|\)를 확인한다.
  5. \(z^3\)은 원점에 3중근을 가지므로 \(z^3+0.2\)도 단위원 내부에 영점 3개를 가진다.
