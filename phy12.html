정리 1

아인슈타인의 질량-에너지 등가성

① 정리 설명
질량 \(m\)은 에너지와 동등하며, 정지에너지는 \[E=mc^2\] 이다.
② 증명 핵심 아이디어
  1. 특수상대성이론에서는 에너지와 운동량이 하나의 네-벡터로 결합된다.
  2. 상대론적 에너지와 운동량 사이에는 \(E^2=(pc)^2+(mc^2)^2\)가 성립한다.
  3. 물체가 정지해 있으면 운동량 \(p=0\)이다.
  4. 따라서 \(E^2=(mc^2)^2\)이고, 양의 에너지를 택하면 \(E=mc^2\)이다.
  5. 즉 질량은 에너지의 한 형태이다.
③ 활용 예제
핵분열, 핵융합, 별의 에너지 생성, 입자물리학, 원자력 발전에 사용된다.
④ 파이썬 코드
# 질량-에너지 등가성 E = mc^2
m = 0.001          # kg
c = 3.0e8          # m/s

E = m * c**2

print("에너지:", E, "J")
⑤ 코드 상세 설명
  1. `m = 0.001` : 질량을 kg 단위로 저장한다.
  2. `c = 3.0e8` : 빛의 속도를 m/s 단위로 저장한다.
  3. `c**2` : 빛의 속도의 제곱을 계산한다.
  4. `E = m * c**2` : \(E=mc^2\) 공식을 적용한다.
  5. `print()` : 질량이 에너지로 바뀔 때의 에너지를 줄 단위로 출력한다.
정리 2

시간 지연 공식

① 정리 설명
속도 \(v\)로 움직이는 관찰자의 시간 간격 \(\Delta t\)는 고유시간 \(\Delta\tau\)에 대해 \[\Delta t=\gamma\Delta\tau,\qquad \gamma=\frac{1}{\sqrt{1-\frac{v^2}{c^2}}}\] 이다.
② 증명 핵심 아이디어
  1. 빛시계 사고실험을 생각한다.
  2. 정지한 관찰자에게 빛은 위아래로 왕복하지만, 움직이는 관찰자를 보는 외부 관찰자에게 빛은 대각선 경로를 지난다.
  3. 빛의 속도 \(c\)는 모든 관성계에서 같아야 한다.
  4. 대각선 경로가 더 길기 때문에 외부 관찰자가 측정한 시간은 더 길다.
  5. 피타고라스 정리를 이용해 정리하면 \(\Delta t=\gamma\Delta\tau\)가 된다.
③ 활용 예제
뮤온 수명 증가, GPS 보정, 고속입자 실험, 우주선 시간 계산에 사용된다.
④ 파이썬 코드
# 특수상대성이론 시간 지연
c = 3.0e8
v = 0.8 * c
proper_time = 1.0

gamma = 1 / (1 - (v/c)**2) ** 0.5
dilated_time = gamma * proper_time

print("감마:", gamma)
print("관측 시간:", dilated_time, "s")
⑤ 코드 상세 설명
  1. `c` : 빛의 속도이다.
  2. `v = 0.8 * c` : 물체가 빛의 속도의 80%로 움직인다고 둔다.
  3. `proper_time` : 물체와 함께 움직이는 관찰자의 고유시간이다.
  4. `gamma` : 로런츠 인자 \(\gamma\)이다.
  5. `dilated_time` : 외부 관찰자가 측정하는 더 길어진 시간이다.
정리 3

드브로이 물질파 관계

① 정리 설명
운동량 \(p\)를 가진 입자는 파장 \(\lambda\)를 가지며 \[\lambda=\frac{h}{p}\] 이다.
② 증명 핵심 아이디어
  1. 빛은 입자처럼 에너지와 운동량을 가지며 \(p=\frac{h}{\lambda}\)를 만족한다.
  2. 드브로이는 이 관계가 빛뿐 아니라 물질입자에도 적용된다고 제안했다.
  3. 전자 회절 실험에서 전자가 파동처럼 간섭무늬를 만든다는 사실이 확인되었다.
  4. 따라서 물질입자의 파장은 \(\lambda=\frac{h}{p}\)로 주어진다.
③ 활용 예제
전자현미경, 입자 회절, 양자역학의 파동성, 나노스케일 측정에 사용된다.
④ 파이썬 코드
# 전자의 드브로이 파장 계산
h = 6.626e-34       # J*s
m = 9.109e-31       # kg
v = 1.0e6           # m/s

p = m * v
wavelength = h / p

print("운동량:", p)
print("드브로이 파장:", wavelength, "m")
⑤ 코드 상세 설명
  1. `h` : 플랑크 상수이다.
  2. `m` : 전자의 질량이다.
  3. `v` : 전자의 속도이다.
  4. `p = m * v` : 비상대론적 운동량을 계산한다.
  5. `wavelength = h / p` : 드브로이 관계 \(\lambda=h/p\)를 적용한다.
정리 4

광전효과 방정식

① 정리 설명
빛의 진동수 \(f\)가 금속에 입사할 때 최대 운동에너지는 \[K_{\max}=hf-\phi\] 이다.
② 증명 핵심 아이디어
  1. 빛은 에너지 \(hf\)를 가진 광자로 이루어져 있다.
  2. 전자가 금속에서 빠져나오려면 일함수 \(\phi\)만큼의 에너지가 필요하다.
  3. 광자 하나의 에너지 중 \(\phi\)를 쓰고 남은 에너지가 전자의 운동에너지가 된다.
  4. 따라서 \(K_{\max}=hf-\phi\)이다.
  5. 만약 \(hf\lt\phi\)이면 전자는 방출되지 않는다.
③ 활용 예제
태양전지, 광센서, 광전관, 양자론의 탄생, 플랑크 상수 측정에 사용된다.
④ 파이썬 코드
# 광전효과 최대 운동에너지
h = 6.626e-34
f = 1.0e15
phi = 3.0e-19

K = h * f - phi

print("최대 운동에너지:", K, "J")
print("전자 방출 가능?", K > 0)
⑤ 코드 상세 설명
  1. `h` : 플랑크 상수이다.
  2. `f` : 입사한 빛의 진동수이다.
  3. `phi` : 금속의 일함수이다.
  4. `K = h * f - phi` : 광전효과 방정식을 계산한다.
  5. `K > 0` : 남는 에너지가 있으면 전자가 방출될 수 있다.
정리 5

하이젠베르크 불확정성 원리

① 정리 설명
입자의 위치 불확정성 \(\Delta x\)와 운동량 불확정성 \(\Delta p\)는 \[\Delta x\,\Delta p\ge \frac{\hbar}{2}\] 를 만족한다.
② 증명 핵심 아이디어
  1. 양자상태는 파동함수로 표현된다.
  2. 위치를 정확히 알기 위해 파동묶음을 좁게 만들면 여러 파수 성분이 필요하다.
  3. 파수의 퍼짐은 운동량의 퍼짐과 연결된다.
  4. 푸리에 해석에서 함수의 폭과 푸리에 변환의 폭은 동시에 임의로 작아질 수 없다.
  5. 이를 양자역학의 위치와 운동량 연산자에 적용하면 \(\Delta x\Delta p\ge\frac{\hbar}{2}\)가 된다.
③ 활용 예제
원자 안정성, 전자현미경 한계, 양자 측정, 터널링, 나노기술에 사용된다.
④ 파이썬 코드
# 불확정성 원리로 최소 운동량 불확정성 계산
hbar = 1.055e-34
delta_x = 1.0e-10

delta_p_min = hbar / (2 * delta_x)

print("최소 운동량 불확정성:", delta_p_min, "kg*m/s")
⑤ 코드 상세 설명
  1. `hbar` : 축약 플랑크 상수 \(\hbar\)이다.
  2. `delta_x` : 위치의 불확정성이다.
  3. `delta_p_min` : 불확정성 원리에서 가능한 최소 운동량 불확정성이다.
  4. `hbar / (2 * delta_x)` : \(\Delta p\ge\hbar/(2\Delta x)\)를 계산한다.
정리 6

시간 독립 슈뢰딩거 방정식

① 정리 설명
정상상태 파동함수 \(\psi(x)\)는 \[\hat H\psi=E\psi\] 를 만족한다.
② 증명 핵심 아이디어
  1. 양자역학에서 에너지는 해밀토니언 연산자 \(\hat H\)로 표현된다.
  2. 정상상태는 시간에 따라 확률분포가 변하지 않는 상태이다.
  3. 이때 파동함수의 공간 부분은 에너지 고유상태가 된다.
  4. 따라서 해밀토니언을 작용시키면 같은 함수에 에너지 \(E\)가 곱해진다.
  5. 이것이 \(\hat H\psi=E\psi\)이다.
③ 활용 예제
원자 에너지 준위, 양자우물, 분자 오비탈, 반도체, 양자컴퓨터의 기본 모델에 사용된다.
④ 파이썬 코드
# 무한정사각우물 에너지 준위
h = 6.626e-34
m = 9.109e-31
L = 1.0e-9
n = 1

E = n**2 * h**2 / (8 * m * L**2)

print("에너지 준위:", E, "J")
⑤ 코드 상세 설명
  1. `h` : 플랑크 상수이다.
  2. `m` : 입자의 질량이다.
  3. `L` : 1차원 상자의 길이이다.
  4. `n` : 양자수이다.
  5. `E = n**2*h**2/(8*m*L**2)` : 무한정사각우물의 에너지 준위 공식이다.
  6. 양자수 \(n\)이 커질수록 에너지는 \(n^2\)에 비례해 증가한다.
정리 7

보른 확률 해석

① 정리 설명
파동함수 \(\psi(x)\)에 대해 \[|\psi(x)|^2\] 는 입자를 위치 \(x\) 근처에서 발견할 확률밀도이다.
② 증명 핵심 아이디어
  1. 파동함수 자체는 일반적으로 복소수 값을 가진다.
  2. 물리적으로 직접 관측되는 것은 복소 진폭이 아니라 확률이다.
  3. 복소수의 크기 제곱 \(|\psi|^2\)는 항상 0 이상이므로 확률밀도로 해석할 수 있다.
  4. 전체 공간에서 확률의 합은 1이어야 하므로 \(\int |\psi(x)|^2 dx=1\)로 정규화한다.
  5. 이 해석은 전자 회절과 간섭 실험 결과와 일치한다.
③ 활용 예제
양자 측정, 원자 오비탈, 전자구름, 확률밀도 해석, 양자 시뮬레이션에 사용된다.
④ 파이썬 코드
# 간단한 복소 파동함수의 확률밀도
psi = 0.6 + 0.8j

prob_density = abs(psi)**2

print("확률밀도:", prob_density)
⑤ 코드 상세 설명
  1. `psi = 0.6 + 0.8j` : 복소수 파동함수 값을 예로 든다.
  2. `abs(psi)` : 복소수의 크기이다.
  3. `abs(psi)**2` : \(|\psi|^2\), 즉 확률밀도를 계산한다.
  4. 복소수 값 자체가 아니라 크기의 제곱이 물리적 확률과 연결된다.
정리 8

파울리 배타 원리

① 정리 설명
두 동일한 페르미온은 완전히 같은 양자상태를 동시에 차지할 수 없다.
② 증명 핵심 아이디어
  1. 전자 같은 페르미온의 전체 파동함수는 두 입자를 교환하면 부호가 바뀌는 반대칭성을 가진다.
  2. 두 입자가 같은 상태에 있다고 하면 교환해도 물리적으로 같은 상태여야 한다.
  3. 하지만 반대칭성 때문에 파동함수는 자기 자신의 음수가 된다.
  4. 즉 \(\psi=-\psi\)이므로 \(\psi=0\)이어야 한다.
  5. 확률이 0이므로 두 동일한 페르미온은 같은 상태를 동시에 차지할 수 없다.
③ 활용 예제
원자 주기율표, 전자껍질, 백색왜성의 축퇴압, 고체의 전자구조에 사용된다.
④ 파이썬 코드
# 전자 상태 중복 검사
states = [
    ("n=1", "l=0", "m=0", "spin=+1/2"),
    ("n=1", "l=0", "m=0", "spin=-1/2"),
    ("n=1", "l=0", "m=0", "spin=+1/2")
]

unique_states = set(states)

print("전체 전자 수:", len(states))
print("서로 다른 상태 수:", len(unique_states))
print("중복 상태가 있는가?", len(states) != len(unique_states))
⑤ 코드 상세 설명
  1. `states` : 전자들의 양자상태를 튜플로 표현한다.
  2. `set(states)` : 중복된 상태를 제거한다.
  3. `len(states)` : 전체 전자 상태 목록의 개수이다.
  4. `len(unique_states)` : 서로 다른 상태의 개수이다.
  5. 둘이 다르면 같은 양자상태가 중복된 것이므로 파울리 배타 원리에 어긋난다.
정리 9

상대론적 에너지-운동량 관계

① 정리 설명
입자의 에너지 \(E\), 운동량 \(p\), 질량 \(m\)은 \[E^2=(pc)^2+(mc^2)^2\] 를 만족한다.
② 증명 핵심 아이디어
  1. 특수상대성이론에서 에너지와 운동량은 에너지-운동량 네-벡터를 이룬다.
  2. 이 네-벡터의 민코프스키 길이는 모든 관성계에서 불변이다.
  3. 정지계에서는 운동량 \(p=0\)이고 에너지는 \(mc^2\)이다.
  4. 모든 관성계에서 불변량이 같아야 하므로 \(E^2-(pc)^2=(mc^2)^2\)이다.
  5. 정리하면 \(E^2=(pc)^2+(mc^2)^2\)이다.
③ 활용 예제
입자물리학, 가속기 실험, 광자 에너지, 우주선 분석, 핵반응 계산에 사용된다.
④ 파이썬 코드
# 상대론적 에너지 계산
m = 9.109e-31
p = 1.0e-22
c = 3.0e8

E = ((p*c)**2 + (m*c**2)**2) ** 0.5

print("상대론적 에너지:", E, "J")
⑤ 코드 상세 설명
  1. `m` : 입자의 질량이다.
  2. `p` : 입자의 운동량이다.
  3. `c` : 빛의 속도이다.
  4. `(p*c)**2` : 운동량에 의한 에너지 항이다.
  5. `(m*c**2)**2` : 정지에너지 항의 제곱이다.
  6. `** 0.5` : 제곱근을 취해 에너지 \(E\)를 구한다.
정리 10

플랑크 복사 양자화

① 정리 설명
전자기파 에너지는 연속적이 아니라 진동수 \(f\)에 대해 \[E=hf\] 단위로 양자화된다.
② 증명 핵심 아이디어
  1. 고전물리학으로 흑체복사를 설명하면 짧은 파장에서 에너지가 무한히 커지는 자외선 파탄이 발생한다.
  2. 플랑크는 진동자가 에너지를 연속적으로 주고받는 것이 아니라 \(hf\)의 정수배로만 주고받는다고 가정했다.
  3. 이 가정은 고주파 영역의 에너지 분포를 자연스럽게 억제한다.
  4. 그 결과 실험과 일치하는 흑체복사 법칙이 얻어진다.
  5. 이것이 양자론의 출발점이 되었다.
③ 활용 예제
양자역학의 탄생, 흑체복사, 광자 에너지, 레이저, 분광학에 사용된다.
④ 파이썬 코드
# 광자 하나의 에너지 계산
h = 6.626e-34
f = 5.0e14

E = h * f

print("광자 에너지:", E, "J")
⑤ 코드 상세 설명
  1. `h` : 플랑크 상수이다.
  2. `f` : 빛의 진동수이다.
  3. `E = h * f` : 광자 하나의 에너지를 계산한다.
  4. 진동수가 클수록 광자 에너지도 커진다.

마무리 핵심 정리

현대물리학은 고전물리학으로 설명하기 어려운 빠른 속도와 미시 세계를 설명한다. 상대성이론은 시간·공간·에너지의 관계를 바꾸었고, 양자역학은 입자와 파동, 확률, 측정의 개념을 새롭게 만들었다. 파이썬 계산을 활용하면 추상적인 공식도 실제 수치로 확인할 수 있다.
