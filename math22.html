정리 1

부정적분과 원시함수

① 정리 설명
함수 \(f\)의 원시함수 \(F\)가 \(F'(x)=f(x)\)를 만족하면 \[\int f(x)\,dx=F(x)+C\] 이다.
② 증명 핵심 아이디어
  1. 적분은 미분의 역과정으로 볼 수 있다.
  2. \(F'(x)=f(x)\)이면 \(F\)를 \(f\)의 원시함수라고 한다.
  3. 상수 \(C\)를 더해도 미분하면 0이 되므로 같은 도함수를 가진다.
  4. 그래서 부정적분에는 항상 적분상수 \(C\)가 붙는다.
  5. 부정적분은 함수의 모든 원시함수 집합을 나타낸다.
③ 활용 예제
속도에서 위치 구하기, 변화율에서 누적량 구하기, 미분방정식의 기본 풀이에 활용된다.
④ 파이썬 코드
# f(x)=2x의 원시함수 F(x)=x^2 확인
def f(x):
    return 2*x

def F(x):
    return x**2

x = 3
h = 1e-6
derivative = (F(x + h) - F(x)) / h

print("F'(x) 근사:", derivative)
print("f(x):", f(x))
⑤ 코드 상세 설명
  1. `f(x)`는 적분할 함수 \(2x\)이다.
  2. `F(x)`는 원시함수 후보 \(x^2\)이다.
  3. 차분몫으로 \(F'(x)\)를 근사한다.
  4. `F'(3)`은 약 6이고 `f(3)`도 6이다.
  5. 따라서 \(F\)는 \(f\)의 원시함수임을 확인할 수 있다.
정리 2

거듭제곱 적분법

① 정리 설명
\(n\ne -1\)일 때 \[\int x^n\,dx=\frac{x^{n+1}}{n+1}+C\] 이다.
② 증명 핵심 아이디어
  1. 거듭제곱 미분법 \(rac{d}{dx}x^{n+1}=(n+1)x^n\)에서 출발한다.
  2. \(x^{n+1}\)을 미분하면 계수 \(n+1\)이 생긴다.
  3. 이를 없애기 위해 \(rac{1}{n+1}\)을 곱한다.
  4. 따라서 \(rac{x^{n+1}}{n+1}\)의 도함수는 \(x^n\)이다.
  5. 단, \(n=-1\)일 때는 로그함수 적분이 된다.
③ 활용 예제
다항함수 적분, 넓이 계산, 물리량 누적 계산에 활용된다.
④ 파이썬 코드
# x^n 적분 공식 계산
def power_integral(x, n):
    return x**(n + 1) / (n + 1)

x = 2
n = 3

print("x^3의 원시함수값:", power_integral(x, n))
⑤ 코드 상세 설명
  1. `power_integral(x, n)`은 \(rac{x^{n+1}}{n+1}\)을 계산한다.
  2. `n = 3`이면 \(x^3\)의 원시함수는 \(rac{x^4}{4}\)이다.
  3. `x = 2`를 대입하면 \(16/4=4\)가 나온다.
정리 3

정적분의 넓이 해석

① 정리 설명
연속함수 \(f(x)\ge0\)에 대해 정적분 \[\int_a^b f(x)\,dx\] 는 \(x=a\)부터 \(x=b\)까지 그래프 아래의 넓이를 의미한다.
② 증명 핵심 아이디어
  1. 구간을 작은 조각으로 나눈다.
  2. 각 조각에서 함수값을 높이로 하는 직사각형을 만든다.
  3. 직사각형들의 넓이를 모두 더하면 그래프 아래 넓이의 근사가 된다.
  4. 조각의 폭을 0에 가깝게 줄이면 정확한 넓이에 가까워진다.
  5. 이 극한이 정적분이다.
③ 활용 예제
면적 계산, 거리 계산, 누적 생산량, 확률밀도 함수 해석에 활용된다.
④ 파이썬 코드
# 리만 합으로 integral_0^1 x^2 dx 근사
def f(x):
    return x**2

a = 0
b = 1
n = 10000
dx = (b - a) / n

area = sum(f(a + i*dx) * dx for i in range(n))

print("넓이 근사:", area)
print("정확값:", 1/3)
⑤ 코드 상세 설명
  1. `f(x)`는 적분할 함수 \(x^2\)이다.
  2. `dx`는 작은 직사각형의 폭이다.
  3. `f(a+i*dx)*dx`는 작은 직사각형 하나의 넓이다.
  4. `sum(...)`은 모든 직사각형 넓이를 더한다.
  5. 결과는 \(\int_0^1x^2dx=rac13\)에 가까워진다.
정리 4

미적분의 기본정리 1

① 정리 설명
\(F(x)=\int_a^x f(t)\,dt\)로 정의하면, \(f\)가 연속일 때 \[F'(x)=f(x)\] 이다.
② 증명 핵심 아이디어
  1. \(F(x)\)는 \(a\)부터 \(x\)까지의 누적 넓이를 나타낸다.
  2. \(x\)가 조금 증가하면 새로 추가되는 넓이는 대략 \(f(x)\Delta x\)이다.
  3. 따라서 \(F(x+\Delta x)-F(x)\)는 \(f(x)\Delta x\)에 가깝다.
  4. 이를 \(\Delta x\)로 나누면 \(f(x)\)에 가까워진다.
  5. 극한을 취하면 \(F'(x)=f(x)\)이다.
③ 활용 예제
누적량 함수의 변화율 계산, 면적 함수 해석, 물리학의 위치-속도 관계에 활용된다.
④ 파이썬 코드
# F(x)=integral_0^x 2t dt = x^2의 도함수 확인
def F(x):
    return x**2

def f(x):
    return 2*x

x = 4
h = 1e-6
Fp = (F(x + h) - F(x)) / h

print("F'(x) 근사:", Fp)
print("f(x):", f(x))
⑤ 코드 상세 설명
  1. `F(x)=x**2`는 \(\int_0^x2t\,dt\)와 같다.
  2. 차분몫으로 \(F'(x)\)를 근사한다.
  3. 기본정리에 따르면 \(F'(x)=2x\)이다.
  4. `x=4`에서 두 값은 약 8로 같다.
정리 5

미적분의 기본정리 2

① 정리 설명
\(F'(x)=f(x)\)이면 \[\int_a^b f(x)\,dx=F(b)-F(a)\] 이다.
② 증명 핵심 아이디어
  1. 정적분은 작은 변화량들의 합이다.
  2. \(F'(x)=f(x)\)이면 작은 구간에서 \(F(x+\Delta x)-F(x)pprox f(x)\Delta x\)이다.
  3. 작은 변화량들을 모두 더하면 중간 항들이 서로 소거된다.
  4. 결국 시작값과 끝값의 차이 \(F(b)-F(a)\)만 남는다.
  5. 따라서 정적분은 원시함수의 끝값 차이로 계산된다.
③ 활용 예제
넓이 계산, 누적 변화량 계산, 물리학의 일과 에너지 계산에 활용된다.
④ 파이썬 코드
# integral_1^3 2x dx = x^2 | 1 to 3
def F(x):
    return x**2

a = 1
b = 3

integral = F(b) - F(a)

print("정적분값:", integral)
⑤ 코드 상세 설명
  1. `F(x)=x**2`는 \(2x\)의 원시함수이다.
  2. `a`, `b`는 적분 구간이다.
  3. `F(b)-F(a)`로 정적분을 계산한다.
  4. 결과는 \(9-1=8\)이다.
정리 6

치환적분법

① 정리 설명
미분 가능한 \(u=g(x)\)에 대해 \[\int f(g(x))g'(x)\,dx=\int f(u)\,du\] 이다.
② 증명 핵심 아이디어
  1. 치환적분은 연쇄법칙의 역과정이다.
  2. 복잡한 함수 안쪽을 \(u\)로 둔다.
  3. 안쪽 함수의 미분 \(g'(x)\,dx\)가 함께 있으면 \(du\)로 바꿀 수 있다.
  4. 그러면 적분식이 더 단순해진다.
  5. 적분 후 다시 \(u=g(x)\)를 대입한다.
③ 활용 예제
합성함수 적분, 삼각함수 적분, 지수함수 적분, 넓이 계산에 활용된다.
④ 파이썬 코드
# integral 2x*cos(x^2) dx의 원시함수는 sin(x^2)
import math

def F(x):
    return math.sin(x**2)

x = 1
h = 1e-6
derivative = (F(x + h) - F(x)) / h
original = 2*x*math.cos(x**2)

print("원시함수 미분 근사:", derivative)
print("원래 integrand:", original)
⑤ 코드 상세 설명
  1. `F(x)=sin(x**2)`이다.
  2. 연쇄법칙으로 미분하면 \(2x\cos(x^2)\)가 된다.
  3. 따라서 \(\int2x\cos(x^2)dx=\sin(x^2)+C\)이다.
  4. 코드는 원시함수를 미분해 원래 함수가 나오는지 확인한다.
정리 7

부분적분법

① 정리 설명
미분 가능한 함수 \(u,v\)에 대해 \[\int u\,dv=uv-\int v\,du\] 이다.
② 증명 핵심 아이디어
  1. 부분적분은 곱의 미분법에서 나온다.
  2. 곱의 미분법은 \((uv)'=u'v+uv'\)이다.
  3. 양변을 적분하면 \(uv=\int u'v\,dx+\int uv'\,dx\)이다.
  4. 이를 정리하면 \(\int u\,dv=uv-\int v\,du\)가 된다.
  5. 곱 형태의 적분을 더 쉬운 적분으로 바꿀 때 사용한다.
③ 활용 예제
로그함수 적분, \(xe^x\), \(x\sin x\) 적분, 물리학 계산에 활용된다.
④ 파이썬 코드
# integral x*e^x dx = e^x(x-1)+C 확인
import math

def F(x):
    return math.exp(x) * (x - 1)

def integrand(x):
    return x * math.exp(x)

x = 2
h = 1e-6
derivative = (F(x + h) - F(x)) / h

print("원시함수 미분 근사:", derivative)
print("원래 integrand:", integrand(x))
⑤ 코드 상세 설명
  1. `F(x)=e^x(x-1)`은 \(xe^x\)의 원시함수이다.
  2. 차분몫으로 \(F'(x)\)를 계산한다.
  3. `integrand(x)`는 원래 적분할 함수 \(xe^x\)이다.
  4. 두 값이 같으면 부분적분 결과를 확인한 것이다.
정리 8

정적분의 평균값 정리

① 정리 설명
연속함수 \(f\)에 대해 어떤 \(c\in[a,b]\)가 존재하여 \[\int_a^b f(x)\,dx=f(c)(b-a)\] 이다.
② 증명 핵심 아이디어
  1. 연속함수는 닫힌구간에서 최댓값과 최솟값을 가진다.
  2. 정적분값은 그 사이의 어떤 높이를 가진 직사각형 넓이로 표현될 수 있다.
  3. 그 높이가 함수의 평균값이다.
  4. 연속성 때문에 평균 높이에 해당하는 실제 점 \(c\)가 존재한다.
  5. 따라서 적분값은 \(f(c)(b-a)\)로 쓸 수 있다.
③ 활용 예제
평균온도, 평균속도, 평균농도, 데이터 평균 해석에 활용된다.
④ 파이썬 코드
# f(x)=x^2 on [0,3]의 평균값
def F(x):
    return x**3 / 3

a = 0
b = 3

integral = F(b) - F(a)
average_value = integral / (b - a)

print("정적분:", integral)
print("함수의 평균값:", average_value)
⑤ 코드 상세 설명
  1. `F(x)=x**3/3`은 \(x^2\)의 원시함수이다.
  2. `integral`은 \(\int_0^3x^2dx\)이다.
  3. `average_value`는 적분값을 구간 길이로 나눈 값이다.
  4. 이는 함수의 평균 높이를 의미한다.
정리 9

이상적분

① 정리 설명
무한구간이나 함수가 무한대로 발산하는 점을 포함한 적분은 극한으로 정의한다. 예를 들어 \[\int_1^\infty \frac{1}{x^2}\,dx=\lim_{b\to\infty}\int_1^b\frac{1}{x^2}\,dx\] 이다.
② 증명 핵심 아이디어
  1. 일반 정적분은 유한한 구간과 유계함수에서 정의된다.
  2. 무한구간에서는 끝점을 무한대로 보내는 극한을 사용한다.
  3. 함수가 어떤 점에서 무한히 커지는 경우도 극한으로 정의한다.
  4. 극한이 유한하면 이상적분은 수렴한다.
  5. 극한이 무한하거나 존재하지 않으면 발산한다.
③ 활용 예제
확률분포, 물리학의 무한 영역 계산, 수렴성 판정에 활용된다.
④ 파이썬 코드
# integral_1^B 1/x^2 dx가 1에 가까워지는지 확인
def F(x):
    return -1/x

for B in [10, 100, 1000, 10000]:
    value = F(B) - F(1)
    print(B, value)
⑤ 코드 상세 설명
  1. `F(x)=-1/x`는 \(1/x^2\)의 원시함수이다.
  2. 상한 `B`를 점점 크게 한다.
  3. `F(B)-F(1)`은 \(\int_1^B1/x^2dx\)이다.
  4. `B`가 커질수록 값은 1에 가까워진다.
  5. 따라서 \(\int_1^\infty1/x^2dx=1\)이다.
정리 10

적분과 넓이 사이의 부호

① 정리 설명
정적분은 부호 있는 넓이이다. \(f(x)\lt0\)인 구간에서는 \[\int_a^b f(x)\,dx\] 가 음수로 계산될 수 있다.
② 증명 핵심 아이디어
  1. 정적분은 단순한 기하학적 넓이와 다르게 부호를 가진다.
  2. 그래프가 \(x\)축 위에 있으면 양의 넓이로 더해진다.
  3. 그래프가 \(x\)축 아래에 있으면 음의 넓이로 더해진다.
  4. 실제 기하학적 넓이를 구하려면 \(\int_a^b |f(x)|\,dx\)를 계산해야 한다.
  5. 따라서 정적분과 면적을 구분해야 한다.
③ 활용 예제
변위와 이동거리 구분, 순변화량 계산, 그래프 해석에 활용된다.
④ 파이썬 코드
# f(x)=x를 [-1,1]에서 적분하면 부호가 상쇄됨
def F(x):
    return x**2 / 2

a = -1
b = 1

signed_area = F(b) - F(a)

print("부호 있는 넓이:", signed_area)
⑤ 코드 상세 설명
  1. `f(x)=x`는 원점 기준으로 대칭이다.
  2. \([-1,0]\)에서는 함수값이 음수이다.
  3. \([0,1]\)에서는 함수값이 양수이다.
  4. 두 부호 있는 넓이가 서로 상쇄된다.
  5. 그래서 정적분값은 0이다.

마무리 핵심 정리

적분법은 변화율을 누적량으로 바꾸는 언어이다. 부정적분은 원시함수를 찾는 과정이고, 정적분은 부호 있는 넓이와 누적 변화량을 의미한다. 미적분의 기본정리는 미분과 적분이 서로 역관계임을 보여 주며, 치환적분과 부분적분은 복잡한 적분을 단순화하는 강력한 도구이다.
