위대한 유산 이야기로 배우는 파이썬 문법 10단계

찰스 디킨스의 《위대한 유산》 상세 줄거리를 따라가며 파이썬 문법을 기초부터 응용·심화까지 익히는 10단계 학습 자료입니다. 각 단계는 이야기, 문법 핵심, 예시 코드 10개, 응용 미션으로 구성되어 있습니다.

1단계. 공동묘지의 핍 — 출력, 변수, 문자열 기초

이야기 흐름
어린 핍은 안개 낀 공동묘지에서 자신의 이름과 두려움을 마주합니다. 이 단계에서는 print(), 변수, 문자열을 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. print()로 문장 출력 

print("핍은 안개 낀 공동묘지에 서 있었다.")
설명: 문장을 화면에 출력합니다.

예시 2. 변수에 이름 저장 

name = "핍"
print(name)
설명: 값을 변수에 저장하고 출력합니다.

예시 3. 문자열 연결 

hero = "핍"
place = "공동묘지"
print(hero + "은 " + place + "에 있었다.")
설명: +로 문자열을 이어 붙입니다.

예시 4. f-string 사용 

name = "핍"
feeling = "두려움"
print(f"{name}은 {feeling}을 느꼈다.")
설명: f-string으로 변수 값을 문장 안에 넣습니다.

예시 5. 여러 줄 출력 

print("안개")
print("묘비")
print("작은 소년")
설명: print()를 여러 번 사용합니다.

예시 6. 문자열 반복 

sound = "쿵 "
print(sound * 3)
설명: 문자열에 숫자를 곱하면 반복됩니다.

예시 7. 문자열 길이 

sentence = "Great Expectations"
print(len(sentence))
설명: len()은 문자열 길이를 구합니다.

예시 8. 인덱싱 

word = "Pip"
print(word[0])
print(word[-1])
설명: 문자열에서 특정 위치 글자를 꺼냅니다.

예시 9. 슬라이싱 

word = "Expectations"
print(word[0:6])
설명: 문자열의 일부를 잘라냅니다.

예시 10. 입력 예시 

name = input("이름을 입력하세요: ")
print(name + "의 이야기가 시작됩니다.")
설명: input()으로 사용자 입력을 받습니다.
응용 미션: 공동묘지의 핍 장면을 바탕으로 변수 이름, 인물 이름, 문장을 바꾸어 자신만의 짧은 파이썬 이야기 코드를 만들어 보세요.

2단계. 탈옥수 매그위치 — 숫자, 연산자, 형 변환

이야기 흐름
핍은 탈옥수에게 먹을 것과 줄칼을 가져다줍니다. 물건의 개수와 시간을 계산하며 숫자와 연산을 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 정수 계산 

bread = 2
file = 1
print(bread + file)
설명: 정수끼리 더합니다.

예시 2. 뺄셈 

food = 5
given = 2
print(food - given)
설명: 남은 양을 계산합니다.

예시 3. 곱셈 

days = 3
fear = 10
print(days * fear)
설명: 반복되는 값을 곱합니다.

예시 4. 나눗셈 

coins = 10
people = 2
print(coins / people)
설명: 나눗셈 결과는 실수입니다.

예시 5. 몫과 나머지 

chains = 17
cut_each_day = 5
print(chains // cut_each_day)
print(chains % cut_each_day)
설명: 몫과 나머지를 구합니다.

예시 6. 거듭제곱 

hope = 2
print(hope ** 3)
설명: 제곱 계산입니다.

예시 7. 비교 연산 

pip_age = 7
magwitch_age = 40
print(pip_age < magwitch_age)
설명: 비교 결과는 True 또는 False입니다.

예시 8. 문자열 숫자 변환 

text = "15"
num = int(text)
print(num + 5)
설명: 문자열을 정수로 바꿉니다.

예시 9. 실수 변환 

distance = "3.5"
print(float(distance) * 2)
설명: 문자열을 실수로 바꿉니다.

예시 10. 복합 대입 

fear = 10
fear += 5
print(fear)
설명: 기존 변수에 값을 더해 저장합니다.
응용 미션: 탈옥수 매그위치 장면을 바탕으로 변수 이름, 인물 이름, 문장을 바꾸어 자신만의 짧은 파이썬 이야기 코드를 만들어 보세요.

3단계. 새티스 하우스 — 조건문 if, elif, else

이야기 흐름
핍은 미스 해비셤의 저택에서 차가운 에스텔라를 만납니다. 상황에 따라 판단하는 조건문을 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 기본 if 

visitor = "핍"
if visitor == "핍":
    print("저택에 들어간다.")
설명: 조건이 참이면 들여쓰기 된 코드가 실행됩니다.

예시 2. if-else 

shoe = "낡음"
if shoe == "깨끗함":
    print("자신감")
else:
    print("부끄러움")
설명: 조건이 거짓이면 else가 실행됩니다.

예시 3. elif 

mood = 70
if mood >= 90:
    print("기쁨")
elif mood >= 50:
    print("긴장")
else:
    print("슬픔")
설명: 여러 조건을 순서대로 검사합니다.

예시 4. and 조건 

dress = True
manners = True
if dress and manners:
    print("신사처럼 보인다.")
설명: 두 조건이 모두 참이어야 합니다.

예시 5. or 조건 

money = False
kindness = True
if money or kindness:
    print("가치 있는 사람")
설명: 둘 중 하나만 참이어도 됩니다.

예시 6. not 조건 

afraid = False
if not afraid:
    print("한 걸음 앞으로")
설명: not은 참과 거짓을 뒤집습니다.

예시 7. 중첩 조건문 

place = "저택"
feeling = "긴장"
if place == "저택":
    if feeling == "긴장":
        print("조심스럽게 걷는다.")
설명: 조건문 안에 조건문을 넣을 수 있습니다.

예시 8. 점수 판정 

score = 85
if score >= 80:
    print("훌륭한 태도")
else:
    print("더 배워야 함")
설명: 기준 점수로 결과를 나눕니다.

예시 9. 삼항 표현식 

rich = False
result = "신사" if rich else "대장장이 견습생"
print(result)
설명: 간단한 조건은 한 줄로 표현할 수 있습니다.

예시 10. 문자 포함 조건 

words = "거친 손과 투박한 구두"
if "구두" in words:
    print("핍은 상처를 받았다.")
설명: in으로 포함 여부를 확인합니다.
응용 미션: 새티스 하우스 장면을 바탕으로 변수 이름, 인물 이름, 문장을 바꾸어 자신만의 짧은 파이썬 이야기 코드를 만들어 보세요.

4단계. 조 가저리의 대장간 — 리스트와 반복문

이야기 흐름
대장간에는 망치, 쇠, 불꽃, 땀이 있습니다. 여러 데이터를 모아 다루는 리스트와 반복문을 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 리스트 생성 

tools = ["망치", "모루", "집게"]
print(tools)
설명: 여러 값을 리스트에 저장합니다.

예시 2. 리스트 인덱싱 

tools = ["망치", "모루", "집게"]
print(tools[0])
설명: 리스트의 첫 번째 값을 꺼냅니다.

예시 3. append 

tools = ["망치"]
tools.append("줄칼")
print(tools)
설명: append()는 리스트 끝에 값을 추가합니다.

예시 4. 반복문 for 

tools = ["망치", "모루", "집게"]
for tool in tools:
    print(tool)
설명: 리스트의 값을 하나씩 꺼냅니다.

예시 5. range 

for strike in range(5):
    print("탕!")
설명: range()로 정해진 횟수만큼 반복합니다.

예시 6. len과 반복 

tools = ["망치", "모루", "집게"]
for i in range(len(tools)):
    print(i, tools[i])
설명: 인덱스와 값을 함께 다룹니다.

예시 7. 리스트 수정 

tools = ["낡은 망치", "모루"]
tools[0] = "새 망치"
print(tools)
설명: 리스트 값은 바꿀 수 있습니다.

예시 8. remove 

tools = ["망치", "줄칼", "모루"]
tools.remove("줄칼")
print(tools)
설명: 특정 값을 제거합니다.

예시 9. 리스트 합계 

strength = [10, 20, 30]
print(sum(strength))
설명: sum()은 숫자 리스트의 합을 구합니다.

예시 10. 리스트 컴프리헨션 기초 

sparks = [i * 2 for i in range(5)]
print(sparks)
설명: 반복으로 새 리스트를 간단히 만듭니다.
응용 미션: 조 가저리의 대장간 장면을 바탕으로 변수 이름, 인물 이름, 문장을 바꾸어 자신만의 짧은 파이썬 이야기 코드를 만들어 보세요.

5단계. 런던으로 가는 핍 — 딕셔너리, 튜플, 세트

이야기 흐름
핍은 신사가 되기 위해 런던으로 떠납니다. 인물 정보와 관계를 구조화하며 자료구조를 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 딕셔너리 생성 

pip = {"이름": "핍", "목표": "신사"}
print(pip)
설명: 키와 값으로 데이터를 저장합니다.

예시 2. 딕셔너리 값 꺼내기 

pip = {"이름": "핍", "나이": 17}
print(pip["이름"])
설명: 키를 사용해 값을 꺼냅니다.

예시 3. 값 추가 

pip = {"이름": "핍"}
pip["도시"] = "런던"
print(pip)
설명: 새로운 키와 값을 추가합니다.

예시 4. 값 수정 

pip = {"마음": "순수"}
pip["마음"] = "허영"
print(pip)
설명: 기존 키의 값을 바꿉니다.

예시 5. items 반복 

pip = {"이름": "핍", "목표": "신사"}
for key, value in pip.items():
    print(key, value)
설명: 딕셔너리의 키와 값을 함께 반복합니다.

예시 6. get 사용 

pip = {"이름": "핍"}
print(pip.get("재산", "아직 모름"))
설명: get()은 기본값을 지정할 수 있습니다.

예시 7. 튜플 

location = ("런던", 1861)
print(location[0])
설명: 튜플은 변경하기 어려운 묶음 데이터입니다.

예시 8. 세트 

friends = {"조", "비디", "조"}
print(friends)
설명: 세트는 중복을 제거합니다.

예시 9. 세트 추가 

friends = {"조"}
friends.add("허버트")
print(friends)
설명: add()로 세트에 값을 추가합니다.

예시 10. 딕셔너리 안 리스트 

life = {"도움": ["조", "비디", "매그위치"]}
print(life["도움"][2])
설명: 자료구조를 중첩할 수 있습니다.
응용 미션: 런던으로 가는 핍 장면을 바탕으로 변수 이름, 인물 이름, 문장을 바꾸어 자신만의 짧은 파이썬 이야기 코드를 만들어 보세요.

6단계. 신사 수업 — 함수와 매개변수

이야기 흐름
핍은 말투, 예절, 독서, 사교를 배웁니다. 반복되는 행동을 함수로 정리하는 법을 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 함수 정의 

def greet():
    print("안녕하세요, 저는 핍입니다.")

greet()
설명: def로 함수를 만듭니다.

예시 2. 매개변수 

def greet(name):
    print(f"{name}님, 런던에 오신 것을 환영합니다.")

greet("핍")
설명: 함수에 값을 전달합니다.

예시 3. return 

def add_coin(a, b):
    return a + b

print(add_coin(3, 5))
설명: return은 결과를 돌려줍니다.

예시 4. 기본값 매개변수 

def lesson(name, topic="예절"):
    print(f"{name}은 {topic}을 배운다.")

lesson("핍")
설명: 값을 생략하면 기본값을 사용합니다.

예시 5. 여러 반환값 

def status():
    return "핍", "런던"

name, city = status()
print(name, city)
설명: 여러 값을 한 번에 반환할 수 있습니다.

예시 6. 지역 변수 

def room():
    candle = "촛불"
    print(candle)

room()
설명: 함수 안에서 만든 변수는 함수 안에서 주로 사용됩니다.

예시 7. 전역 변수 읽기 

city = "런던"
def show_city():
    print(city)

show_city()
설명: 함수 밖 변수도 읽을 수 있습니다.

예시 8. 리스트를 함수에 전달 

def show(items):
    for item in items:
        print(item)

show(["책", "모자", "장갑"])
설명: 리스트도 함수에 전달할 수 있습니다.

예시 9. 조건이 있는 함수 

def judge(money):
    if money > 100:
        return "부자"
    return "평범"

print(judge(150))
설명: 함수 안에서 조건문을 사용할 수 있습니다.

예시 10. 문자열 처리 함수 

def clean_name(name):
    return name.strip().title()

print(clean_name(" pip "))
설명: 문자열 메서드를 함수로 묶습니다.
응용 미션: 신사 수업 장면을 바탕으로 변수 이름, 인물 이름, 문장을 바꾸어 자신만의 짧은 파이썬 이야기 코드를 만들어 보세요.

7단계. 뜻밖의 후원자 — 파일, 예외 처리, 모듈

이야기 흐름
핍은 후원자의 정체를 알지 못한 채 돈과 편지를 받습니다. 파일과 오류를 다루는 법을 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 파일 쓰기 

with open("letter.txt", "w", encoding="utf-8") as f:
    f.write("핍에게: 런던에서 배워라.")
설명: with open으로 파일을 안전하게 씁니다.

예시 2. 파일 읽기 

with open("letter.txt", "r", encoding="utf-8") as f:
    print(f.read())
설명: 파일 내용을 읽습니다.

예시 3. 여러 줄 쓰기 

lines = ["첫 번째 편지\n", "두 번째 편지\n"]
with open("letters.txt", "w", encoding="utf-8") as f:
    f.writelines(lines)
설명: writelines()로 여러 줄을 저장합니다.

예시 4. try-except 

try:
    print(10 / 0)
except ZeroDivisionError:
    print("0으로 나눌 수 없습니다.")
설명: 오류가 발생해도 프로그램이 멈추지 않게 합니다.

예시 5. 파일 없음 처리 

try:
    with open("unknown.txt", "r", encoding="utf-8") as f:
        print(f.read())
except FileNotFoundError:
    print("편지를 찾을 수 없습니다.")
설명: 파일이 없을 때 예외를 처리합니다.

예시 6. finally 

try:
    number = int("100")
except ValueError:
    print("변환 실패")
finally:
    print("확인은 끝났다.")
설명: finally는 마지막에 실행됩니다.

예시 7. import math 

import math
print(math.sqrt(49))

8단계. 매그위치의 귀환 — 클래스와 객체

이야기 흐름
폭풍우 치는 밤, 후원자의 정체가 드러납니다. 인물과 행동을 객체로 표현하는 OOP를 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 클래스 만들기 

class Person:
    pass

pip = Person()
print(type(pip))
설명: class로 새로운 자료형을 만듭니다.

예시 2. 생성자 

class Person:
    def __init__(self, name):
        self.name = name

pip = Person("핍")
print(pip.name)
설명: __init__은 객체가 만들어질 때 실행됩니다.

예시 3. 메서드 

class Person:
    def __init__(self, name):
        self.name = name
    def speak(self):
        print(f"{self.name}이 말한다.")

Person("핍").speak()
설명: 클래스 안 함수는 메서드입니다.

예시 4. 속성 변경 

class Person:
    def __init__(self, name, mood):
        self.name = name
        self.mood = mood

pip = Person("핍", "혼란")
pip.mood = "깨달음"
print(pip.mood)
설명: 객체 속성은 바꿀 수 있습니다.

예시 5. 여러 객체 

class Person:
    def __init__(self, name):
        self.name = name

people = [Person("핍"), Person("조"), Person("매그위치")]
for p in people:
    print(p.name)
설명: 같은 클래스로 여러 객체를 만들 수 있습니다.

예시 6. 문자열 표현 

class Person:
    def __init__(self, name):
        self.name = name
    def __str__(self):
        return f"인물: {self.name}"

print(Person("에스텔라"))
설명: __str__은 객체 출력 방식을 정합니다.

예시 7. 상속 

class Person:
    def speak(self):

class Gentleman(Person):
    pass

Gentleman().speak()
설명: 자식 클래스는 부모 기능을 물려받습니다.

예시 8. 메서드 오버라이딩

예시 9. 캡슐화 느낌

9단계. 도망 계획과 추적 — 리스트 컴프리헨션, 람다, 정렬

이야기 흐름
핍은 매그위치를 구하려고 계획을 세웁니다. 데이터를 빠르게 가공하고 정렬하는 응용 문법을 배웁니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 조건 리스트 컴프리헨션 

debts = [5, 20, 100, 3]
big = [d for d in debts if d >= 20]
print(big)
설명: 조건에 맞는 값만 새 리스트에 담습니다.

예시 2. 값 변환 

coins = [1, 2, 3]
doubled = [c * 2 for c in coins]
print(doubled)
설명: 각 값을 변환합니다.

예시 3. 람다 

add = lambda a, b: a + b
print(add(3, 4))
설명: lambda는 짧은 익명 함수입니다.

예시 4. sorted 

names = ["Pip", "Joe", "Magwitch"]
print(sorted(names))
설명: sorted()는 정렬된 새 리스트를 만듭니다.

예시 5. key 정렬 

people = [("핍", 17), ("조", 35), ("에스텔라", 18)]
print(sorted(people, key=lambda x: x[1]))
설명: key로 정렬 기준을 지정합니다.

예시 6. reverse 정렬 

money = [10, 200, 50]
print(sorted(money, reverse=True))
설명: 큰 값부터 정렬합니다.

예시 7. map 

nums = [1, 2, 3]
print(list(map(lambda x: x * 10, nums)))
설명: map은 각 요소에 함수를 적용합니다.

예시 8. filter 

nums = [1, 2, 3, 4, 5]
print(list(filter(lambda x: x % 2 == 0, nums)))
설명: filter는 조건에 맞는 값만 남깁니다.

예시 9. zip 

names = ["핍", "조"]
roles = ["학생", "대장장이"]
print(list(zip(names, roles)))
설명: zip은 여러 리스트를 짝지어 줍니다.

예시 10. enumerate

응용 미션: 도망 계획과 추적 장면을 바탕으로 변수 이름, 인물 이름, 문장을 바꾸어 자신만의 짧은 파이썬 이야기 코드를 만들어 보세요.

10단계. 진짜 위대한 유산 — 종합 프로젝트

이야기 흐름
핍은 돈보다 사랑, 용서, 성숙이 진짜 유산임을 깨닫습니다. 지금까지 배운 문법을 종합해 작은 프로그램을 만듭니다.

학습 목표 이야기 속 인물과 사건을 데이터로 표현하며 파이썬 문법을 익힙니다.

예시 1. 인물 카드 만들기 

characters = [{"name": "핍", "lesson": "성장"}, {"name": "조", "lesson": "사랑"}]
for c in characters:
    print(f"{c['name']}의 유산: {c['lesson']}")
설명: 딕셔너리 리스트를 반복합니다.

예시 2. 점수 계산 프로젝트 

scores = {"용기": 80, "겸손": 60, "사랑": 100}
avg = sum(scores.values()) / len(scores)
print(round(avg, 1))
설명: 딕셔너리 값으로 평균을 구합니다.

예시 3. 감정 분석 미니 

text = "핍은 조에게 미안했고 사랑을 깨달았다"
keywords = ["미안", "사랑", "깨달음"]
count = sum(1 for k in keywords if k in text)
print(count)
설명: 문자열 포함 여부로 간단히 분석합니다.

예시 4. 선택형 이야기 

choice = "조"
if choice == "조":
    print("핍은 진정한 사랑을 배운다.")
elif choice == "돈":
    print("핍은 허영에 빠진다.")
else:
    print("새로운 길을 찾는다.")
설명: 조건문으로 이야기 흐름을 만듭니다.

예시 5. 함수형 요약기 

def summarize(name, lesson):
    return f"{name}은 {lesson}을 배웠다."
print(summarize("핍", "겸손"))
설명: 함수로 문장 생성을 자동화합니다.

예시 6. 클래스 프로젝트 

class Legacy:
    def __init__(self, owner, value):
        self.owner = owner
        self.value = value
    def message(self):
        return f"{self.owner}의 진짜 유산은 {self.value}이다."

print(Legacy("핍", "사랑과 용서").message())
설명: 객체로 핵심 메시지를 표현합니다.

예시 7. 파일 저장 프로젝트 

result = "핍의 진짜 유산: 사랑과 성숙"
with open("legacy.txt", "w", encoding="utf-8") as f:
    f.write(result)
print("저장 완료")
설명: 결과를 파일로 저장합니다.
