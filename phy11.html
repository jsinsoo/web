정리 1

뉴턴의 제2법칙

① 정리 설명
물체에 작용하는 알짜힘 \(\vec F\)는 질량 \(m\)과 가속도 \(\vec a\)의 곱이다. \[\vec F=m\vec a\]
② 증명 핵심 아이디어
  1. 힘은 물체의 운동 상태를 변화시키는 원인이다.
  2. 같은 힘을 가하면 질량이 큰 물체일수록 가속도가 작다.
  3. 같은 질량에서는 힘이 클수록 가속도가 크다.
  4. 따라서 가속도는 힘에 비례하고 질량에 반비례한다.
  5. 이를 정리하면 \(\vec F=m\vec a\)가 된다.
③ 활용 예제
자동차 가속, 엘리베이터 운동, 로켓 발사, 충돌 분석, 기계 설계에 사용된다.
④ 파이썬 코드
# 뉴턴의 제2법칙: F = m*a
m = 5       # 질량 kg
a = 3       # 가속도 m/s^2

F = m * a

print("힘 F =", F, "N")
⑤ 코드 상세 설명
  1. `m = 5` : 물체의 질량을 kg 단위로 저장한다.
  2. `a = 3` : 물체의 가속도를 \(m/s^2\) 단위로 저장한다.
  3. `F = m * a` : 뉴턴의 제2법칙 \(F=ma\)를 계산한다.
  4. `print()` : 힘의 크기를 뉴턴 단위로 출력한다.
정리 2

일-에너지 정리

① 정리 설명
물체에 한 알짜일 \(W\)는 운동에너지 변화량과 같다. \[W=\Delta K=\frac12mv^2-\frac12mv_0^2\]
② 증명 핵심 아이디어
  1. 일은 힘과 이동거리의 곱으로 정의된다.
  2. 일차원 운동에서 \(W=Fs\)이다.
  3. 뉴턴의 제2법칙에 의해 \(F=ma\)이다.
  4. 등가속도 운동식 \(v^2-v_0^2=2as\)를 사용한다.
  5. 따라서 \(W=mas=m\frac{v^2-v_0^2}{2}=\frac12mv^2-\frac12mv_0^2\)이다.
③ 활용 예제
자동차 제동거리, 낙하 운동, 놀이기구 속도, 기계 에너지 계산에 사용된다.
④ 파이썬 코드
# 일-에너지 정리
m = 2
v0 = 3
v = 7

delta_K = 0.5*m*v**2 - 0.5*m*v0**2

print("운동에너지 변화량:", delta_K, "J")
⑤ 코드 상세 설명
  1. `m` : 물체의 질량이다.
  2. `v0` : 처음 속도이다.
  3. `v` : 나중 속도이다.
  4. `0.5*m*v**2` : 나중 운동에너지이다.
  5. `0.5*m*v0**2` : 처음 운동에너지이다.
  6. `delta_K` : 운동에너지 변화량이며, 알짜일과 같다.
정리 3

역학적 에너지 보존 법칙

① 정리 설명
마찰이 없고 보존력만 작용하면 역학적 에너지 \(E=K+U\)는 일정하다. \[\frac12mv^2+mgh=\text{constant}\]
② 증명 핵심 아이디어
  1. 중력 같은 보존력에서는 위치에너지 \(U\)를 정의할 수 있다.
  2. 보존력이 한 일은 위치에너지 감소량과 같다.
  3. 일-에너지 정리에 의해 보존력이 한 일은 운동에너지 증가량과 같다.
  4. 따라서 운동에너지 증가량과 위치에너지 감소량이 서로 상쇄된다.
  5. 그러므로 \(K+U\)는 일정하다.
③ 활용 예제
롤러코스터, 진자 운동, 낙하 속도, 수력발전, 궤도 운동에 사용된다.
④ 파이썬 코드
# 높이 h에서 떨어진 물체의 속도
g = 9.8
h = 10

v = (2*g*h) ** 0.5

print("바닥에 도달하기 직전 속도:", v, "m/s")
⑤ 코드 상세 설명
  1. `g = 9.8` : 중력가속도이다.
  2. `h = 10` : 처음 높이이다.
  3. `mgh = 1/2 mv^2`에서 질량 \(m\)은 약분된다.
  4. `v = (2*g*h) ** 0.5` : 에너지 보존으로 속도를 구한다.
  5. `** 0.5` : 제곱근을 의미한다.
정리 4

운동량 보존 법칙

① 정리 설명
외력이 없으면 계의 전체 운동량은 보존된다. \[\sum_i m_i\vec v_i=\text{constant}\]
② 증명 핵심 아이디어
  1. 두 물체가 서로 힘을 주고받을 때 작용과 반작용은 크기가 같고 방향이 반대이다.
  2. 따라서 내부힘들의 합은 0이 된다.
  3. 뉴턴의 제2법칙을 운동량 형태로 쓰면 \(\vec F=\frac{d\vec p}{dt}\)이다.
  4. 외력이 0이면 전체 운동량의 시간변화율도 0이다.
  5. 따라서 전체 운동량은 일정하다.
③ 활용 예제
충돌 문제, 로켓 추진, 당구공 운동, 폭발 현상, 입자물리 실험에 사용된다.
④ 파이썬 코드
# 완전비탄성 충돌 후 속도
m1 = 2
v1 = 5
m2 = 3
v2 = -1

v_final = (m1*v1 + m2*v2) / (m1 + m2)

print("충돌 후 공통 속도:", v_final, "m/s")
⑤ 코드 상세 설명
  1. `m1`, `m2` : 두 물체의 질량이다.
  2. `v1`, `v2` : 충돌 전 속도이다.
  3. `m1*v1 + m2*v2` : 충돌 전 전체 운동량이다.
  4. `m1 + m2` : 붙어서 움직일 때 전체 질량이다.
  5. `v_final` : 운동량 보존으로 구한 충돌 후 속도이다.
정리 5

각운동량 보존 법칙

① 정리 설명
외부 토크가 0이면 계의 전체 각운동량은 보존된다. \[\vec L=I\vec\omega=\text{constant}\]
② 증명 핵심 아이디어
  1. 회전 운동에서 토크는 각운동량을 변화시키는 원인이다.
  2. 관계식은 \(\vec\tau=\frac{d\vec L}{dt}\)이다.
  3. 외부 토크 \(\vec\tau\)가 0이면 \(\frac{d\vec L}{dt}=0\)이다.
  4. 따라서 각운동량 \(\vec L\)은 시간에 따라 변하지 않는다.
  5. 관성모멘트 \(I\)가 작아지면 각속도 \(\omega\)가 커진다.
③ 활용 예제
피겨스케이팅 회전, 행성 궤도, 자이로스코프, 회전 운동 분석에 사용된다.
④ 파이썬 코드
# 관성모멘트가 줄어들 때 각속도 증가
I1 = 4
omega1 = 2
I2 = 1

omega2 = I1 * omega1 / I2

print("나중 각속도:", omega2, "rad/s")
⑤ 코드 상세 설명
  1. `I1` : 처음 관성모멘트이다.
  2. `omega1` : 처음 각속도이다.
  3. `I2` : 나중 관성모멘트이다.
  4. `I1 * omega1` : 처음 각운동량이다.
  5. `omega2 = I1 * omega1 / I2` : 각운동량 보존으로 나중 각속도를 구한다.
정리 6

훅의 법칙

① 정리 설명
스프링의 복원력은 변위에 비례하고 방향은 반대이다. \[F=-kx\]
② 증명 핵심 아이디어
  1. 작은 변형 범위에서는 물체의 변형과 복원력이 거의 비례한다.
  2. 스프링을 더 많이 늘릴수록 더 큰 복원력이 생긴다.
  3. 복원력은 항상 평형 위치로 되돌리려는 방향이다.
  4. 따라서 변위 \(x\)와 반대 방향이므로 음의 부호가 붙는다.
  5. 비례상수 \(k\)를 스프링 상수라고 한다.
③ 활용 예제
스프링 저울, 자동차 서스펜션, 진동 분석, 탄성체 모델링에 사용된다.
④ 파이썬 코드
# 훅의 법칙
k = 100     # N/m
x = 0.05    # m

F = -k * x

print("복원력:", F, "N")
⑤ 코드 상세 설명
  1. `k` : 스프링 상수이다.
  2. `x` : 평형 위치에서 늘어난 길이이다.
  3. `F = -k * x` : 훅의 법칙을 계산한다.
  4. 음수 부호는 힘이 변위와 반대 방향임을 의미한다.
정리 7

단진동 주기 공식

① 정리 설명
질량 \(m\)이 스프링 상수 \(k\)인 스프링에 매달려 단진동할 때 주기는 \[T=2\pi\sqrt{\frac{m}{k}}\] 이다.
② 증명 핵심 아이디어
  1. 훅의 법칙에 의해 힘은 \(F=-kx\)이다.
  2. 뉴턴의 제2법칙 \(F=ma\)를 쓰면 \(m\frac{d^2x}{dt^2}=-kx\)이다.
  3. 정리하면 \(\frac{d^2x}{dt^2}+\frac{k}{m}x=0\)이다.
  4. 이 미분방정식의 해는 사인 또는 코사인 형태이다.
  5. 각진동수는 \(\omega=\sqrt{\frac{k}{m}}\)이고, 주기는 \(T=\frac{2\pi}{\omega}\)이다.
  6. 따라서 \(T=2\pi\sqrt{\frac{m}{k}}\)이다.
③ 활용 예제
스프링 진자, 진동계, 지진계, 자동차 충격흡수 장치, 센서 설계에 사용된다.
④ 파이썬 코드
# 스프링 단진동 주기 계산
import math

m = 2
k = 50

T = 2 * math.pi * (m / k) ** 0.5

print("주기:", T, "s")
⑤ 코드 상세 설명
  1. `import math` : 원주율 \(\pi\)를 사용하기 위해 불러온다.
  2. `m` : 진동하는 물체의 질량이다.
  3. `k` : 스프링 상수이다.
  4. `(m / k) ** 0.5` : \(\sqrt{m/k}\)를 계산한다.
  5. `T` : 단진동의 주기이다.
정리 8

케플러 제3법칙

① 정리 설명
같은 중심천체를 도는 행성의 공전주기 \(T\)와 궤도긴반지름 \(a\)는 \[T^2\propto a^3\] 를 만족한다.
② 증명 핵심 아이디어
  1. 원궤도 근사에서 중력은 구심력 역할을 한다.
  2. 만유인력은 \(F=\frac{GMm}{r^2}\)이다.
  3. 구심력은 \(F=\frac{mv^2}{r}\)이다.
  4. 두 식을 같게 두면 \(v^2=\frac{GM}{r}\)이다.
  5. 원운동에서 \(v=\frac{2\pi r}{T}\)이다.
  6. 이를 대입하면 \(T^2=\frac{4\pi^2}{GM}r^3\)이므로 \(T^2\propto r^3\)이다.
③ 활용 예제
행성 궤도, 인공위성 주기 계산, 천체물리학, 우주탐사 궤도 설계에 사용된다.
④ 파이썬 코드
# 케플러 제3법칙: 지구 기준 상대 공전주기
# a가 AU 단위이면 T^2 = a^3, T는 년 단위
a = 4       # 궤도 반지름 AU

T = (a**3) ** 0.5

print("공전주기:", T, "년")
⑤ 코드 상세 설명
  1. `a = 4` : 궤도긴반지름을 AU 단위로 둔다.
  2. `a**3` : \(a^3\)을 계산한다.
  3. `T = (a**3) ** 0.5` : \(T=\sqrt{a^3}\)을 계산한다.
  4. 태양 주위 행성에 대해 AU와 년 단위를 쓰면 \(T^2=a^3\)으로 단순화된다.
정리 9

가우스 법칙

① 정리 설명
닫힌면을 지나는 전기선속은 내부 전하량에 비례한다. \[\oint_S \vec E\cdot d\vec A=\frac{Q_{\mathrm{in}}}{\varepsilon_0}\]
② 증명 핵심 아이디어
  1. 점전하의 전기장은 거리의 제곱에 반비례한다.
  2. 전하를 중심으로 하는 구면을 생각하면 전기장은 구면 전체에서 같은 크기이고 방사형이다.
  3. 전기선속은 \(E\cdot 4\pi r^2\)이다.
  4. 쿨롱 법칙의 전기장 \(E=\frac{1}{4\pi\varepsilon_0}\frac{Q}{r^2}\)를 대입한다.
  5. 그러면 선속은 \(\frac{Q}{\varepsilon_0}\)가 된다.
  6. 더 일반적인 닫힌면에서도 같은 결과가 성립한다.
③ 활용 예제
대칭적인 전기장 계산, 구대칭·원통대칭·평면대칭 전하분포 분석에 사용된다.
④ 파이썬 코드
# 점전하를 둘러싼 구면의 전기선속
epsilon0 = 8.854e-12
Q = 1e-9

flux = Q / epsilon0

print("전기선속:", flux)
⑤ 코드 상세 설명
  1. `epsilon0` : 진공 유전율이다.
  2. `Q` : 닫힌면 내부의 전하량이다.
  3. `flux = Q / epsilon0` : 가우스 법칙으로 전기선속을 계산한다.
  4. 전기선속은 닫힌면의 모양이 아니라 내부 전하량에 의해 결정된다.
정리 10

패러데이 전자기 유도 법칙

① 정리 설명
회로에 유도되는 기전력은 자기선속의 시간 변화율의 음수이다. \[\mathcal E=-\frac{d\Phi_B}{dt}\]
② 증명 핵심 아이디어
  1. 자기선속 \(\Phi_B\)는 자기장이 회로를 얼마나 통과하는지 나타낸다.
  2. 자기선속이 시간에 따라 변하면 회로에 전기장이 유도된다.
  3. 유도 전기장은 전하를 움직여 기전력을 만든다.
  4. 렌츠 법칙에 의해 유도 전류는 자기선속의 변화를 방해하는 방향으로 흐른다.
  5. 따라서 음의 부호가 붙어 \(\mathcal E=-\frac{d\Phi_B}{dt}\)가 된다.
③ 활용 예제
발전기, 변압기, 무선충전, 전자기 브레이크, 인덕터 회로에 사용된다.
④ 파이썬 코드
# 자기선속 변화로 유도기전력 계산
phi_initial = 0.02   # Wb
phi_final = 0.08     # Wb
dt = 0.5             # s

emf = -(phi_final - phi_initial) / dt

print("유도기전력:", emf, "V")
⑤ 코드 상세 설명
  1. `phi_initial` : 처음 자기선속이다.
  2. `phi_final` : 나중 자기선속이다.
  3. `dt` : 변화가 일어난 시간이다.
  4. `(phi_final - phi_initial) / dt` : 자기선속의 변화율이다.
  5. `emf` : 패러데이 법칙에 따른 유도기전력이다.
  6. 음수 부호는 렌츠 법칙에 따른 방향을 나타낸다.

마무리 핵심 정리

고전물리학은 자연 현상을 힘, 에너지, 운동량, 장의 관점에서 설명한다. 뉴턴 법칙은 운동의 원인을, 보존법칙은 변하지 않는 물리량을, 전자기 법칙은 전기장과 자기장의 관계를 보여 준다. 파이썬 계산을 함께 사용하면 공식이 실제 수치 문제에서 어떻게 작동하는지 직접 확인할 수 있다.
