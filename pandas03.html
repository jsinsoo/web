예제 01

📊 선그래프: 월별 매출 추이

🐼 Pandas🎨 Seaborn📈 Matplotlib🧩 sns.lineplot

활용 목적: 시간 흐름에 따른 매출 변화

💻 코드

import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

df = pd.DataFrame({"월":["1월","2월","3월","4월"],"매출":[100,150,130,180]})
sns.lineplot(data=df, x="월", y="매출", marker="o")
plt.title("월별 매출 추이")
plt.show()

🔍 상세 설명

  • Pandas는 표 형태의 데이터를 DataFrame으로 만든다.
  • Seaborn은 DataFrame 열 이름을 이용해 그래프를 쉽게 그린다.
  • Matplotlib은 제목, 축 이름, 격자, 범례, 출력 같은 세부 요소를 조정한다.
  • 이 예제의 핵심 함수는 sns.lineplot이며, 목적은 시간 흐름에 따른 매출 변화이다.
