2022 개정 교육과정 정보 과세특

1회·2회·3회와 다른 성실하고 특색 있는 15세트 · 총 75문장 · 4회
구성: 각 세트는 1단원부터 5단원까지 한 문장씩 포함합니다. 앞선 1·2·3회와 표현 및 학습 장면이 겹치지 않도록 시스템 이해, 데이터 탐구, 프로그래밍 과정, 인공지능 해석, 디지털 시민성의 관점을 새롭게 구성했습니다.

01 세트

  1. 1단원 컴퓨팅 시스템

    입력 장치에서 받아들인 정보가 처리 장치와 기억 장치를 거쳐 출력되는 흐름을 실제 컴퓨팅 사례로 설명하며 시스템의 전체 동작을 정확히 이해함.

  2. 2단원 데이터

    생활 속 문제에서 필요한 자료를 스스로 정의하고 수집 기준을 세운 뒤 분석 가능한 형태로 정리하는 탐구 태도가 매우 성실함.

  3. 3단원 알고리즘과 프로그래밍

    문제를 해결하기 전에 핵심 조건과 제한 사항을 정리하고 단계별 알고리즘으로 표현하여 구현 과정의 오류를 줄이는 능력이 우수함.

  4. 4단원 인공지능

    인공지능이 정답을 외우는 것이 아니라 데이터의 특징과 규칙을 학습한다는 점을 이해하고 새로운 사례에 적용되는 원리를 설명함.

  5. 5단원 디지털 문화

    온라인 공간에서도 실제 생활과 동일한 책임이 따른다는 점을 이해하고 말과 행동이 타인에게 미치는 영향을 신중하게 고려함.

02 세트

  1. 1단원 컴퓨팅 시스템

    주기억 장치와 보조기억 장치의 역할과 속도 차이를 비교하고 프로그램 실행 과정에서 각각이 담당하는 기능을 논리적으로 설명함.

  2. 2단원 데이터

    자료를 수집한 뒤 값의 단위와 표현 형식을 통일하여 서로 다른 출처의 데이터를 공정하게 비교하는 능력이 뛰어남.

  3. 3단원 알고리즘과 프로그래밍

    해결해야 할 기능을 작은 모듈로 나누고 각 기능의 입력과 출력을 명확히 하여 프로그램의 구조를 체계적으로 설계함.

  4. 4단원 인공지능

    훈련 데이터와 시험 데이터를 구분하여 사용하는 이유를 이해하고 보지 못한 데이터에 대한 예측 성능의 중요성을 설명함.

  5. 5단원 디지털 문화

    타인의 창작물을 활용할 때 단순히 출처만 남기는 데 그치지 않고 이용 조건과 변경 가능 여부까지 꼼꼼하게 확인함.

03 세트

  1. 1단원 컴퓨팅 시스템

    컴퓨터가 이진수로 정보를 표현하는 원리를 이해하고 문자·수치·이미지가 디지털 데이터로 변환되는 과정을 구체적으로 설명함.

  2. 2단원 데이터

    수집한 데이터에서 중복과 누락을 찾아 정리하고 수정 전후의 차이를 비교하여 분석 결과의 신뢰도를 높임.

  3. 3단원 알고리즘과 프로그래밍

    순서도와 의사코드를 활용하여 복잡한 문제의 흐름을 시각적으로 표현하고 이를 실제 코드와 정확히 연결함.

  4. 4단원 인공지능

    분류 문제에서 입력 특성과 정답 범주의 관계를 파악하고 모델이 판단 기준을 형성하는 과정을 논리적으로 이해함.

  5. 5단원 디지털 문화

    개인정보가 한 번 공개되면 완전히 회수하기 어려울 수 있음을 이해하고 게시 전 공개 범위와 필요성을 스스로 점검함.

04 세트

  1. 1단원 컴퓨팅 시스템

    운영체제가 여러 프로그램의 실행 순서와 자원을 관리하는 과정을 이해하고 사용자와 하드웨어를 연결하는 역할을 설명함.

  2. 2단원 데이터

    데이터의 평균만으로는 전체 특성을 설명하기 어렵다는 점을 이해하고 중앙값과 분포를 함께 살펴 균형 있게 해석함.

  3. 3단원 알고리즘과 프로그래밍

    조건문을 단순히 나열하지 않고 우선순위와 예외 상황을 고려하여 복합적인 판단 구조를 정확하게 구현함.

  4. 4단원 인공지능

    인공지능 모델의 정확도가 높더라도 특정 집단에서 오류가 커질 수 있음을 이해하고 세부 성능을 함께 살펴봄.

  5. 5단원 디지털 문화

    온라인 갈등 상황에서 즉각적으로 반응하기보다 사실관계를 확인하고 상대를 존중하는 표현으로 의견을 전달함.

05 세트

  1. 1단원 컴퓨팅 시스템

    네트워크에서 주소 체계가 필요한 이유를 이해하고 장치의 위치와 서비스의 이름이 연결되는 과정을 체계적으로 설명함.

  2. 2단원 데이터

    자료의 수집 시점과 환경이 결과에 영향을 줄 수 있음을 인식하고 조건을 기록하여 해석의 한계를 명확히 제시함.

  3. 3단원 알고리즘과 프로그래밍

    반복 횟수가 큰 문제에서 불필요한 연산을 줄이는 방법을 탐구하고 실행 결과를 비교하여 효율성을 개선함.

  4. 4단원 인공지능

    회귀 모델의 예측값과 실제값 차이를 분석하여 모델이 어느 구간에서 강점과 한계를 보이는지 구체적으로 해석함.

  5. 5단원 디지털 문화

    가짜 정보가 감정적 표현과 자극적인 제목을 이용해 확산될 수 있음을 이해하고 여러 신뢰 가능한 자료를 교차 확인함.

  2. 2단원 데이터

    표본이 전체 집단을 충분히 대표하는지 검토하고 편향된 자료로 인한 잘못된 결론을 경계하는 태도가 돋보임.

  3. 3단원 알고리즘과 프로그래밍

  4. 4단원 인공지능

    군집화 과정에서 비슷한 특성을 가진 데이터가 묶이는 원리를 이해하고 각 집단의 공통점을 스스로 해석함.

  5. 5단원 디지털 문화

  2. 2단원 데이터

  5. 5단원 디지털 문화
