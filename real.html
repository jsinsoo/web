1. 카페 메뉴별 판매량 비교

서로 다른 항목의 크기를 비교할 때 사용합니다. 어떤 메뉴가 가장 많이 팔렸는지 한눈에 볼 수 있습니다.
import matplotlib.pyplot as plt

plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False

menus = ['아메리카노', '라떼', '녹차라떼', '에이드', '스무디']
sales = [120, 95, 55, 70, 40]

plt.figure(figsize=(8, 5))
plt.bar(menus, sales)
plt.title('카페 메뉴별 판매량')
plt.xlabel('메뉴')
plt.ylabel('판매량')
plt.show()

실제 출력 결과 그래프

핵심: 항목별 수량 비교에는 막대그래프가 가장 직관적입니다.
꺾은선그래프 Line Chart

2. 월별 전기요금 변화

시간에 따른 변화 추세를 볼 때 사용합니다. 여름철 전기요금 증가 같은 흐름을 확인할 수 있습니다.

plt.plot(months, bill, marker='o')

실제 출력 결과 그래프

3. 하루 시간 사용 비율

전체 중 각 항목이 차지하는 비율을 표현할 때 사용합니다.
import matplotlib.pyplot as plt

plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False

labels = ['수면', '학교', '공부', '식사', '운동', '여가']
hours = [8, 7, 4, 2, 1, 2]

plt.figure(figsize=(7, 7))
plt.pie(hours, labels=labels, autopct='%1.1f%%', startangle=90)
plt.title('하루 시간 사용 비율')
plt.show()

7. 과목별 점수 상관관계

여러 변수 사이의 관련성을 색의 진하기로 표현합니다.
import pandas as pd
import matplotlib.pyplot as plt

plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False

scores = pd.DataFrame({
    '국어': [80, 75, 90, 60, 85],
    '영어': [78, 70, 88, 65, 82],
    '수학': [92, 85, 95, 70, 88],
    '과학': [90, 82, 96, 72, 86]
})

corr = scores.corr()

plt.figure(figsize=(7, 5))
plt.imshow(corr)
plt.colorbar()
plt.xticks(range(len(corr.columns)), corr.columns)
plt.yticks(range(len(corr.columns)), corr.columns)
plt.title('과목별 점수 상관관계')
plt.show()

실제 출력 결과 그래프

핵심: 상관계수가 1에 가까울수록 두 과목의 점수가 함께 움직이는 경향이 큽니다.
누적 막대그래프 Stacked Bar

8. 월별 지출 구성 비교

월별 총지출과 항목별 구성 비율을 동시에 볼 수 있습니다.
import matplotlib.pyplot as plt

plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False

months = ['3월', '4월', '5월']
food = [250000, 270000, 260000]
transport = [60000, 65000, 70000]
culture = [80000, 90000, 120000]

plt.figure(figsize=(8, 5))
plt.bar(months, food, label='식비')
plt.bar(months, transport, bottom=food, label='교통비')

bottom2 = [food[i] + transport[i] for i in range(len(months))]
plt.bar(months, culture, bottom=bottom2, label='문화비')

plt.title('월별 지출 구성')
plt.xlabel('월')
plt.ylabel('금액(원)')
plt.legend()
plt.show()

실제 출력 결과 그래프

핵심: 누적 막대그래프는 전체 크기와 세부 구성을 함께 보여 줍니다.
영역그래프 Area Chart

9. 누적 저축액 변화

시간이 지나면서 누적되는 양의 변화를 강조할 때 사용합니다.
import matplotlib.pyplot as plt

plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False

months = ['1월', '2월', '3월', '4월', '5월', '6월']
savings = [100000, 220000, 350000, 500000, 680000, 850000]

plt.figure(figsize=(8, 5))
plt.fill_between(months, savings, alpha=0.4)
plt.plot(months, savings, marker='o')
plt.title('누적 저축액 변화')
plt.xlabel('월')
plt.ylabel('누적 저축액(원)')
plt.show()

실제 출력 결과 그래프

핵심: 영역그래프는 증가량이 쌓여 가는 느낌을 시각적으로 강조합니다.
버블차트 Bubble Chart

10. 도시별 인구·소득·관광객 비교

x축, y축, 원의 크기까지 사용하여 세 가지 정보를 동시에 표현합니다.
import matplotlib.pyplot as plt

plt.rcParams['font.family'] = 'Malgun Gothic'
plt.rcParams['axes.unicode_minus'] = False

cities = ['서울', '부산', '전주', '제주', '대전']
population = [950, 330, 65, 68, 145]      # 단위: 만 명
income = [4200, 3600, 3300, 3500, 3700]   # 단위: 만 원
tourists = [1200, 800, 500, 1500, 300]    # 단위: 만 명

bubble_size = [t * 0.5 for t in tourists]

plt.figure(figsize=(8, 5))
plt.scatter(population, income, s=bubble_size, alpha=0.5)

for i, city in enumerate(cities):
    plt.text(population[i], income[i], city)

plt.title('도시별 인구, 소득, 관광객 비교')
plt.xlabel('인구(만 명)')
plt.ylabel('평균 소득(만 원)')
plt.show()

실제 출력 결과 그래프

핵심: 버블차트는 두 변수의 관계와 세 번째 변수의 크기를 동시에 표현할 수 있습니다.
