변수와 출력: 작은 이름표가 희망이 되다

이야기
산골 마을 도서관에는 낡은 책들이 많았습니다. 어느 날, 민재는 책마다 이름표를 붙였습니다. 이름표가 생기자 사람들은 책을 더 쉽게 찾았고, 도서관은 다시 활기를 찾았습니다. 파이썬의 변수도 값에 붙이는 이름표입니다.
핵심 문법
변수는 데이터를 저장하는 이름입니다. print()는 값을 화면에 출력합니다.
book = "어린 왕자"
owner = "민재"
message = owner + "가 " + book + "에 이름표를 붙였습니다."

print(book)
print(owner)
print(message)
어린 왕자 민재 민재가 어린 왕자에 이름표를 붙였습니다.
응용 미션
자신의 이름, 좋아하는 책, 오늘의 다짐을 변수에 저장하고 세 줄로 출력해 보세요.
확인 문제
변수 school = "전북과학고"를 출력하려면 어떤 코드를 써야 할까요?

도서관 모금함에는 사람들의 작은 정성이 모였습니다. 민재는 사람들이 입력한 금액을 모아 새 책을 사기로 했습니다.
핵심 문법

조건문: 비 오는 날 우산을 나누다

이야기
비가 오는 날, 민재는 우산이 없는 친구에게 자신의 우산을 건넸습니다. 상황에 따라 선택하는 마음, 그것이 조건문입니다.
핵심 문법
if, elif, else로 조건에 따라 다른 코드를 실행합니다.
rain = True
umbrella = 1

if rain and umbrella > 0:
    print("우산을 나눠 씁니다.")
elif rain:
    print("잠시 비를 피합니다.")
else:
    print("맑은 길을 걷습니다.")
우산을 나눠 씁니다.
응용 미션
점수를 입력받아 90점 이상 A, 80점 이상 B, 그 외 C를 출력하세요.

5단계 · 응용

반복문: 매일 한 줄의 편지가 기적을 만들다

이야기
아픈 친구를 위해 반 친구들은 매일 응원 편지를 썼습니다. 하루하루 반복된 작은 문장이 친구에게 큰 힘이 되었습니다.
핵심 문법
for는 정해진 횟수나 자료를 순회하고, while은 조건이 참인 동안 반복합니다.
messages = ["괜찮아", "힘내", "우리가 함께해"]

for msg in messages:
    print("편지:", msg)

count = 3
while count > 0:
    print("응원", count)
    count -= 1
편지: 괜찮아 편지: 힘내 편지: 우리가 함께해 응원 3 응원 2 응원 1
응용 미션

딕셔너리와 집합: 사람마다 빛나는 이름이 있다

이야기
선생님은 학생들의 장점을 하나씩 기록했습니다. 이름과 장점이 연결된 기록은 딕셔너리, 중복 없는 장점 목록은 집합입니다.
핵심 문법
딕셔너리는 키와 값의 쌍입니다. 집합은 중복을 제거합니다.
strengths = {

}
words = ["배려", "끈기", "배려", "용기"]
unique_words = set(words)

print(strengths["지아"])
print(unique_words)

함수: 따뜻한 일을 다시 사용할 수 있게 하다

이야기
마을 사람들은 누군가 힘들 때마다 같은 방식으로 위로했습니다. 좋은 행동을 하나의 함수로 만들면 필요할 때마다 다시 사용할 수 있습니다.
핵심 문법
함수는 입력을 받아 처리하고 결과를 반환하는 코드 묶음입니다.
def cheer(name, count=3):
    result = []
    for i in range(count):
        result.append(name + "아, 넌 할 수 있어!")
    return result

for line in cheer("민재", 2):
    print(line)
민재아, 넌 할 수 있어! 민재아, 넌 할 수 있어!
응용 미션
두 수와 연산자(+, -, *, /)를 받아 계산 결과를 반환하는 함수를 만드세요.

파일 처리와 예외 처리: 사라지지 않는 마음의 기록

이야기
민재는 도서관을 도운 사람들의 이름을 파일에 남겼습니다. 실수로 파일이 없어도 당황하지 않고 다시 만들었습니다.
핵심 문법
open()으로 파일을 읽고 씁니다. try-except는 오류를 안전하게 처리합니다.
try:
    with open("thanks.txt", "w", encoding="utf-8") as f:
        f.write("고마운 사람들\n")
        f.write("민재\n지아\n하준\n")

    with open("thanks.txt", "r", encoding="utf-8") as f:
        print(f.read())
except FileNotFoundError:
    print("파일을 찾을 수 없습니다.")
고마운 사람들 민재 지아 하준
응용 미션

10단계 · 심화 프로젝트

클래스와 프로젝트: 마을 도서관 시스템 만들기

이야기
작은 도서관은 이제 마을의 중심이 되었습니다. 민재는 책, 사람, 대출 기록을 하나의 프로그램으로 관리하기로 했습니다. 클래스는 현실의 대상을 코드로 표현하는 방법입니다.
핵심 문법
클래스는 속성과 기능을 가진 객체를 만드는 설계도입니다.
class Book:
    def __init__(self, title, writer):
        self.title = title
        self.writer = writer
        self.borrowed = False

    def borrow(self):
        if not self.borrowed:
            self.borrowed = True
            return self.title + " 대출 완료"
        return self.title + "은 이미 대출 중"

    def info(self):
        state = "대출 중" if self.borrowed else "대출 가능"
        return f"{self.title} / {self.writer} / {state}"

book1 = Book("어린 왕자", "생텍쥐페리")
print(book1.info())
print(book1.borrow())
print(book1.info())
어린 왕자 / 생텍쥐페리 / 대출 가능 어린 왕자 대출 완료 어린 왕자 / 생텍쥐페리 / 대출 중
최종 프로젝트
도서관 관리 프로그램을 확장해 보세요. 책 추가, 책 검색, 대출, 반납, 전체 목록 출력 기능을 함수 또는 클래스로 구현합니다.
