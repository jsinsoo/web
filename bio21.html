개념 1

생체분자

① 핵심 개념
생명체를 이루는 주요 생체분자는 탄수화물, 지질, 단백질, 핵산이다.
② 개념 상세 설명
  1. 탄수화물은 주로 에너지원과 구조 물질로 사용된다.
  2. 지질은 세포막 구성, 에너지 저장, 호르몬 생성에 관여한다.
  3. 단백질은 효소, 수송체, 구조 단백질, 신호 단백질로 작용한다.
  4. 핵산은 유전 정보를 저장하고 전달한다.
  5. 생화학은 이러한 분자들이 생명현상을 어떻게 만드는지 연구한다.
③ 활용 예제
영양소 분석, 의학 검사, 세포 기능 이해, 생명공학 기초에 활용된다.
④ 파이썬 코드
# 생체분자 목록 분류
biomolecules = ["탄수화물", "지질", "단백질", "핵산"]

for i, name in enumerate(biomolecules, 1):
    print(i, name)
⑤ 코드 상세 설명
  1. `biomolecules`는 주요 생체분자 이름을 저장한 리스트이다.
  2. `enumerate(..., 1)`은 번호를 1부터 붙인다.
  3. `for`문은 각 생체분자를 하나씩 출력한다.
  4. 생화학의 기본 출발점은 생체분자의 종류와 기능을 구분하는 것이다.
개념 2

아미노산

① 핵심 개념
아미노산은 아미노기 \(-NH_2\), 카복실기 \(-COOH\), 곁사슬 \(R\)을 가진 단백질의 기본 단위이다.
② 개념 상세 설명
  1. 아미노산은 단백질을 구성하는 단량체이다.
  2. 중심 탄소에는 아미노기, 카복실기, 수소, 곁사슬이 결합한다.
  3. 곁사슬의 성질에 따라 아미노산의 극성, 전하, 소수성이 달라진다.
  4. 아미노산은 펩타이드 결합으로 연결되어 단백질을 만든다.
  5. 생명체는 주로 20종의 표준 아미노산을 사용한다.
③ 활용 예제
단백질 구조 예측, 영양학, 효소 기능 이해, 질병 돌연변이 분석에 활용된다.
④ 파이썬 코드
# 아미노산 개수 세기
sequence = "MKTLLV"
length = len(sequence)

print("아미노산 서열:", sequence)
print("아미노산 개수:", length)
⑤ 코드 상세 설명
  1. `sequence`는 아미노산 서열을 한 글자 코드로 표현한 문자열이다.
  2. `len(sequence)`는 서열의 길이, 즉 아미노산 개수를 구한다.
  3. 단백질은 아미노산이 순서대로 연결된 고분자이다.
개념 3

펩타이드 결합

① 핵심 개념
두 아미노산은 탈수축합 반응으로 펩타이드 결합을 형성한다.
② 개념 상세 설명
  1. 한 아미노산의 카복실기와 다른 아미노산의 아미노기가 반응한다.
  2. 이때 물 한 분자가 빠져나가는 탈수축합이 일어난다.
  3. 형성된 결합을 펩타이드 결합이라고 한다.
  4. 여러 아미노산이 펩타이드 결합으로 연결되면 폴리펩타이드가 된다.
  5. 단백질의 1차 구조는 아미노산의 연결 순서이다.
③ 활용 예제
단백질 합성, 리보솜 작용, 단백질 분해효소 이해에 활용된다.
④ 파이썬 코드
# 펩타이드 결합 수 계산
amino_acids = 8
peptide_bonds = amino_acids - 1

print("아미노산 수:", amino_acids)
print("펩타이드 결합 수:", peptide_bonds)
⑤ 코드 상세 설명
  1. `amino_acids`는 연결된 아미노산 개수이다.
  2. 선형 펩타이드에서 결합 수는 아미노산 수보다 1개 적다.
  3. `peptide_bonds = amino_acids - 1`로 펩타이드 결합 수를 계산한다.
개념 4

단백질 구조

① 핵심 개념
단백질 구조는 1차, 2차, 3차, 4차 구조로 설명할 수 있다.
② 개념 상세 설명
  1. 1차 구조는 아미노산 서열이다.
  2. 2차 구조는 알파 나선과 베타 병풍 같은 국소 구조이다.
  3. 3차 구조는 하나의 폴리펩타이드 전체 입체 구조이다.
  4. 4차 구조는 여러 폴리펩타이드 소단위체가 모인 구조이다.
  5. 단백질 구조는 기능과 밀접하게 연결된다.
③ 활용 예제
효소 활성, 항체 결합, 단백질 변성, 신약 개발에 활용된다.
④ 파이썬 코드
# 단백질 서열에서 소수성 아미노산 비율 계산
sequence = "MALWMRLLPLL"
hydrophobic = set("AILMFWV")

count = sum(1 for aa in sequence if aa in hydrophobic)
ratio = count / len(sequence)

print("소수성 아미노산 비율:", ratio)
⑤ 코드 상세 설명
  1. `sequence`는 단백질 아미노산 서열이다.
  2. `hydrophobic`은 소수성 아미노산 집합이다.
  3. `sum(1 for aa in sequence if aa in hydrophobic)`은 소수성 아미노산 개수를 센다.
  4. 소수성 아미노산은 단백질 접힘에 큰 영향을 준다.
개념 5

효소

① 핵심 개념
효소는 활성화 에너지 \(E_a\)를 낮추어 반응 속도를 증가시키는 생체 촉매이다.
② 개념 상세 설명
  1. 효소는 대부분 단백질로 이루어진 촉매이다.
  2. 효소는 반응 전후에 소모되지 않는다.
  3. 효소는 특정 기질과 결합하는 활성부위를 가진다.
  4. 효소는 반응의 활성화 에너지를 낮춘다.
  5. 온도와 pH가 지나치게 변하면 효소가 변성될 수 있다.
③ 활용 예제
소화, 대사, 의약품 개발, 산업용 효소, 질병 진단에 활용된다.
④ 파이썬 코드
# 효소가 반응속도를 증가시키는 단순 모델
rate_without = 1
catalytic_factor = 1000

rate_with = rate_without * catalytic_factor

print("효소 없는 속도:", rate_without)
print("효소 있는 속도:", rate_with)
⑤ 코드 상세 설명
  1. `rate_without`은 효소가 없을 때의 반응속도이다.
  2. `catalytic_factor`는 효소에 의한 속도 증가 배율이다.
  3. `rate_with`는 효소가 있을 때의 반응속도이다.
  4. 효소는 반응을 빠르게 하지만 평형 자체를 바꾸지는 않는다.
개념 6

미하엘리스-멘텐 식

① 핵심 개념
효소 반응속도는 기질 농도 \([S]\)에 따라 \[v=\frac{V_{\max}[S]}{K_m+[S]}\] 로 근사할 수 있다.
② 개념 상세 설명
  1. 기질 농도가 낮을 때는 기질이 늘수록 반응속도가 크게 증가한다.
  2. 기질 농도가 높아지면 효소 활성부위가 포화된다.
  3. 포화 상태에서는 속도가 \(V_{\max}\)에 가까워진다.
  4. \(K_m\)은 반응속도가 \(V_{\max}/2\)가 되는 기질 농도이다.
  5. 이 식은 효소 반응의 기본 모델이다.
③ 활용 예제
효소 활성 측정, 약물 저해 분석, 생화학 실험 데이터 해석에 활용된다.
④ 파이썬 코드
# 미하엘리스-멘텐 식 계산
Vmax = 100
Km = 2
S = 5

v = Vmax * S / (Km + S)

print("반응속도:", v)
⑤ 코드 상세 설명
  1. `Vmax`는 최대 반응속도이다.
  2. `Km`은 미하엘리스 상수이다.
  3. `S`는 기질 농도이다.
  4. `v = Vmax * S / (Km + S)`는 미하엘리스-멘텐 식이다.
  5. 기질 농도가 높아질수록 속도는 \(V_{\max}\)에 가까워진다.
개념 7

ATP

① 핵심 개념
ATP는 세포의 에너지 화폐이며, 가수분해될 때 에너지를 방출한다. \[\mathrm{ATP}+H_2O\to \mathrm{ADP}+P_i+\text{energy}\]
② 개념 상세 설명
  1. ATP는 아데닌, 리보스, 세 개의 인산기로 이루어진다.
  2. 인산기 사이 결합이 가수분해되면 에너지가 방출된다.
  3. 세포는 이 에너지를 근육 수축, 능동수송, 생합성에 사용한다.
  4. ATP는 계속 분해되고 다시 합성된다.
  5. ATP는 에너지 저장량이 크다기보다 빠르게 쓰고 재생하기 좋은 분자이다.
③ 활용 예제
근육 운동, 세포막 능동수송, 대사 경로, 세포 에너지 계산에 활용된다.
④ 파이썬 코드
# ATP 분해로 얻는 에너지 근사
atp_count = 1000
energy_per_atp = 30.5  # kJ/mol 단순값

total_energy = atp_count * energy_per_atp

print("상대적 총 에너지:", total_energy)
⑤ 코드 상세 설명
  1. `atp_count`는 ATP 개수를 단순화한 값이다.
  2. `energy_per_atp`는 ATP 1단위당 방출 에너지로 둔 값이다.
  3. `total_energy`는 전체 에너지이다.
  4. 실제 세포에서는 ATP 양과 조건에 따라 에너지 값이 달라질 수 있다.
개념 8

해당과정

① 핵심 개념
해당과정은 포도당 \(1\)분자를 피루브산 \(2\)분자로 분해하며 ATP와 NADH를 만든다.
② 개념 상세 설명
  1. 해당과정은 세포질에서 일어난다.
  2. 산소가 없어도 진행될 수 있다.
  3. 포도당은 여러 단계의 효소 반응을 거쳐 피루브산이 된다.
  4. 순수하게 ATP 2개와 NADH 2개가 생성된다.
  5. 해당과정은 세포호흡의 첫 단계이다.
③ 활용 예제
운동 생리, 발효, 암세포 대사, 당 대사 이해에 활용된다.
④ 파이썬 코드
# 포도당 개수에 따른 해당과정 ATP 순생산
glucose = 6
net_atp_per_glucose = 2

net_atp = glucose * net_atp_per_glucose

print("순생산 ATP:", net_atp)
⑤ 코드 상세 설명
  1. `glucose`는 분해되는 포도당 분자 수이다.
  2. `net_atp_per_glucose`는 포도당 1개당 순생산 ATP이다.
  3. `net_atp`는 전체 ATP 순생산량이다.
  4. 해당과정은 포도당 1개당 ATP 2개를 순생산한다.
개념 9

TCA 회로

① 핵심 개념
TCA 회로는 아세틸-CoA를 산화하여 \(CO_2\), NADH, FADH\(_2\)를 만든다.
② 개념 상세 설명
  1. TCA 회로는 미토콘드리아 기질에서 일어난다.
  2. 아세틸-CoA가 옥살아세트산과 결합해 시트르산을 만든다.
  3. 여러 단계의 산화 반응을 통해 \(CO_2\)가 방출된다.
  4. NADH와 FADH2는 전자전달계로 전자를 운반한다.
  5. TCA 회로는 탄수화물, 지방, 단백질 대사의 중심 허브이다.
③ 활용 예제
세포호흡, 에너지 대사, 대사질환, 운동 생리학에 활용된다.
④ 파이썬 코드
# 아세틸-CoA 개수에 따른 NADH 생산
acetyl_coa = 4
nadh_per_turn = 3

nadh = acetyl_coa * nadh_per_turn

print("생성 NADH:", nadh)
⑤ 코드 상세 설명
  1. `acetyl_coa`는 TCA 회로에 들어가는 아세틸-CoA 수이다.
  2. `nadh_per_turn`은 회로 1회전당 생성되는 NADH 수이다.
  3. `nadh`는 전체 NADH 생성량이다.
  4. NADH는 전자전달계에서 ATP 생산에 사용된다.
개념 10

전자전달계

① 핵심 개념
전자전달계는 NADH와 FADH\(_2\)의 전자를 이용해 양성자 기울기를 만들고 ATP를 합성한다.
② 개념 상세 설명
  1. 전자전달계는 미토콘드리아 내막에 있다.
  2. NADH와 FADH2는 고에너지 전자를 전달한다.
  3. 전자 이동 에너지는 양성자를 막 사이 공간으로 펌프질하는 데 쓰인다.
  4. 양성자 기울기는 ATP 합성효소를 돌려 ATP를 만든다.
  5. 산소는 최종 전자수용체로 물을 형성한다.
③ 활용 예제
세포 에너지 생산, 미토콘드리아 질환, 독성 물질 작용, 운동 대사에 활용된다.
④ 파이썬 코드
# NADH와 FADH2로부터 ATP 생산량 근사
NADH = 10
FADH2 = 2

ATP = NADH * 2.5 + FADH2 * 1.5

print("예상 ATP:", ATP)
⑤ 코드 상세 설명
  1. `NADH`와 `FADH2`는 전자운반체의 수이다.
  2. NADH 1개는 대략 ATP 2.5개 생산에 기여한다고 근사한다.
  3. FADH2 1개는 대략 ATP 1.5개 생산에 기여한다고 근사한다.
  4. `ATP`는 예상 ATP 생산량이다.
개념 11

산화와 환원

① 핵심 개념
산화는 전자를 잃는 과정이고 환원은 전자를 얻는 과정이다.
② 개념 상세 설명
  1. 생화학 반응에서 전자의 이동은 에너지 흐름과 연결된다.
  2. 산화되는 물질은 전자를 잃는다.
  3. 환원되는 물질은 전자를 얻는다.
  4. NAD+는 전자를 받아 NADH로 환원된다.
  5. 세포호흡은 포도당을 산화하고 산소를 환원하는 과정이다.
③ 활용 예제
세포호흡, 광합성, 항산화 반응, 대사 경로 이해에 활용된다.
④ 파이썬 코드
# NAD+가 전자를 받아 NADH가 되는 과정 표현
NAD_plus = 5
electrons_available = 3

NADH = min(NAD_plus, electrons_available)

print("생성 가능한 NADH:", NADH)
⑤ 코드 상세 설명
  1. `NAD_plus`는 전자를 받을 수 있는 NAD+ 수이다.
  2. `electrons_available`은 사용 가능한 전자쌍 수로 단순화한 값이다.
  3. `min()`은 둘 중 더 적은 값만큼 반응할 수 있음을 나타낸다.
  4. NAD+는 환원되어 NADH가 된다.
개념 12

완충용액

① 핵심 개념
완충용액은 산이나 염기가 조금 첨가되어도 pH 변화를 작게 유지한다.
② 개념 상세 설명
  1. 완충용액은 약산과 그 짝염기 또는 약염기와 그 짝산으로 이루어진다.
  2. 산이 들어오면 짝염기가 이를 받아 pH 변화를 줄인다.
  3. 염기가 들어오면 약산이 이를 중화한다.
  4. 혈액은 탄산-중탄산 완충계를 이용해 pH를 조절한다.
  5. pH 조절은 단백질 구조와 효소 기능 유지에 중요하다.
③ 활용 예제
혈액 pH 조절, 생화학 실험, 세포 배양액, 의약품 제조에 활용된다.
④ 파이썬 코드
# 헨더슨-하셀바흐 식
import math

pKa = 6.1
base = 24
acid = 1.2

pH = pKa + math.log10(base / acid)

print("pH:", pH)
⑤ 코드 상세 설명
  1. `pKa`는 약산의 산해리상수에 대응하는 값이다.
  2. `base`는 짝염기의 농도이다.
  3. `acid`는 약산의 농도이다.
  4. `math.log10(base / acid)`는 농도비의 로그이다.
  5. 헨더슨-하셀바흐 식으로 pH를 근사한다.
개념 13

pH

① 핵심 개념
pH는 수소 이온 농도 \([H^+]\)를 로그 척도로 나타낸 값이다. \[\mathrm{pH}=-\log_{10}[H^+]\]
② 개념 상세 설명
  1. pH가 낮을수록 산성이 강하다.
  2. pH가 높을수록 염기성이 강하다.
  3. pH 7은 대략 중성이다.
  4. pH는 로그 척도이므로 pH가 1 차이 나면 수소 이온 농도는 10배 차이 난다.
  5. 효소와 세포는 적절한 pH 범위에서 정상 작동한다.
③ 활용 예제
혈액 검사, 효소 활성, 식품 산도, 수질 검사에 활용된다.
④ 파이썬 코드
# 수소 이온 농도로 pH 계산
import math

H = 1e-7
pH = -math.log10(H)

print("pH:", pH)
⑤ 코드 상세 설명
  1. `H`는 수소 이온 농도이다.
  2. `math.log10(H)`는 10을 밑으로 하는 로그이다.
  3. 음수를 붙여 pH를 계산한다.
  4. 수소 이온 농도 \(10^{-7}\)이면 pH는 7이다.

탄수화물

① 핵심 개념
탄수화물은 주로 \(C\), \(H\), \(O\)로 이루어진 에너지원이며 단당류, 이당류, 다당류로 나뉜다.
② 개념 상세 설명
  1. 포도당은 대표적인 단당류이다.
  2. 설탕은 포도당과 과당으로 이루어진 이당류이다.
  3. 녹말과 글리코겐은 포도당이 많이 연결된 다당류이다.
  4. 탄수화물은 빠른 에너지원으로 사용된다.
  5. 식물의 셀룰로스는 구조 탄수화물이다.
③ 활용 예제
영양학, 혈당 조절, 운동 에너지, 식품 과학에 활용된다.
④ 파이썬 코드
# 탄수화물 칼로리 계산
grams = 50
kcal_per_gram = 4

energy = grams * kcal_per_gram

print("에너지:", energy, "kcal")
⑤ 코드 상세 설명
  1. `grams`는 탄수화물 섭취량 g이다.
  2. `kcal_per_gram`은 탄수화물 1 g당 에너지이다.
  3. `energy`는 총 에너지이다.
  4. 탄수화물은 1 g당 약 4 kcal를 제공한다.

② 개념 상세 설명
  1. 지질은 소수성이 큰 분자이다.
  2. 중성지방은 에너지 저장에 중요하다.
  3. 인지질은 세포막의 기본 구성 성분이다.
  4. 스테로이드는 호르몬과 세포막 안정성에 관여한다.
  5. 지질은 탄수화물보다 단위 질량당 더 많은 에너지를 저장한다.
③ 활용 예제
영양학, 세포막 연구, 호르몬 생합성, 대사질환 이해에 활용된다.

# 지방 칼로리 계산
grams = 20
kcal_per_gram = 9

energy = grams * kcal_per_gram

⑤ 코드 상세 설명
  1. `grams`는 지방 섭취량 g이다.
  2. `kcal_per_gram`은 지방 1 g당 에너지이다.
  3. `energy`는 총 에너지이다.
  4. 지질은 1 g당 약 9 kcal를 제공한다.

인지질 이중층

① 핵심 개념
인지질은 친수성 머리와 소수성 꼬리를 가지며, 물속에서 이중층을 형성한다.
② 개념 상세 설명
  1. 인지질의 머리 부분은 물과 잘 상호작용한다.
  2. 꼬리 부분은 물을 피하는 소수성이다.
  3. 물속에서 소수성 꼬리끼리 안쪽을 향해 모인다.
  4. 이 구조가 세포막의 기본 골격이다.
  5. 막 단백질이 함께 작용해 물질 수송과 신호 전달을 수행한다.
③ 활용 예제
세포막 구조, 약물 전달, 리포좀, 막 단백질 연구에 활용된다.

# 막 통과 가능성 단순 판단
molecule = {"size": "small", "polar": False}

can_cross = molecule["size"] == "small" and not molecule["polar"]

print("인지질 이중층 통과 가능?", can_cross)
⑤ 코드 상세 설명
  1. `molecule`은 분자 특성을 저장한 딕셔너리이다.
  2. 작고 비극성이면 막을 통과하기 쉽다.
  3. `and`는 두 조건이 모두 참인지 확인한다.
  4. `can_cross`는 막 통과 가능성을 나타낸다.
개념 17

핵산

① 핵심 개념
핵산은 뉴클레오타이드가 연결된 고분자로 DNA와 RNA가 대표적이다.
② 개념 상세 설명
  1. DNA는 유전 정보를 안정적으로 저장한다.
  2. RNA는 단백질 합성과 유전자 발현 조절에 관여한다.
  3. 뉴클레오타이드는 당, 인산, 염기로 이루어진다.
  4. DNA에는 A, T, G, C가 있고 RNA에는 A, U, G, C가 있다.
  5. 핵산 서열은 생명정보의 기본 코드이다.
③ 활용 예제
유전체 분석, PCR, RNA 백신, 유전자 발현 연구에 활용된다.
④ 파이썬 코드
# DNA를 RNA로 전사하기
dna = "ATGCTTACG"
rna = dna.replace("T", "U")

print("DNA:", dna)
print("RNA:", rna)
⑤ 코드 상세 설명
  1. `dna`는 DNA 염기서열이다.
  2. RNA에서는 T 대신 U를 사용한다.
  3. `replace('T', 'U')`는 T를 U로 바꾼다.
  4. `rna`는 전사된 RNA 서열이다.
개념 18

대사 경로

① 핵심 개념
대사 경로는 여러 효소 반응이 순서대로 연결되어 물질을 합성하거나 분해하는 과정이다.
② 개념 상세 설명
  1. 대사는 동화작용과 이화작용으로 나뉜다.
  2. 동화작용은 작은 분자를 큰 분자로 합성하며 에너지를 사용한다.
  3. 이화작용은 큰 분자를 분해하며 에너지를 방출한다.
  4. 대사 경로는 효소에 의해 정교하게 조절된다.
  5. 피드백 억제는 최종 산물이 앞 단계 효소를 억제하는 조절 방식이다.
③ 활용 예제
약물 작용, 유전 대사 질환, 영양 대사, 세포 에너지 연구에 활용된다.
④ 파이썬 코드
# 피드백 억제 단순 모델
product = 80
threshold = 50

if product > threshold:
    enzyme_activity = "억제됨"
else:
    enzyme_activity = "활성"

print("효소 상태:", enzyme_activity)
⑤ 코드 상세 설명
  1. `product`는 최종 산물의 양이다.
  2. `threshold`는 억제가 시작되는 기준값이다.
  3. 최종 산물이 많으면 효소가 억제된다.
  4. 피드백 억제는 대사 균형을 유지한다.
개념 19

막수송

① 핵심 개념
물질은 확산, 촉진확산, 능동수송을 통해 세포막을 이동한다.
② 개념 상세 설명
  1. 단순확산은 농도 기울기를 따라 이동한다.
  2. 촉진확산은 막 단백질을 이용하지만 에너지를 쓰지 않는다.
  3. 능동수송은 ATP를 사용해 농도 기울기에 거슬러 이동한다.
  4. 이온 펌프는 대표적인 능동수송이다.
  5. 막수송은 세포 내부 환경 유지에 필수적이다.
③ 활용 예제
신경세포 전위, 영양분 흡수, 신장 기능, 약물 이동 이해에 활용된다.
④ 파이썬 코드
# 농도 기울기에 따른 확산 방향
inside = 2
outside = 8

if outside > inside:
    direction = "밖에서 안으로 확산"
elif inside > outside:
    direction = "안에서 밖으로 확산"
else:
    direction = "순확산 없음"

print(direction)
⑤ 코드 상세 설명
  1. `inside`는 세포 안 농도이다.
  2. `outside`는 세포 밖 농도이다.
  3. 물질은 보통 높은 농도에서 낮은 농도로 확산한다.
  4. 조건문으로 확산 방향을 판단한다.
개념 20

신호전달

① 핵심 개념
세포 신호전달은 외부 신호가 수용체와 결합하여 세포 내부 반응으로 이어지는 과정이다.
② 개념 상세 설명
  1. 호르몬이나 신경전달물질은 신호분자로 작용한다.
  2. 수용체는 특정 신호분자를 인식한다.
  3. 신호가 결합하면 세포 내부 전달 경로가 활성화된다.
  4. 2차 전달자가 신호를 증폭할 수 있다.
  5. 신호전달 오류는 암, 당뇨, 면역질환과 관련될 수 있다.
③ 활용 예제
호르몬 작용, 약물 표적, 암 신호 경로, 면역 반응 이해에 활용된다.
④ 파이썬 코드
# 신호 증폭 단순 모델
signal = 1
amplification = 3

for step in range(5):
    signal *= amplification
    print(step + 1, signal)
⑤ 코드 상세 설명
  1. `signal`은 초기 신호 크기이다.
  2. `amplification`은 단계별 증폭 배율이다.
  3. 반복될수록 신호가 커진다.
  4. 세포 신호전달은 작은 신호를 큰 반응으로 증폭할 수 있다.

마무리 핵심 정리

생화학은 생명현상을 분자 수준에서 설명하는 과학이다. 효소는 반응을 빠르게 하고, ATP는 에너지 전달의 중심이며, 탄수화물·지질·단백질·핵산은 세포 구조와 기능을 만든다. 대사 경로와 신호전달은 세포가 환경에 반응하고 생명을 유지하는 핵심 체계이다.
