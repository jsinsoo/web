1. 입력과 출력: 두 수의 합

상세 설명

class AddAlgorithm:

3. 출력 최소 1개: 최댓값 찾기

상세 설명
리스트를 입력받아 반복문과 조건문으로 최댓값을 찾고, 결과를 하나 출력합니다.

💻 제시 코드

class MaxAlgorithm:
    def __init__(self, numbers):
        self.numbers = numbers

    def run(self):
        m = self.numbers[0]
        for n in self.numbers:
            if n > m:
                m = n
        return m

algo = MaxAlgorithm([3, 9, 2, 7])
print("최댓값:", algo.run())

5. 유한성: 정해진 횟수만 반복

상세 설명
5부터 1까지 정해진 횟수만 반복하므로 반드시 종료됩니다. 유한성은 빠르다는 뜻이 아니라 끝난다는 뜻입니다.

💻 제시 코드

class CountDown:
    def __init__(self, start):
        self.start = start

    def run(self):
        result = []
        for i in range(self.start, 0, -1):
            result.append(i)
        return result

algo = CountDown(5)
print(algo.run())
문항 9OX 퀴즈

6. 유효성: 실행 가능한 나눗셈

class DivideAlgorithm:

7. 명확한 단계: 평균 계산

상세 설명
점수 합계를 구하고 개수로 나누는 과정이 명확합니다. 평균 계산 과정은 누구나 같은 결과로 이해할 수 있습니다.

💻 제시 코드

class AverageAlgorithm:
    def __init__(self, scores):
        self.scores = scores

    def run(self):
        total = 0
        for s in self.scores:
            total += s
        return total / len(self.scores)

algo = AverageAlgorithm([80, 90, 100])
print("평균:", algo.run())
문항 13OX 퀴즈

8. 의사코드와 실제 코드 구분

class PseudoCodeExample:
    def __init__(self, n):
        self.n = n

    def run(self):

9. 순서도는 표현 도구

상세 설명
입력, 처리, 출력 단계를 순서대로 보여 줍니다. 순서도는 이런 알고리즘 절차를 도형과 화살표로 시각화하는 표현 도구입니다.

💻 제시 코드

class FlowStep:
    def __init__(self, steps):
        self.steps = steps

    def run(self):
        for i, step in enumerate(self.steps, start=1):
            print(f"{i}단계: {step}")

algo = FlowStep(["입력", "처리", "출력"])
algo.run()
문항 17OX 퀴즈

순서도는 알고리즘 자체라기보다 알고리즘을 시각적으로 표현하는 도구이다.

10. 다섯 조건 종합 점검

상세 설명
출력, 유한성, 명확성을 간단히 점검하는 코드입니다. 실제 알고리즘은 입력, 출력, 명확성, 유한성, 유효성을 모두 고려해야 합니다.

💻 제시 코드

class AlgorithmChecker:
    def __init__(self, has_output, finite, clear):
        self.has_output = has_output
        self.finite = finite
        self.clear = clear

    def run(self):
        if self.has_output and self.finite and self.clear:
            return "알고리즘 조건 만족"
        return "보완 필요"

algo = AlgorithmChecker(True, True, True)
print(algo.run())
