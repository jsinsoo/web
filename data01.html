분산과 표준편차에 대한 핵심 개념을 데이터 과학 관점에서 정리한다.
② 핵심 수식 또는 구조

③ 개념 상세 설명
  1. 분산과 표준편차는 데이터 과학의 핵심 개념이다.

④ 활용 예제

⑤ 파이썬 코드

② 핵심 수식 또는 구조
\[r=\frac{\sum (x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum (x_i-\bar{x})^2}\sqrt{\sum (y_i-\bar{y})^2}}\]
③ 개념 상세 설명
  1. 상관관계는 데이터 과학의 핵심 개념이다.
  2. 개념을 정확히 이해하면 분석 과정의 오류를 줄일 수 있다.
  3. 실제 데이터 문제에서는 개념, 통계, 코딩, 해석이 함께 필요하다.
  4. 분석 결과는 항상 문제 상황과 연결해 해석해야 한다.
④ 활용 예제
학교 데이터 분석, 공공데이터 활용, 과학 실험 결과 분석, 인공지능 모델 평가에 활용된다.
⑤ 파이썬 코드
import math
x = [1, 2, 3, 4, 5]
y = [2, 4, 5, 4, 6]
xb = sum(x)/len(x); yb = sum(y)/len(y)
num = sum((a-xb)*(b-yb) for a,b in zip(x,y))
den = math.sqrt(sum((a-xb)**2 for a in x) * sum((b-yb)**2 for b in y))
print('상관계수:', num/den)
⑥ 코드 상세 설명
  1. 코드는 상관관계 개념을 간단한 예제로 보여 준다.
  2. 리스트, 반복문, 조건문, 수식 계산을 이용해 핵심 원리를 확인한다.
