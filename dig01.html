1. 디지털 기술 분류기

개념 상세 설명

디지털 기술은 디지털 정보를 처리하고 저장하며 전송하는 장치와 시스템을 연구·개발하는 기술입니다. 대표적으로 인공지능, 빅데이터, 사물 인터넷, 클라우드 등이 있습니다.
class DigitalTechnology:
    def __init__(self, name, description):
        self.name = name
        self.description = description

    def category(self):
        if self.name in ["인공지능", "AI"]:
            return "지능형 판단 기술"
        if self.name == "빅데이터":
            return "대량 데이터 분석 기술"
        if self.name == "사물 인터넷":
            return "초연결 기술"
        if self.name == "클라우드":
            return "인터넷 기반 자원 제공 기술"
        return "기타 디지털 기술"

tech = DigitalTechnology("빅데이터", "대량의 데이터를 분석하여 가치를 찾는 기술")
print("기술명:", tech.name)
print("분류:", tech.category())

코드 상세 설명

  • DigitalTechnology 클래스는 기술 이름과 설명을 저장합니다.
  • category()는 기술 이름에 따라 핵심 역할을 분류합니다.
  • 빅데이터는 대량 데이터 분석 기술로 분류됩니다.

2. 정보 공유 가능 여부 판단기

개념 상세 설명

정보 공유는 네트워크 환경에서 정보를 얻고 다른 사람과 나누는 것입니다. 하지만 개인 정보, 위치 정보, 의료 정보, 영상 정보 등은 보호해야 합니다.
class InformationSharingChecker:
    def __init__(self, info_type):
        self.info_type = info_type

    def can_share(self):
        shareable = ["교통 정보", "날씨 정보", "재난 정보", "버스 도착 시간", "병원 진료 시간"]

        if self.info_type in shareable:
            return "공유 가능 정보"
        if self.info_type in protected:
            return "보호 필요 정보"
        return "목적과 범위 확인 필요"

print(InformationSharingChecker("날씨 정보").can_share())

코드 상세 설명

  • protected에는 보호해야 할 정보가 들어 있습니다.

3. 개인 정보 종류 분류기

개인 정보는 살아 있는 개인에 관한 정보입니다. 직접 식별 정보뿐 아니라 다른 정보와 결합해 개인을 식별할 수 있는 정보도 포함합니다.
class PersonalInfoClassifier:
    def __init__(self, info):
        self.info = info

    def classify(self):
        categories = {
            "주민등록번호": "인적 사항",
            "집 주소": "인적 사항",
            "얼굴 사진": "신체적 정보",

5. 디지털 윤리 판단기

개념 상세 설명

디지털 윤리는 디지털 사회에서 윤리적인 것과 비윤리적인 것을 바르게 판단하는 기준입니다.
class DigitalEthicsChecker:
    def __init__(self, action):
        self.action = action

    def check(self):
        unethical_keywords = ["무단 공유", "비밀번호 요구", "허위 정보", "불법 촬영", "딥페이크 악용"]
        for keyword in unethical_keywords:
            if keyword in self.action:
                return "비윤리적 행동"
        return "윤리적 행동 또는 추가 검토 필요"

case = DigitalEthicsChecker("친구 사진을 무단 공유")
print(case.check())

코드 상세 설명

  • unethical_keywords에는 위험 행동 키워드가 들어 있습니다.
  • check()는 행동 설명을 검사합니다.
  • 무단 공유는 타인에게 피해를 줄 수 있습니다.

7. 개인 정보 보호 수칙 점검기

개념 상세 설명

개인 정보 보호를 위해 비밀번호 변경, 개인 정보 비공유, 출처 불명 자료 다운로드 금지, 자동 로그인 해제 등을 실천해야 합니다.
class PrivacyChecklist:
    def __init__(self, practices):
        self.practices = practices

    def score(self):
        good_rules = ["비밀번호 주기적 변경", "개인 정보 친구에게 알리지 않기",
                      "P2P 폴더에 개인 정보 저장하지 않기", "출처 불명 자료 다운로드하지 않기", "자동 로그인 해제"]
        count = 0
        for rule in good_rules:
            if rule in self.practices:
                count += 1
        return count

    def level(self):
        s = self.score()
        if s >= 4:
            return "우수"
        if s >= 2:
            return "보통"
        return "개선 필요"

practices = ["비밀번호 주기적 변경", "출처 불명 자료 다운로드하지 않기", "자동 로그인 해제"]
check = PrivacyChecklist(practices)
print("실천 점수:", check.score())
print("보호 수준:", check.level())

코드 상세 설명

  • score()는 실천 중인 수칙 개수를 셉니다.
  • level()은 보호 수준을 분류합니다.
  • 3개 실천은 보통 수준입니다.

8. 비밀번호 강도 평가기

개념 상세 설명

안전한 비밀번호는 충분한 길이와 숫자, 대문자, 특수문자를 포함할수록 보안성이 높아집니다.
class PasswordStrengthChecker:
    def __init__(self, password):
        self.password = password

    def score(self):
        score = 0
        if len(self.password) >= 8:
            score += 1
        if any(c.isdigit() for c in self.password):
            score += 1
        if any(c.isupper() for c in self.password):
            score += 1
        if any(not c.isalnum() for c in self.password):
            score += 1
        return score

    def result(self):
        if self.score() >= 3:
            return "강한 비밀번호"
        if self.score() == 2:
            return "보통 비밀번호"
        return "약한 비밀번호"

checker = PasswordStrengthChecker("Data!2026")
print("점수:", checker.score())
print("결과:", checker.result())

코드 상세 설명

  • score()는 네 가지 조건을 검사합니다.
  • result()는 점수에 따라 강도를 반환합니다.
  • Data!2026은 강한 비밀번호입니다.

10. 앱 권한 필요성 판단기

개념 상세 설명

앱 권한에는 위치, 연락처, 카메라, 마이크, 저장소 등 민감한 권한이 포함될 수 있어 불필요한 권한을 점검해야 합니다.
class AppPermissionChecker:
    def __init__(self, app_name, permissions):
        self.app_name = app_name
        self.permissions = permissions

    def risky_permissions(self):
        risky = ["위치", "연락처", "카메라", "마이크", "저장소"]
        return [p for p in self.permissions if p in risky]

    def advice(self):
        risky_count = len(self.risky_permissions())
        if risky_count >= 3:
            return "권한 설정을 자세히 점검하세요."
        if risky_count >= 1:
            return "필요한 권한인지 확인하세요."
        return "위험 권한이 적습니다."

app = AppPermissionChecker("메모 앱", ["저장소", "위치", "마이크"])
print("위험 권한:", app.risky_permissions())
print("조언:", app.advice())

코드 상세 설명

  • permissions는 앱 권한 목록입니다.
  • risky_permissions()는 민감 권한을 추출합니다.
  • 민감 권한이 3개면 자세한 점검이 필요합니다.

11. 학교 정보 공유·보호 관리 시스템

개념 상세 설명

학교에서는 급식 메뉴나 행사 일정은 공유할 수 있지만, 학생 주민등록번호나 건강 기록은 보호해야 합니다.
class SchoolInfoManager:
    def __init__(self):
        self.data = []

    def add_info(self, title, info_type):
        self.data.append({"제목": title, "유형": info_type})

    def shareable_info(self):
        return [item for item in self.data if item["유형"] == "공유"]

    def protected_info(self):
        return [item for item in self.data if item["유형"] == "보호"]

manager = SchoolInfoManager()
manager.add_info("급식 메뉴", "공유")
manager.add_info("학생 주민등록번호", "보호")
manager.add_info("학교 행사 일정", "공유")
manager.add_info("학생 건강 기록", "보호")

print("공유 가능:", manager.shareable_info())
print("보호 필요:", manager.protected_info())

코드 상세 설명

  • add_info()는 정보 제목과 유형을 저장합니다.
  • shareable_info()는 공유 정보를 추출합니다.
  • protected_info()는 보호 정보를 추출합니다.

12. 피싱 메시지 탐지 에이전트

개념 상세 설명

피싱은 위조 사이트 접속을 유도하여 개인 정보를 탈취하는 행위입니다. 위험 키워드를 탐지하면 피싱 가능성을 판단할 수 있습니다.
class PhishingDetector:
    def __init__(self, text):
        self.text = text

    def risk_score(self):
        keywords = ["비밀번호", "계좌", "즉시", "링크", "개인 정보", "인증"]
        score = 0
        for keyword in keywords:
            if keyword in self.text:
                score += 1
        return score

    def classify(self):
        if self.risk_score() >= 3:
            return "피싱 위험 높음"
        if self.risk_score() >= 1:
            return "주의 필요"
        return "위험 낮음"

msg = PhishingDetector("즉시 링크에 접속해 계좌 인증과 비밀번호를 입력하세요.")
print("위험 점수:", msg.risk_score())
print("분류:", msg.classify())

코드 상세 설명

  • keywords는 위험 단어 목록입니다.
  • risk_score()는 위험 키워드 개수를 계산합니다.
  • 3개 이상이면 피싱 위험 높음입니다.

13. CCTV 영상 정보 접근 기록 관리

개념 상세 설명

CCTV는 얼굴, 움직임, 위치, 차량 번호판 등 다양한 영상 정보를 수집할 수 있어 목적과 절차에 맞게 열람해야 합니다.
class CCTVAccessLog:
    def __init__(self):
        self.logs = []

    def add_access(self, user, purpose):
        self.logs.append({"사용자": user, "목적": purpose})

    def allowed_access(self):
        allowed = ["범죄 예방", "안전 관리", "교통 관리"]
        return [log for log in self.logs if log["목적"] in allowed]

    def suspicious_access(self):
        allowed = ["범죄 예방", "안전 관리", "교통 관리"]
        return [log for log in self.logs if log["목적"] not in allowed]

log = CCTVAccessLog()
log.add_access("관리자A", "안전 관리")
log.add_access("사용자B", "개인 호기심")

print("허용 접근:", log.allowed_access())
print("의심 접근:", log.suspicious_access())

코드 상세 설명

  • add_access()는 접근 기록을 저장합니다.
  • allowed_access()는 허용 목적 접근만 찾습니다.
  • 개인 호기심은 의심 접근입니다.

15. 정보 보안 실천 종합 점검 대시보드

개념 상세 설명

사용자는 안전한 비밀번호 설정, 백신 업데이트, 보안 패치, 개인 정보 삭제, 의심 링크 주의 등을 실천해야 합니다.
class SecurityPracticeDashboard:
    def __init__(self, user_name):
        self.user_name = user_name
        self.items = {}

    def set_item(self, item, done):
        self.items[item] = done

    def completion_rate(self):
        if not self.items:
            return 0
        done_count = sum(1 for done in self.items.values() if done)
        return done_count / len(self.items) * 100

    def report(self):
        rate = self.completion_rate()
        if rate >= 80:
            level = "우수"
        elif rate >= 50:
            level = "보통"
        else:
            level = "개선 필요"
        return f"{self.user_name}의 정보 보안 실천율은 {rate:.1f}%이며 수준은 {level}입니다."

dashboard = SecurityPracticeDashboard("김학생")
dashboard.set_item("비밀번호 주기적 변경", True)
dashboard.set_item("백신 업데이트", True)
dashboard.set_item("불필요한 개인 정보 삭제", False)
dashboard.set_item("출처 불명 링크 클릭 금지", True)

print(dashboard.report())

코드 상세 설명

  • set_item()은 보안 실천 항목을 저장합니다.
  • completion_rate()는 실천율을 계산합니다.
  • report()는 실천 수준을 문장으로 반환합니다.
