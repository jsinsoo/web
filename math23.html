정리 1

테일러 정리

① 정리 설명
충분히 미분 가능한 함수는 한 점 근처에서 다항식과 나머지항으로 표현할 수 있다.
② 핵심 수식
\[ f(x) = \sum_{k=0}^{n} \frac{f^{(k)}(a)}{k!}(x-a)^k + R_n(x) \]
③ 증명 핵심 아이디어
  1. 함수값뿐 아니라 여러 차수의 도함수값까지 점 \(a\)에서 일치하는 다항식을 만든다.
  2. \(k\)차 항의 계수는 \(\frac{f^{(k)}(a)}{k!}\)로 정해진다.
  3. 다항식으로 설명되지 않는 부분은 나머지항 \(R_n(x)\)로 둔다.
  4. 차수가 커질수록 점 \(a\) 근처에서 더 정밀한 근사가 된다.
④ 활용 예제
지수함수·삼각함수 근사, 물리학의 작은 진동 근사, 계산기 알고리즘, 수치해석에 활용된다.
⑤ 파이썬 코드
# e^x의 테일러 근사
def exp_taylor(x, n):
    total = 0
    factorial = 1

    for k in range(n + 1):
        if k > 0:
            factorial *= k
        total += x**k / factorial

    return total

print(exp_taylor(1, 10))
⑥ 코드 상세 설명
  1. `total`은 테일러 급수 항들의 합을 저장한다.
  2. `factorial`은 \(k!\) 값을 저장한다.
  3. `x**k / factorial`은 \(\frac{x^k}{k!}\) 항이다.
  4. 반복문으로 \(k=0\)부터 \(n\)까지 더해 \(e^x\)를 근사한다.
정리 2

로피탈 정리

① 정리 설명
\(0/0\) 또는 \(\infty/\infty\) 꼴의 극한에서 조건이 맞으면 분자와 분모를 각각 미분해 극한을 계산할 수 있다.
② 핵심 수식
\[ \lim_{x\to a}\frac{f(x)}{g(x)} = \lim_{x\to a}\frac{f'(x)}{g'(x)} \]
③ 증명 핵심 아이디어
  1. 분자와 분모가 동시에 0으로 가거나 동시에 무한대로 가는 부정형을 다룬다.
  2. 코시 평균값 정리를 적용해 함수값의 비를 도함수의 비와 연결한다.
  3. 도함수 비의 극한이 존재하면 원래 함수 비의 극한도 같은 값이 된다.
  4. 단, 미분 가능성, 분모 도함수가 0이 아님, 극한 존재 조건을 확인해야 한다.
④ 활용 예제
삼각함수 극한, 지수·로그 극한, 부정형 극한 계산에 활용된다.
⑤ 파이썬 코드
# sin(x)/x의 x=0 극한을 수치적으로 확인
import math

for x in [1, 0.1, 0.01, 0.001]:
    print(x, math.sin(x) / x)
⑥ 코드 상세 설명
  1. `math.sin(x) / x`는 \(\frac{\sin x}{x}\)를 계산한다.
  2. \(x\)를 0에 가깝게 줄이면 값이 1에 가까워진다.
  3. 로피탈 정리를 쓰면 \(\frac{\cos 0}{1}=1\)이다.
정리 3

다변수 연쇄법칙

① 정리 설명
\(z=f(x,y)\), \(x=x(t)\), \(y=y(t)\)이면 \(z\)의 전체 변화율은 각 경로의 변화율을 더한 것이다.
② 핵심 수식
\[ \frac{dz}{dt} = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt} \]
③ 증명 핵심 아이디어
  1. \(z\)는 \(x\)와 \(y\)에 의존하고, \(x,y\)는 다시 \(t\)에 의존한다.
  2. \(x\) 방향 변화가 \(z\)에 미치는 영향은 \(\frac{\partial f}{\partial x}\frac{dx}{dt}\)이다.
  3. \(y\) 방향 변화가 \(z\)에 미치는 영향은 \(\frac{\partial f}{\partial y}\frac{dy}{dt}\)이다.
  4. 두 영향의 합이 \(\frac{dz}{dt}\)가 된다.
④ 활용 예제
온도장 속 이동 물체의 온도 변화, 다변수 최적화, 머신러닝 역전파에 활용된다.
⑤ 파이썬 코드
# z=x^2+y^2, x=t, y=2t일 때 dz/dt
t = 3
x = t
y = 2*t

fx = 2*x
fy = 2*y
dxdt = 1
dydt = 2

dzdt = fx*dxdt + fy*dydt
print("dz/dt:", dzdt)
⑥ 코드 상세 설명
  1. \(z=x^2+y^2\)이므로 \(f_x=2x\), \(f_y=2y\)이다.
  2. `x=t`이므로 \(dx/dt=1\)이다.
  3. `y=2t`이므로 \(dy/dt=2\)이다.
  4. 공식에 대입해 전체 변화율을 계산한다.
정리 4

라그랑주 승수법

① 정리 설명
제약조건 \(g(x,y)=0\) 아래에서 \(f(x,y)\)가 극값을 가지면 보통 두 기울기 벡터가 서로 평행하다.
② 핵심 수식
\[ \nabla f = \lambda \nabla g \]
③ 증명 핵심 아이디어
  1. 제약곡선 위에서 움직일 수 있는 방향은 제약곡선의 접선 방향이다.
  2. 극값에서는 \(f\)가 그 접선 방향으로 더 이상 증가하거나 감소하지 않는다.
  3. 따라서 \(\nabla f\)는 제약곡선의 접선에 수직이다.
  4. \(\nabla g\)도 제약곡선에 수직이다.
  5. 결국 두 벡터는 평행하므로 \(\nabla f=\lambda\nabla g\)가 된다.
④ 활용 예제
제약 최적화, 경제학 효용 극대화, 물리학 에너지 최소화, 머신러닝 제약 문제에 활용된다.
⑤ 파이썬 코드
# x+y 최대, 제약 x^2+y^2=1에서 대칭해 확인
import math

x = 1 / math.sqrt(2)
y = 1 / math.sqrt(2)

f = x + y
constraint = x**2 + y**2

print("f 값:", f)
print("제약조건:", constraint)
⑥ 코드 상세 설명
  1. 목표함수는 \(f=x+y\)이다.
  2. 제약조건은 \(x^2+y^2=1\)이다.
  3. 대칭성에 의해 최대점은 \(x=y\)에서 나타난다.
  4. \(x=y=1/\sqrt2\)를 대입해 제약조건을 확인한다.
정리 5

중적분과 푸비니 정리

① 정리 설명
적절한 조건에서 직사각형 영역 위의 중적분은 반복적분으로 계산할 수 있다.
② 핵심 수식
\[ \iint_R f(x,y)\,dA = \int_a^b\int_c^d f(x,y)\,dy\,dx \]
③ 증명 핵심 아이디어
  1. 영역을 작은 직사각형 조각으로 나눈다.
  2. 각 조각에서 함수값과 면적을 곱해 누적한다.
  3. 한 방향으로 먼저 더하고, 다른 방향으로 다시 더할 수 있다.
  4. 함수가 충분히 좋으면 적분 순서를 바꾸어도 값이 같다.
④ 활용 예제
면적, 질량, 확률밀도, 열분포, 영상 데이터 누적 계산에 활용된다.
⑤ 파이썬 코드
# f(x,y)=x+y를 [0,1]x[0,1]에서 수치 중적분
n = 200
dx = 1 / n
dy = 1 / n
total = 0

for i in range(n):
    for j in range(n):
        x = i * dx
        y = j * dy
        total += (x + y) * dx * dy

print("중적분 근사:", total)
⑥ 코드 상세 설명
  1. `dx`, `dy`는 작은 직사각형의 가로·세로 길이이다.
  2. `(x+y)*dx*dy`는 작은 부피 조각이다.
  3. 이중 반복문으로 모든 조각을 더해 중적분을 근사한다.
정리 6

그린 정리

① 정리 설명
평면 영역의 경계선 위 선적분은 영역 내부의 회전량에 대한 이중적분과 연결된다.
② 핵심 수식
\[ \oint_{\partial D} P\,dx + Q\,dy = \iint_D \left( \frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y} \right) dA \]
③ 증명 핵심 아이디어
  1. 영역을 작은 직사각형들로 나눈다.
  2. 작은 직사각형들의 경계 적분을 모두 더하면 내부 경계는 서로 반대 방향으로 상쇄된다.
  3. 결국 바깥 경계만 남는다.
  4. 내부에 누적되는 양은 \(\frac{\partial Q}{\partial x}-\frac{\partial P}{\partial y}\)이다.
④ 활용 예제
평면 벡터장 순환, 면적 계산, 유체 회전 분석, 전자기학에 활용된다.
⑤ 파이썬 코드
# 단위원 넓이: P=-y/2, Q=x/2이면 선적분이 넓이
import math

N = 10000
area = 0

for k in range(N):
    t1 = 2*math.pi*k/N
    t2 = 2*math.pi*(k+1)/N
    x1, y1 = math.cos(t1), math.sin(t1)
    x2, y2 = math.cos(t2), math.sin(t2)
    dx, dy = x2-x1, y2-y1
    P, Q = -y1/2, x1/2
    area += P*dx + Q*dy

print("넓이 근사:", area)
print("pi:", math.pi)
⑥ 코드 상세 설명
  1. \(P=-y/2\), \(Q=x/2\)이면 \(\partial Q/\partial x-\partial P/\partial y=1\)이다.
  2. 따라서 선적분은 영역의 넓이가 된다.
  3. 단위원 경계를 작은 선분으로 나누어 근사한다.
정리 7

발산정리

① 정리 설명
공간 내부의 총 발산량은 경계면을 통과하는 총 플럭스와 같다.
② 핵심 수식
\[ \iiint_V \nabla\cdot\vec{F}\,dV = \iint_S \vec{F}\cdot\vec{n}\,dS \]
③ 증명 핵심 아이디어
  1. 입체 영역을 작은 부피 조각으로 나눈다.
  2. 각 조각의 표면 플럭스를 더하면 내부 면에서 흐름이 서로 상쇄된다.
  3. 전체 바깥 경계면을 통과하는 흐름만 남는다.
  4. 그 결과 내부 발산의 부피적분과 경계 플럭스가 같아진다.
④ 활용 예제
유체 흐름, 전기장 가우스 법칙, 열전달, 보존법칙에 활용된다.
⑤ 파이썬 코드
# F=(x,y,z)의 발산은 3, 단위정육면체 부피는 1
divergence = 3
volume = 1

flux = divergence * volume
print("총 플럭스:", flux)
⑥ 코드 상세 설명
  1. \(\vec{F}=(x,y,z)\)의 발산은 \(1+1+1=3\)이다.
  2. 단위정육면체의 부피는 1이다.
  3. 총 플럭스는 발산과 부피의 곱이다.
정리 8

스토크스 정리

① 정리 설명
곡면 내부의 회전량을 모두 더하면 경계곡선을 따라 도는 순환량과 같다.
② 핵심 수식
\[ \iint_S(\nabla\times\vec{F})\cdot\vec{n}\,dS = \oint_{\partial S}\vec{F}\cdot d\vec{r} \]
③ 증명 핵심 아이디어
  1. 스토크스 정리는 그린 정리의 3차원 일반화이다.
  2. 곡면을 작은 조각으로 나누면 내부 경계는 서로 반대 방향으로 상쇄된다.
  3. 결국 전체 경계곡선만 남는다.
  4. 내부 회전량의 면적분과 경계 순환 선적분이 같아진다.
④ 활용 예제
전자기학, 유체 소용돌이, 회전장 분석, 곡면 위 적분 계산에 활용된다.
⑤ 파이썬 코드
# F=(-y/2, x/2, 0)의 curl z성분은 1
import math

curl_z = 1
area = math.pi * 1**2
surface_integral = curl_z * area

print("스토크스 정리 면적분 값:", surface_integral)
⑥ 코드 상세 설명
  1. \(\vec{F}=(-y/2,x/2,0)\)의 회전은 z방향으로 1이다.
  2. 단위원판의 넓이는 \(\pi\)이다.
  3. 면적분 값은 \(1\cdot\pi\)이다.
정리 9

균등수렴과 적분 교환

① 정리 설명
함수열이 균등수렴하면 적절한 조건에서 극한과 적분을 교환할 수 있다.
② 핵심 수식
\[ \lim_{n\to\infty}\int_a^b f_n(x)\,dx = \int_a^b \lim_{n\to\infty}f_n(x)\,dx \]
③ 증명 핵심 아이디어
  1. 점별수렴만으로는 적분과 극한을 항상 바꿀 수 없다.
  2. 균등수렴은 모든 \(x\)에서 동시에 오차가 작아지는 강한 조건이다.
  3. \(|f_n-f|\)가 전체 구간에서 작으면 적분 오차도 작다.
  4. 구간 길이를 곱해 오차를 제어할 수 있어 극한과 적분을 교환할 수 있다.
④ 활용 예제
함수열, 푸리에 급수, 수치근사, 확률밀도 근사에 활용된다.
⑤ 파이썬 코드
# f_n(x)=x/n이 0으로 균등수렴할 때 적분도 0으로 감
for n in [1, 2, 5, 10, 100]:
    integral = 1 / (2*n)
    print(n, integral)
⑥ 코드 상세 설명
  1. \(f_n(x)=x/n\)은 \([0,1]\)에서 0으로 균등수렴한다.
  2. \(\int_0^1 x/n\,dx=1/(2n)\)이다.
  3. \(n\)이 커질수록 적분값은 0으로 간다.
정리 10

야코비안 치환 공식

① 정리 설명
다변수 적분에서 좌표를 바꾸면 면적요소는 야코비안 절댓값만큼 확대 또는 축소된다.
② 핵심 수식
\[ dx\,dy = \left| \frac{\partial(x,y)}{\partial(u,v)} \right| du\,dv \]
③ 증명 핵심 아이디어
  1. 변수변환은 작은 \(du\,dv\) 조각을 \(xy\)-평면의 작은 평행사변형으로 보낸다.
  2. 그 평행사변형의 넓이 비율이 야코비안이다.
  3. 극좌표에서는 \(x=r\cos\theta\), \(y=r\sin\theta\)이다.
  4. 이때 야코비안은 \(r\)이므로 \(dx\,dy=r\,dr\,d\theta\)가 된다.
④ 활용 예제
극좌표 적분, 원판 넓이, 확률분포 변환, 물리학 좌표변환에 활용된다.
⑤ 파이썬 코드
# 극좌표로 반지름 R인 원의 넓이
import math

R = 3
area = math.pi * R**2

print("원의 넓이:", area)
⑥ 코드 상세 설명
  1. 극좌표에서 면적요소는 \(r\,dr\,d\theta\)이다.
  2. 반지름 \(R\)인 원판의 넓이는 \(\pi R^2\)이다.
  3. `R=3`이면 넓이는 \(9\pi\)이다.

마무리 핵심 정리

고급미적분은 단순한 미분과 적분을 넘어 함수의 근사, 다변수 변화율, 제약 최적화, 면적·부피 누적, 벡터장의 흐름과 회전을 다룬다. 테일러 정리는 함수를 다항식으로 근사하고, 그린 정리·발산정리·스토크스 정리는 경계와 내부의 정보를 연결한다.
