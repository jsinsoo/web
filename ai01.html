1. 지능 에이전트: 인식·판단·행동 구조

개념 상세 설명

지능 에이전트는 센서를 통해 환경을 인식하고, 인공지능으로 판단한 뒤 구동기를 통해 행동하는 구조를 가집니다.
class IntelligentAgent:
    def __init__(self, name):
        self.name = name

    def perceive(self, environment):
        return f"{self.name}이/가 환경 정보 '{environment}'을/를 인식했습니다."

    def decide(self, environment):
        if "어두움" in environment:
            return "불 켜기"
        return "대기하기"

    def act(self, action):
        return f"{self.name}의 행동: {action}"

agent = IntelligentAgent("스마트홈 에이전트")
env = "거실이 어두움"
print(agent.perceive(env))
action = agent.decide(env)
print(agent.act(action))

코드 상세 설명

  • perceive()는 환경 정보를 인식합니다.
  • decide()는 조건에 따라 행동을 선택합니다.
  • act()는 선택된 행동을 출력합니다.

2. 규칙 기반 인공지능 분류기

개념 상세 설명

규칙 기반 인공지능은 사람이 조건을 직접 작성하여 입력을 판단합니다. 단순하고 설명 가능하지만 복잡한 문제에서는 한계가 있습니다.
class RuleBasedClassifier:
    def __init__(self):
        self.rules = []

    def add_rule(self, condition, label):
        self.rules.append((condition, label))

    def predict(self, data):
        for condition, label in self.rules:
            if condition(data):
                return label
        return "판단 불가"

classifier = RuleBasedClassifier()
classifier.add_rule(lambda x: x["온도"] >= 30, "더움")
classifier.add_rule(lambda x: x["온도"] <= 10, "추움")
print(classifier.predict({"온도": 35}))
print(classifier.predict({"온도": 20}))

코드 상세 설명

  • add_rule()로 판단 규칙을 추가합니다.
  • predict()는 규칙을 순서대로 검사합니다.
  • 어떤 규칙에도 맞지 않으면 판단 불가를 반환합니다.

3. 기계학습 데이터셋 객체

class Dataset:
    def __init__(self, features, labels=None):
        self.features = features
        self.labels = labels

    def size(self):
        return len(self.features)

    def has_labels(self):
        return self.labels is not None

    def summary(self):
        return {"데이터 개수": self.size(), "정답 포함 여부": self.has_labels()}

X = [[1, 2], [2, 3], [3, 4]]
y = ["A", "A", "B"]
dataset = Dataset(X, y)
print(dataset.summary())

  • labels는 지도학습에서 사용하는 정답입니다.
  • has_labels()로 지도학습 가능성을 확인합니다.

4. 지도학습과 비지도학습 구분기

개념 상세 설명

기계학습은 정답 데이터가 있는지에 따라 지도학습과 비지도학습으로 구분됩니다. 지도학습에는 회귀와 분류, 비지도학습에는 군집이 있습니다.
class LearningTypeChecker:
    def __init__(self, has_label, goal):
        self.has_label = has_label
        self.goal = goal

    def check(self):
        if self.has_label and self.goal == "값 예측":
            return "지도학습 - 회귀"
        if self.has_label and self.goal == "종류 분류":
            return "지도학습 - 분류"
        if not self.has_label and self.goal == "그룹 찾기":
            return "비지도학습 - 군집"
        return "추가 분석 필요"

print(LearningTypeChecker(True, "값 예측").check())
print(LearningTypeChecker(False, "그룹 찾기").check())

코드 상세 설명

  • has_label은 정답 데이터 여부입니다.
  • goal은 문제 해결 목표입니다.
  • 목표에 따라 회귀, 분류, 군집으로 구분합니다.

class AIProblemSolvingProcess:
    def __init__(self, problem):
        self.problem = problem
        self.steps = []

    def add_step(self, step):
        self.steps.append(step)

    def show_steps(self):
        for i, step in enumerate(self.steps, start=1):
            print(f"{i}단계: {step}")

process = AIProblemSolvingProcess("자동차 CO2 배출량 예측")

    process.add_step(step)
process.show_steps()

6. 단순 회귀 모델: 공부 시간으로 점수 예측

개념 상세 설명

회귀는 속성 간 관계를 함수식으로 학습하여 연속적인 값을 예측하는 지도학습 방법입니다.
class SimpleRegression:
    def __init__(self):
        self.a = 0
        self.b = 0

    def fit(self, X, y):
        x_mean = sum(X) / len(X)
        y_mean = sum(y) / len(y)
        numerator = sum((x - x_mean) * (target - y_mean) for x, target in zip(X, y))
        denominator = sum((x - x_mean) ** 2 for x in X)
        self.a = numerator / denominator
        self.b = y_mean - self.a * x_mean

    def predict(self, x):
        return self.a * x + self.b

hours = [1, 2, 3, 4, 5]
scores = [50, 60, 65, 75, 85]
model = SimpleRegression()
model.fit(hours, scores)
print("6시간 공부 예측 점수:", model.predict(6))

코드 상세 설명

  • fit()은 기울기와 절편을 학습합니다.
  • predict()는 새 입력에 대한 예측값을 반환합니다.
  • 공부 시간과 점수의 관계를 직선으로 근사합니다.

7. KNN 분류 모델: 가까운 이웃으로 종류 판단

개념 상세 설명

분류는 정답 레이블이 있는 데이터를 학습하여 새 데이터가 어느 그룹에 속하는지 판단합니다.
class KNNClassifier:
    def __init__(self, k=3):
        self.k = k
        self.X_train = []
        self.y_train = []

    def fit(self, X, y):
        self.X_train = X
        self.y_train = y

    def distance(self, a, b):
        return sum((x - y) ** 2 for x, y in zip(a, b)) ** 0.5

    def predict_one(self, x):
        distances = [(self.distance(x, tx), label) for tx, label in zip(self.X_train, self.y_train)]
        distances.sort()
        nearest = distances[:self.k]
        votes = {}
        for _, label in nearest:
            votes[label] = votes.get(label, 0) + 1
        return max(votes, key=votes.get)

X = [[1, 1], [2, 1], [8, 8], [9, 8]]
y = ["고양이", "고양이", "강아지", "강아지"]
model = KNNClassifier(k=3)
model.fit(X, y)
print(model.predict_one([2, 2]))

코드 상세 설명

  • distance()는 유클리드 거리를 계산합니다.
  • 가까운 k개 이웃을 찾습니다.
  • 다수결로 라벨을 결정합니다.

8. 군집 모델: 비슷한 학생 그룹 묶기

개념 상세 설명

군집은 정답 없이 비슷한 속성을 가진 데이터끼리 묶는 비지도학습 방법입니다.
class SimpleClusterer:
    def __init__(self, threshold):
        self.threshold = threshold

    def cluster(self, students):
        active, passive = [], []
        for student in students:
            score = student["발표횟수"] + student["자료조회"]
            if score >= self.threshold:
                active.append(student["이름"])
            else:
                passive.append(student["이름"])
        return {"적극 참여 그룹": active, "보통 참여 그룹": passive}

students = [
    {"이름": "김", "발표횟수": 10, "자료조회": 30},
    {"이름": "이", "발표횟수": 2, "자료조회": 8},
    {"이름": "박", "발표횟수": 8, "자료조회": 25}
]
print(SimpleClusterer(threshold=30).cluster(students))

코드 상세 설명

  • 발표횟수와 자료조회 수로 참여도 점수를 만듭니다.
  • 기준값 이상이면 적극 참여 그룹입니다.
  • 정답 없이 특징을 기준으로 묶습니다.

9. 모델 성능 평가: 정확도 계산

개념 상세 설명

정확도는 전체 예측 중 정답과 일치한 비율로, 분류 모델 성능 평가에 자주 사용됩니다.
class AccuracyEvaluator:
    def __init__(self, y_true, y_pred):
        self.y_true = y_true
        self.y_pred = y_pred

    def accuracy(self):
        correct = 0
        for true, pred in zip(self.y_true, self.y_pred):
            if true == pred:
                correct += 1
        return correct / len(self.y_true)

true_labels = ["A", "B", "A", "C", "B"]
pred_labels = ["A", "B", "C", "C", "B"]
print("정확도:", AccuracyEvaluator(true_labels, pred_labels).accuracy())

코드 상세 설명

  • y_true는 실제 정답입니다.
  • y_pred는 예측 결과입니다.
  • 일치 개수를 전체 개수로 나누어 정확도를 계산합니다.

10. 데이터 정규화 객체

개념 상세 설명

정규화는 서로 다른 크기의 수치 데이터를 0과 1 사이 범위로 변환하여 학습 안정성을 높이는 전처리 방법입니다.
class MinMaxScalerSimple:
    def __init__(self, data):
        self.data = data

    def normalize(self):
        min_value = min(self.data)
        max_value = max(self.data)
        return [(x - min_value) / (max_value - min_value) for x in self.data]

values = [10, 20, 30, 40, 50]
print(MinMaxScalerSimple(values).normalize())

코드 상세 설명

  • 최솟값과 최댓값을 구합니다.
  • 각 값을 전체 범위로 나누어 0~1 사이로 변환합니다.
  • 수치 속성의 크기 차이를 줄입니다.

11. 스마트 냉장고 추천 에이전트

class SmartFridgeAgent:
    def __init__(self, ingredients):
        self.ingredients = ingredients

    def perceive(self):
        return f"냉장고 재료 인식: {', '.join(self.ingredients)}"

    def recommend_menu(self):

class MedicalTriageAgent:

13. CO2 배출량 예측 회귀 모델

개념 상세 설명

자동차 엔진 크기와 연료 소비량으로 CO2 배출량을 예측하는 문제는 지도학습의 회귀 문제입니다.
class CO2EmissionPredictor:
    def __init__(self, engine_weight=40, fuel_weight=20, bias=50):
        self.engine_weight = engine_weight
        self.fuel_weight = fuel_weight
        self.bias = bias

    def predict(self, engine_size, fuel_consumption):
        return self.engine_weight * engine_size + self.fuel_weight * fuel_consumption + self.bias

    def explain(self, engine_size, fuel_consumption):
        return f"예상 CO2 배출량: {self.predict(engine_size, fuel_consumption):.1f}"

model = CO2EmissionPredictor()
print(model.explain(engine_size=3.0, fuel_consumption=7.5))

코드 상세 설명

  • engine_weight와 fuel_weight는 속성의 영향입니다.
  • predict()가 배출량을 계산합니다.
  • 실제 모델은 데이터로 가중치를 학습합니다.

14. 학업 성취 수준 분류 모델

개념 상세 설명

학생의 발표 횟수, 자료 접근 횟수, 결석 횟수 등으로 성취 수준을 분류하는 문제는 지도학습의 분류 문제입니다.
class AcademicPerformanceClassifier:
    def predict(self, raised_hands, visited_resources, absences):
        score = raised_hands * 0.4 + visited_resources * 0.4 - absences * 2
        if score >= 50:
            return "상위"
        elif score >= 25:
            return "중위"
        return "하위"

classifier = AcademicPerformanceClassifier()
print(classifier.predict(raised_hands=80, visited_resources=70, absences=2))
print(classifier.predict(raised_hands=10, visited_resources=15, absences=8))

코드 상세 설명

  • 수업 참여와 자료 조회는 긍정 요인입니다.
  • 결석은 점수를 낮추는 요인입니다.

15. SNS 취향 군집 추천 시스템

개념 상세 설명

SNS 관심 데이터로 비슷한 취향의 사용자를 묶고 추천하는 과정은 비지도학습의 군집 개념과 연결됩니다.
class SNSClusterRecommender:
    def __init__(self, users):
        self.users = users

    def cluster_users(self):
        clusters = {"운동 관심 그룹": [], "음악 관심 그룹": [], "학습 관심 그룹": []}
        for user in self.users:
            if user["운동"] >= user["음악"] and user["운동"] >= user["학습"]:
                clusters["운동 관심 그룹"].append(user["이름"])
            elif user["음악"] >= user["운동"] and user["음악"] >= user["학습"]:
                clusters["음악 관심 그룹"].append(user["이름"])
            else:
                clusters["학습 관심 그룹"].append(user["이름"])
        return clusters

users = [
    {"이름": "김", "운동": 9, "음악": 3, "학습": 5},
    {"이름": "이", "운동": 2, "음악": 8, "학습": 4},
    {"이름": "박", "운동": 4, "음악": 5, "학습": 9}
]
print(SNSClusterRecommender(users).cluster_users())
