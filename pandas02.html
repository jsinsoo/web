예제 01

📊 선그래프: 월별 매출 추이

🐼 Pandas🎨 Seaborn📈 Matplotlib

💻 코드

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.DataFrame({"월":["1월","2월","3월","4월"], "매출":[100,150,130,180]})
sns.lineplot(data=df, x="월", y="매출", marker="o")
plt.title("월별 매출 추이")
plt.xlabel("월")
plt.ylabel("매출")
plt.grid(True)
plt.show()

🔍 상세 설명

  1. Pandas로 월별 매출 표를 만든다.
  2. lineplot은 시간 흐름에 따른 변화를 표현한다.
  3. marker="o"는 각 데이터 지점을 강조한다.
  4. Matplotlib으로 제목, 축 이름, 격자를 설정한다.
예제 02

📊 막대그래프: 지역별 매출 비교

barplot범주 비교

💻 코드

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.DataFrame({"지역":["서울","부산","대구","광주"], "매출":[320,210,180,150]})
sns.barplot(data=df, x="지역", y="매출")
plt.title("지역별 매출 비교")
plt.xlabel("지역")
plt.ylabel("매출")
plt.show()

🔍 상세 설명

  1. 막대그래프는 범주별 크기 비교에 적합하다.
  2. x축은 지역, y축은 매출이다.
  3. 막대가 높을수록 매출이 크다.
  4. 반별 평균, 제품별 판매량 비교에도 사용할 수 있다.

📊 히트맵: 과목 간 상관관계

import pandas as pd

import seaborn as sns

예제 12

📊 피벗 테이블 히트맵: 지역·제품별 매출

pivot_tableheatmap

💻 코드

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.DataFrame({"지역":["서울","서울","부산","부산","대구","대구"], "제품":["A","B","A","B","A","B"], "매출":[100,150,80,120,90,110]})
pivot = pd.pivot_table(df, values="매출", index="지역", columns="제품", aggfunc="sum")
sns.heatmap(pivot, annot=True, fmt=".0f")
plt.title("지역·제품별 매출 히트맵")
plt.show()

🔍 상세 설명

  1. pivot_table은 긴 데이터를 요약 표로 바꾼다.
  2. index는 행 기준, columns는 열 기준이다.
  3. values는 계산할 값이고 aggfunc는 계산 방식이다.
  4. 히트맵은 큰 값과 작은 값을 색으로 빠르게 구분한다.

예제 14

📊 groupby 전처리 후 막대그래프

groupby집계

💻 코드

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.DataFrame({"지역":["서울","서울","부산","부산","서울","부산"], "매출":[100,120,80,90,140,110]})
summary = df.groupby("지역", as_index=False)["매출"].sum()
sns.barplot(data=summary, x="지역", y="매출")
plt.title("지역별 총매출")
plt.xlabel("지역")
plt.ylabel("총매출")
plt.show()

🔍 상세 설명

  1. 시각화 전에 pandas groupby로 지역별 총매출을 계산한다.
  2. as_index=False는 결과를 DataFrame 형태로 유지한다.
  3. Seaborn은 정리된 summary를 그래프로 표현한다.
  4. Pandas는 전처리, Seaborn은 시각화 역할을 한다.

예제 17

📊 FacetGrid: 지역별 산점도 분리

FacetGrid분할 그래프

💻 코드

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.DataFrame({"광고비":[10,20,30,15,25,35,12,22,32], "매출":[100,150,210,90,160,220,95,145,205], "지역":["서울","서울","서울","부산","부산","부산","대구","대구","대구"]})

  3. map_dataframe은 각 지역 데이터에 산점도를 적용한다.
  4. 한 그래프가 복잡할 때 분리해서 보기 좋다.

  3. 대각선 밖에는 두 변수의 산점도가 나타난다.
  4. 탐색적 데이터 분석 EDA에서 유용하다.
예제 19

📊 결측값 시각화: 누락 데이터 히트맵

isnamissing values

💻 코드

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

df = pd.DataFrame({"이름":["김","이","박","최"], "점수":[90,None,85,None], "출석":[10,9,None,8]})

  1. df.isna()는 결측값이면 True, 아니면 False를 반환한다.
  2. heatmap으로 결측 위치를 색으로 표현한다.
  3. 어느 행과 열에 결측값이 있는지 한눈에 확인한다.
  4. 데이터 정제 전에 결측 패턴을 파악할 수 있다.
예제 20

📊 종합 예제: 전처리 + 집계 + 시각화

pipelinecleaningvisualization

💻 코드

import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

data = {"지역":["서울 ","서울","부산","부산","대구"], "매출":[120,None,80,150,90]}
df = pd.DataFrame(data)
df["지역"] = df["지역"].str.strip()
df["매출"] = df["매출"].fillna(df["매출"].mean())
df["등급"] = df["매출"].apply(lambda x: "높음" if x >= 100 else "낮음")
summary = df.groupby("지역", as_index=False)["매출"].sum()
sns.barplot(data=summary, x="지역", y="매출")
plt.title("전처리 후 지역별 매출 합계")
plt.xlabel("지역")
plt.ylabel("매출 합계")
plt.show()
print(df)
print(summary)

🔍 상세 설명

  1. str.strip()은 문자열 앞뒤 공백을 제거한다.
  2. fillna(mean)는 결측값을 평균으로 채운다.
  3. apply와 lambda로 매출 등급을 새 열로 만든다.
  4. groupby로 지역별 매출 합계를 계산한다.
  5. 마지막으로 막대그래프로 요약 결과를 시각화한다.

🧩 핵심 정리

Pandas는 데이터 생성·정리·집계를 담당하고, Seaborn은 통계 그래프를 쉽게 그리며, Matplotlib은 제목·축·격자·범례 같은 세부 디자인을 조정합니다. 세 라이브러리를 함께 사용하면 데이터 분석 수업, 과학 탐구, 인공지능 프로젝트에서 강력한 시각화 자료를 만들 수 있습니다. ✨
