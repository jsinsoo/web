1. 한강, 「소년이 온다」

결말작품은 망각이 아니라 기억과 애도를 요구한다. 살아남은 사람들은 끝내 사라지지 않는 질문과 함께 살아간다.

작품 분석 모델: \[ A = F + L + M + E \] \(F\): 페미니즘과 돌봄, \(L\): 노동·청년 현실, \(M\): 기억과 애도, \(E\): 윤리적 상상력

reasons = {'경쟁': 9, '집값': 8, '노동': 8, '행복': 3}

8. 김영하, 「살인자의 기억법」

2013 장편소설·스릴러·기억과 정체성 기억·정체성·죄·불신 서술자

도입·전개·절정·결말 줄거리

도입은퇴한 연쇄살인범 김병수는 알츠하이머에 걸려 자신의 기억을 잃어 간다.
전개그는 딸을 지키기 위해 또 다른 살인자를 의심하지만, 자신의 기억이 믿을 수 있는지 확신하지 못한다.
절정기억의 공백과 왜곡이 커지며 독자는 무엇이 사실이고 무엇이 망상인지 판단하기 어려워진다.
결말작품은 살인자의 기억이 무너지는 과정을 통해 정체성과 죄, 서술의 신뢰성을 질문한다.

작가의 생애와 사상

김영하는 도시적 감각, 죽음, 욕망, 기억, 서사의 불확실성을 즐겨 탐구하는 작가이다. 「살인자의 기억법」은 장르적 긴장감과 철학적 질문을 결합해 기억이 곧 나 자신인가를 묻는다.

작품 분석 모델: \[ A = F + L + M + E \] \(F\): 페미니즘과 돌봄, \(L\): 노동·청년 현실, \(M\): 기억과 애도, \(E\): 윤리적 상상력

파이썬 코드 예제

Python
memory = ['기억', '망각', '의심', '정체성 붕괴']
for step in memory:
    print('상태:', step)
  1. 기억이 무너지는 단계를 리스트로 저장합니다.
  2. for문으로 상태를 순서대로 출력합니다.
  3. 서술자의 신뢰성이 흔들리는 구조를 확인합니다.

9. 김초엽, 「우리가 빛의 속도로 갈 수 없다면」

2019 SF 소설집·과학과 돌봄 SF·돌봄·차이·연결

도입·전개·절정·결말 줄거리

도입여러 단편은 우주, 과학기술, 장애, 기억, 관계의 문제를 미래적 상상력 속에서 펼친다.
전개과학기술은 인간을 구원하기도 하지만, 동시에 누군가를 배제하거나 기다리게 만들기도 한다.
절정인물들은 불가능해 보이는 거리와 시간, 차이를 넘어 타인에게 닿으려 한다.
결말작품들은 과학의 속도보다 중요한 것은 연결, 돌봄, 타자를 향한 상상력임을 보여 준다.

작가의 생애와 사상

김초엽은 과학적 상상력과 섬세한 감정, 돌봄과 차이의 윤리를 결합한 작가이다. 그는 2010년대 후반 한국 SF의 대중적 확산을 이끈 대표 작가 중 한 명으로 평가된다.

작품 분석 모델: \[ A = F + L + M + E \] \(F\): 페미니즘과 돌봄, \(L\): 노동·청년 현실, \(M\): 기억과 애도, \(E\): 윤리적 상상력

파이썬 코드 예제

Python
sf = {'과학기술': 8, '거리': 9, '돌봄': 10, '연결': 10}
for key, value in sf.items():
    print(key, '중요도:', value)
  1. SF적 요소와 윤리적 가치를 딕셔너리로 저장합니다.
  2. 기술보다 돌봄과 연결의 중요도를 높게 표현합니다.
  3. 과학소설이 인간관계의 질문으로 확장됨을 분석합니다.

10. 천선란, 「천 개의 파랑」

2019 SF 장편소설·비인간·돌봄 SF·비인간·돌봄·느림의 윤리

도입·전개·절정·결말 줄거리

도입경마장에서 로봇 기수 콜리는 말 투데이와 함께 달린다.
전개콜리, 투데이, 연재와 은혜의 이야기는 인간, 동물, 로봇이 서로를 돌보는 관계로 확장된다.
절정속도와 효율을 중시하는 사회 속에서 다친 존재들이 서로를 지키려는 선택을 한다.
결말작품은 가장 빠른 것이 아니라 함께 멈추고 돌보는 삶의 가치를 보여 준다.

작가의 생애와 사상

천선란은 SF적 상상력을 통해 비인간 존재, 동물, 로봇, 약한 존재의 감각을 따뜻하게 그리는 작가이다. 그의 작품은 기술 발전보다 생명과 돌봄의 윤리를 중시한다.

작품 분석 모델: \[ A = F + L + M + E \] \(F\): 페미니즘과 돌봄, \(L\): 노동·청년 현실, \(M\): 기억과 애도, \(E\): 윤리적 상상력

파이썬 코드 예제

Python
beings = {'콜리': '로봇', '투데이': '말', '연재': '인간', '은혜': '가족'}
for name, kind in beings.items():
    print(name, '→', kind)
  1. 서로 다른 존재들을 딕셔너리로 저장합니다.
  2. 인간, 동물, 로봇이 함께 서사를 만든다는 점을 출력합니다.
  3. 비인간 존재까지 포함하는 돌봄 윤리를 이해합니다.
  4. 문학 작품의 주제·상징·갈등을 파이썬 자료구조로 바꾸는 연습입니다.

종합 파이썬 활동

아래 코드는 2010년대 대표작을 연도순으로 정리하고, 핵심 주제를 출력하는 예제입니다.

Python
novels = [
    (2013, "살인자의 기억법", "기억과 정체성"),
    (2014, "소년이 온다", "광주와 애도"),
    (2015, "보건교사 안은영", "판타지와 돌봄"),
    (2015, "한국이 싫어서", "청년 불안과 이민"),
    (2016, "82년생 김지영", "페미니즘 리얼리즘"),
    (2016, "쇼코의 미소", "관계와 공감"),
    (2016, "피프티 피플", "도시 공동체"),
    (2017, "아몬드", "감정과 성장"),
    (2019, "우리가 빛의 속도로 갈 수 없다면", "SF와 연결"),
    (2019, "천 개의 파랑", "비인간과 돌봄")
]

for year, title, theme in novels:
    print(f"{year}년 - {title}: {theme}")
  1. novels는 연도, 제목, 주제를 튜플로 저장합니다.
  2. for year, title, theme in novels는 각 튜플을 세 변수로 나누어 꺼냅니다.
  3. f-string으로 문학사 흐름을 보기 좋게 출력합니다.
