1. 최인훈, 「광장」

8. 박경리, 「시장과 전장」

1964 전쟁 장편·여성 서사 전쟁과 일상·여성 서사·생명 의식·역사 현실

도입·전개·절정·결말 줄거리

도입한국전쟁이라는 거대한 사건 속에서 시장의 삶과 전장의 폭력이 나란히 제시된다.
전개인물들은 피난, 생계, 이념, 가족 문제 속에서 흔들린다. 전쟁은 전선에만 있는 것이 아니라 일상과 시장에도 스며든다.
절정생존을 위한 현실과 이념적 폭력이 충돌하며 개인들의 삶이 무너진다.
결말작품은 전쟁이 인간의 일상, 가족, 윤리를 어떻게 훼손하는지를 폭넓게 보여 준다.

작가의 생애와 사상

박경리는 한국 현대문학의 대표적 장편소설 작가이다. 그는 여성의 삶, 가족사, 역사적 폭력, 생명 존중의 문제를 깊이 있게 다루었다. 「시장과 전장」은 전쟁을 이념의 사건이 아니라 일상과 생존의 문제로 확장해 바라본다.

작품 분석 모델: \[ A = T + C + I + S \] \(T\): 시대 변화, \(C\): 갈등 구조, \(I\): 인물의 내면, \(S\): 상징

파이썬 코드 예제

Python
worlds = {'시장': '생존', '전장': '폭력', '가족': '상처'}
for place, meaning in worlds.items():
    print(place, '→', meaning)
  1. 작품 제목의 두 공간을 딕셔너리로 해석합니다.
  2. 시장과 전장이 각각 무엇을 상징하는지 출력합니다.
  3. 전쟁이 일상으로 확장되는 구조를 이해합니다.
  4. 문학 작품의 주제·상징·갈등을 파이썬 자료구조로 바꾸는 연습입니다.

9. 방영웅, 「분례기」

1967 장편소설·농촌 현실 농촌 현실·여성 수난·생명력·토속성

도입·전개·절정·결말 줄거리

도입똥례는 가난하고 억압적인 농촌 현실 속에서 태어나 이름부터 멸시를 안고 살아간다.
전개가족의 무능, 남성 폭력, 빈곤 속에서 똥례는 고통을 겪지만 삶을 포기하지 않는다.
절정성폭력과 사회적 낙인, 결혼 문제를 거치며 똥례의 삶은 극한으로 몰린다.
결말작품은 비참한 현실 속에서도 살아남으려는 여성의 생명력과 농촌 사회의 어두운 구조를 보여 준다.

작가의 생애와 사상

방영웅은 농촌 하층민의 거친 생명력과 현실을 토속적 언어로 그린 작가이다. 「분례기」는 여성의 몸에 가해지는 폭력, 빈곤, 가족과 공동체의 억압을 통해 개발 이전 농촌 현실의 그늘을 보여 준다.

작품 분석 모델: \[ A = T + C + I + S \] \(T\): 시대 변화, \(C\): 갈등 구조, \(I\): 인물의 내면, \(S\): 상징

파이썬 코드 예제

Python
hardships = ['가난', '멸시', '폭력', '생존 의지']
for h in hardships:
    print('똥례의 삶:', h)
  1. 똥례가 겪는 고난과 생존 의지를 리스트로 저장합니다.
  2. 반복문으로 핵심 요소를 하나씩 출력합니다.
  3. 인물의 삶을 키워드 중심으로 정리합니다.
  4. 문학 작품의 주제·상징·갈등을 파이썬 자료구조로 바꾸는 연습입니다.

10. 이호철, 「닳아지는 살들」

1962 분단소설·실향민 문학 분단 상처·실향민·기다림·상실

도입·전개·절정·결말 줄거리

도입전쟁과 분단 이후 가족과 고향을 잃은 사람들의 삶이 제시된다.
전개인물들은 남쪽에서 살아가지만, 북쪽에 남은 가족과 과거의 기억 때문에 현재에 온전히 정착하지 못한다.
절정기다림과 상실감이 반복되며 분단이 개인의 시간과 몸을 계속 갉아먹는 현실이 드러난다.
결말작품은 해결되지 않는 분단의 고통을 ‘닳아지는 살’이라는 이미지로 남긴다.

작가의 생애와 사상

이호철은 실향민 체험과 분단 현실을 지속적으로 다룬 작가이다. 그의 문학은 전쟁이 끝난 뒤에도 계속되는 분단의 상처, 고향 상실, 가족 해체, 기다림의 고통을 사실적으로 보여 준다.

작품 분석 모델: \[ A = T + C + I + S \] \(T\): 시대 변화, \(C\): 갈등 구조, \(I\): 인물의 내면, \(S\): 상징

파이썬 코드 예제

Python
loss = {'고향': 10, '가족': 9, '시간': 8}
total = sum(loss.values())
print('상실의 합계:', total)
  1. 상실한 대상들을 딕셔너리로 저장합니다.
  2. values()로 점수만 꺼냅니다.
  3. sum()으로 분단이 만든 상실감을 합산합니다.
  4. 문학 작품의 주제·상징·갈등을 파이썬 자료구조로 바꾸는 연습입니다.

종합 파이썬 활동

아래 코드는 10개 작품을 연도순으로 정리하고, 핵심 주제를 출력하는 예제입니다.

Python
novels = [
    (1960, "광장", "분단과 자유"),
    (1960, "젊은 느티나무", "청춘과 금지된 사랑"),
    (1962, "꺼삐딴 리", "기회주의 풍자"),
    (1962, "닳아지는 살들", "분단과 실향"),
    (1964, "무진기행", "허무와 자기기만"),
    (1964, "시장과 전장", "전쟁과 일상"),
    (1965, "서울, 1964년 겨울", "도시 소외"),
    (1966, "병신과 머저리", "전쟁 상처와 예술"),
    (1966, "모래톱 이야기", "민중 저항"),
    (1967, "분례기", "농촌 현실과 생명력")
]

for year, title, theme in novels:
    print(f"{year}년 - {title}: {theme}")
  1. novels는 연도, 제목, 주제를 튜플로 저장합니다.
  2. for year, title, theme in novels는 각 튜플을 세 변수로 나누어 꺼냅니다.
  3. f-string으로 문학사 흐름을 보기 좋게 출력합니다.
