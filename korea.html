대한민국 시도·시군구 선택 Folium 지도

시도와 시군구를 드롭다운으로 선택하면 해당 지역 경계를 표시하고 지도가 자동 이동합니다.

지역 선택

행정구역 경계 GeoJSON을 불러오고 있습니다.

사용 방법
① 시도 선택 → ② 시군구 선택 → ③ 지도 보기. 마우스로 확대·축소, 드래그 이동이 가능합니다.

Folium에서 같은 구조로 생성하는 기본 코드 
import folium

m = folium.Map(
    location=[36.5, 127.8],
    zoom_start=7,
    tiles="OpenStreetMap"
)

# Folium은 Python에서 HTML/JavaScript 지도를 생성합니다.
# 시도·시군구 드롭다운처럼 브라우저 안에서 동작하는 기능은
# Folium HTML에 JavaScript를 추가하여 구현합니다.

m.save("korea_folium_dropdown_map.html")
