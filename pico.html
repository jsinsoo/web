4. 어두우면 LED 켜기

💻 제시 코드

import time

light = ADC(26)

예제 10Pico 센서

10. 초음파 거리 측정

수업자료 설명
초음파 거리 측정 예제는 Raspberry Pi Pico에서 MicroPython으로 센서값을 읽거나 LED 같은 출력 장치를 제어하는 실습입니다.

💻 제시 코드

from machine import Pin, time_pulse_us
import time

trig = Pin(3, Pin.OUT)
echo = Pin(2, Pin.IN)

while True:
    trig.low(); time.sleep_us(2)
    trig.high(); time.sleep_us(10)
    trig.low()
    d = time_pulse_us(echo, 1) * 0.0343 / 2
    print("거리: 계산값 cm", d)
    time.sleep(0.5)
