정리 1

확률의 기본 공리

① 정리 설명
확률은 사건이 일어날 가능성을 0과 1 사이의 수로 나타낸다.
② 핵심 수식
\[0\le P(A)\le 1,\qquad P(\Omega)=1\]
③ 증명 핵심 아이디어
  1. 확률은 음수가 될 수 없다.
  2. 전체 표본공간의 확률은 1이다.
  3. 서로 겹치지 않는 사건들은 확률을 더할 수 있다.
  4. 이 공리들이 확률론의 출발점이다.
④ 활용 예제
주사위, 동전, 보험, 통계 모델링에 활용된다.
⑤ 파이썬 코드
# 주사위에서 짝수가 나올 확률
sample_space = [1, 2, 3, 4, 5, 6]
event = [x for x in sample_space if x % 2 == 0]
probability = len(event) / len(sample_space)
print("짝수가 나올 확률:", probability)
⑥ 코드 상세 설명
  1. `sample_space`는 가능한 모든 결과이다.
  2. `event`는 짝수 사건이다.
  3. 사건의 개수를 전체 개수로 나누어 확률을 구한다.
정리 2

덧셈정리

① 정리 설명
두 사건 중 적어도 하나가 일어날 확률은 중복된 교집합을 한 번 빼서 구한다.
② 핵심 수식
\[P(A\cup B)=P(A)+P(B)-P(A\cap B)\]
③ 증명 핵심 아이디어
  1. \(P(A)+P(B)\)는 교집합을 두 번 센다.
  2. 합사건에서는 같은 결과를 한 번만 세어야 한다.
  3. 따라서 \(P(A\cap B)\)를 한 번 뺀다.
④ 활용 예제
설문조사, 카드 문제, 데이터 중복 제거에 활용된다.
⑤ 파이썬 코드
# 2의 배수 또는 3의 배수가 나올 확률
S = set(range(1, 7))
A = {x for x in S if x % 2 == 0}
B = {x for x in S if x % 3 == 0}
print(len(A | B) / len(S))
print(len(A)/len(S) + len(B)/len(S) - len(A & B)/len(S))
⑥ 코드 상세 설명
  1. `A | B`는 합사건이다.
  2. `A & B`는 교사건이다.
  3. 덧셈정리로 두 계산값이 같음을 확인한다.
정리 3

조건부확률

① 정리 설명
조건부확률은 사건 \(B\)가 일어났다는 조건 아래에서 \(A\)가 일어날 확률이다.
② 핵심 수식
\[P(A\mid B)=\frac{P(A\cap B)}{P(B)}\]
③ 증명 핵심 아이디어
  1. 조건 \(B\)가 주어지면 표본공간이 \(B\)로 줄어든다.
  2. 그 안에서 \(A\)도 함께 일어나는 부분은 \(A\cap B\)이다.
  3. 따라서 \(B\) 안에서 \(A\cap B\)가 차지하는 비율이다.
④ 활용 예제
진단검사, 추천 시스템, 날씨 예측에 활용된다.
⑤ 파이썬 코드
# 짝수라는 조건 아래 4 이상일 확률
A = {4, 5, 6}
B = {2, 4, 6}
conditional = len(A & B) / len(B)
print("P(A|B):", conditional)
⑥ 코드 상세 설명
  1. `A`는 4 이상 사건이다.
  2. `B`는 짝수 사건이다.
  3. `A & B`는 두 사건이 동시에 일어나는 경우이다.
정리 4

곱셈정리

① 정리 설명
두 사건이 동시에 일어날 확률은 조건부확률과 조건 사건의 확률의 곱으로 구한다.
② 핵심 수식
\[P(A\cap B)=P(A\mid B)P(B)\]
③ 증명 핵심 아이디어
  1. 조건부확률 정의에서 시작한다.
  2. \(P(A\mid B)=P(A\cap B)/P(B)\)이다.
  3. 양변에 \(P(B)\)를 곱하면 곱셈정리가 된다.
④ 활용 예제
비복원 추출, 연속 사건 확률, 의사결정나무에 활용된다.
⑤ 파이썬 코드
# 카드 두 장이 모두 에이스일 확률
p_first = 4 / 52
p_second_given_first = 3 / 51
probability = p_first * p_second_given_first
print(probability)
⑥ 코드 상세 설명
  1. 첫 번째 에이스 확률은 \(4/52\)이다.
  2. 첫 번째가 에이스였으면 남은 에이스는 3장이다.
  3. 곱셈정리로 두 확률을 곱한다.
정리 5

독립사건 정리

① 정리 설명
두 사건이 독립이면 한 사건의 발생이 다른 사건의 확률을 바꾸지 않는다.
② 핵심 수식
\[P(A\cap B)=P(A)P(B)\]
③ 증명 핵심 아이디어
  1. 독립이면 \(P(A\mid B)=P(A)\)이다.
  2. 곱셈정리 \(P(A\cap B)=P(A\mid B)P(B)\)를 사용한다.
  3. 대입하면 \(P(A\cap B)=P(A)P(B)\)가 된다.
④ 활용 예제
동전 반복 시행, 확률모형 단순화, 독립 가정 검토에 활용된다.
⑤ 파이썬 코드
# 동전 두 번 던지기
outcomes = [(a,b) for a in "HT" for b in "HT"]
A = [o for o in outcomes if o[0] == "H"]
B = [o for o in outcomes if o[1] == "H"]
AB = [o for o in outcomes if o[0] == "H" and o[1] == "H"]
print(len(AB)/4, (len(A)/4)*(len(B)/4))
⑥ 코드 상세 설명
  1. 첫 번째 앞면과 두 번째 앞면 사건을 만든다.
  2. 두 사건의 교집합 확률을 구한다.
  3. 각 확률의 곱과 같으면 독립이다.
정리 6

전확률정리

① 정리 설명
전체를 나누는 여러 경우가 있을 때, 각 경우의 조건부확률을 가중합하여 전체 확률을 구한다.
② 핵심 수식
\[P(A)=\sum_{i=1}^{n}P(A\mid B_i)P(B_i)\]
③ 증명 핵심 아이디어
  1. \(B_i\)들이 표본공간을 분할한다고 하자.
  2. \(A\)는 \(A\cap B_i\)들의 합으로 나뉜다.
  3. 각 조각의 확률을 더한다.
  4. 곱셈정리를 적용해 식을 얻는다.
④ 활용 예제
공장별 불량률, 고객군별 구매확률, 질병 발생률 계산에 활용된다.
⑤ 파이썬 코드
# 공장별 불량률로 전체 불량률 계산
p_factory = [0.7, 0.3]
p_defect = [0.02, 0.05]
total = sum(p_factory[i] * p_defect[i] for i in range(2))
print("전체 불량률:", total)
⑥ 코드 상세 설명
  1. `p_factory`는 공장별 생산 비율이다.
  2. `p_defect`는 공장별 불량률이다.
  3. 각 비율과 조건부확률을 곱해 더한다.
정리 7

베이즈 정리

① 정리 설명
관측된 증거를 바탕으로 원인의 확률을 갱신하는 정리이다.
② 핵심 수식
\[P(B_i\mid A)=\frac{P(A\mid B_i)P(B_i)}{\sum_{j=1}^{n}P(A\mid B_j)P(B_j)}\]
③ 증명 핵심 아이디어
  1. 곱셈정리로 \(P(A\cap B_i)=P(A\mid B_i)P(B_i)\)를 얻는다.
  2. 조건부확률 정의에 의해 \(P(B_i\mid A)=P(A\cap B_i)/P(A)\)이다.
  3. 분모 \(P(A)\)는 전확률정리로 계산한다.
④ 활용 예제
의학 진단, 스팸 필터, 인공지능, 추천 시스템에 활용된다.
⑤ 파이썬 코드
# 양성일 때 실제 질병일 확률
p_disease = 0.01
p_pos_given_disease = 0.99
p_pos_given_no = 0.05
p_no = 1 - p_disease
p_positive = p_pos_given_disease*p_disease + p_pos_given_no*p_no
posterior = p_pos_given_disease*p_disease / p_positive
print(posterior)
⑥ 코드 상세 설명
  1. `p_disease`는 사전확률이다.
  2. `p_positive`는 전체 양성 확률이다.
  3. `posterior`는 양성 후 갱신된 질병 확률이다.
정리 8

기댓값의 선형성

① 정리 설명
기댓값은 확률변수의 평균적인 값이며, 독립이 아니어도 선형성이 성립한다.
② 핵심 수식
\[E(aX+bY)=aE(X)+bE(Y)\]
③ 증명 핵심 아이디어
  1. 기댓값은 값과 확률의 곱을 모두 더한 것이다.
  2. 합의 기댓값은 각 기댓값의 합으로 분리된다.
  3. 상수배는 기댓값 밖으로 나올 수 있다.
  4. 독립성은 필요하지 않다.
④ 활용 예제
평균 점수, 기대수익, 보험료 산정, 통계 추정에 활용된다.
⑤ 파이썬 코드
# 주사위 눈의 기댓값
values = [1, 2, 3, 4, 5, 6]
p = 1/6
expected = sum(x*p for x in values)
print("기댓값:", expected)
⑥ 코드 상세 설명
  1. 각 눈의 값에 확률 \(1/6\)을 곱한다.
  2. 모든 값을 더하면 기댓값 \(3.5\)가 된다.
정리 9

분산 공식

① 정리 설명
분산은 확률변수가 평균에서 얼마나 퍼져 있는지를 나타낸다.
② 핵심 수식
\[\operatorname{Var}(X)=E(X^2)-\{E(X)\}^2\]
③ 증명 핵심 아이디어
  1. 분산의 정의는 \(E((X-\mu)^2)\)이다.
  2. 제곱을 전개한다.
  3. 기댓값의 선형성을 적용한다.
  4. \(\mu=E(X)\)를 대입하면 공식이 나온다.
④ 활용 예제
데이터 변동성, 시험 점수 분포, 투자 리스크 계산에 활용된다.
⑤ 파이썬 코드
# 주사위 눈의 분산
values = [1, 2, 3, 4, 5, 6]
p = 1/6
EX = sum(x*p for x in values)
EX2 = sum((x**2)*p for x in values)
variance = EX2 - EX**2
print("분산:", variance)
⑥ 코드 상세 설명
  1. `EX`는 \(E(X)\)이다.
  2. `EX2`는 \(E(X^2)\)이다.
  3. 공식으로 분산을 계산한다.
정리 10

큰수의 법칙과 중심극한정리

① 정리 설명
시행을 많이 반복하면 표본평균은 기댓값에 가까워지고, 표준화된 표본평균은 정규분포에 가까워진다.
② 핵심 수식
\[\overline{X}_n\to E(X),\qquad \frac{\overline{X}_n-\mu}{\sigma/\sqrt n}\xrightarrow[]{d}N(0,1)\]
③ 증명 핵심 아이디어
  1. 큰수의 법칙은 표본평균이 이론적 평균에 가까워짐을 말한다.
  2. 중심극한정리는 표본평균의 분포가 정규분포에 가까워짐을 말한다.
  3. 개별 분포가 정규분포가 아니어도 표본 수가 크면 정규근사가 가능하다.
  4. 통계적 추론의 핵심 기반이다.
④ 활용 예제
여론조사, 신뢰구간, 가설검정, 몬테카를로 시뮬레이션에 활용된다.
⑤ 파이썬 코드
# 표본평균 시뮬레이션
import random

means = []
for _ in range(5000):
    sample = [random.random() for _ in range(30)]
    means.append(sum(sample) / 30)

print("표본평균들의 평균:", sum(means) / len(means))
print("처음 10개:", means[:10])
⑥ 코드 상세 설명
  1. 균등분포 난수 30개의 평균을 여러 번 구한다.
  2. 표본평균들이 특정 중심 근처에 모인다.
  3. 시행을 많이 하면 중심극한정리의 직관을 확인할 수 있다.

마무리 핵심 정리

확률론은 불확실성을 수학적으로 다루는 언어이다. 조건부확률과 베이즈 정리는 정보를 얻었을 때 믿음을 갱신하는 방법을 제공하고, 기댓값과 분산은 확률변수의 중심과 퍼짐을 설명한다. 큰수의 법칙과 중심극한정리는 통계적 추론과 데이터 과학의 핵심 기반이다.
