1. 이상, 「날개」

1936모더니즘자의식·분열

줄거리

주인공 ‘나’는 아내에게 경제적으로 의존하며 방 안에 갇힌 듯 살아간다. 그는 잠, 돈, 아내의 방, 외출의 기억을 통해 자신이 사회와 단절된 존재임을 자각한다. 마지막의 “날개” 이미지는 추락한 자아가 다시 비상하고 싶어 하는 욕망을 상징한다.

작가와 사상

이상은 1930년대 한국 모더니즘을 대표하는 작가다. 그의 문학은 현실을 직접 고발하기보다 분열된 의식, 도시적 불안, 무의식, 초현실적 이미지로 식민지 지식인의 내면을 드러낸다.

문학적 긴장도 모델: \[T = A + I + D\] \(A\): 불안, \(I\): 아이러니, \(D\): 자아분열

파이썬 코드: 상징어 빈도 분석

Python
text = "날개 돈 아내 방 잠 외출 날개 자아 방"
words = text.split()
count = {}
for w in words:
    count[w] = count.get(w, 0) + 1
print(count)
  1. text: 작품의 핵심 상징어를 문자열로 저장합니다.
  2. split(): 띄어쓰기를 기준으로 단어를 나눕니다.
  3. count = {}: 단어 빈도를 저장할 딕셔너리입니다.
  4. get(w,0)+1: 처음 나온 단어는 1, 이미 나온 단어는 1 증가시킵니다.

2. 이효석, 「메밀꽃 필 무렵」

1936향토적 서정인연

줄거리

장돌뱅이 허생원은 봉평 장터에서 젊은 동이를 만나 대화 장으로 가는 밤길을 함께 걷는다. 허생원은 오래전 성서방네 처녀와의 인연을 회상하고, 동이가 자신의 아들일 가능성이 은근히 암시된다. 메밀꽃 핀 밤길은 추억과 운명을 감싸는 서정적 배경이 된다.

작가와 사상

이효석은 초기에는 사회적 관심을 보였으나 이후 순수문학과 서정적 문체를 추구했다. 그의 대표적 특징은 향토성, 자연미, 인간 본능과 인연의 서정화이다.

서정성 지수: \[L = N + M + R\] \(N\): 자연 이미지, \(M\): 기억, \(R\): 관계

파이썬 코드: 등장인물 관계표

Python
people = ["허생원", "동이", "조선달", "성서방네 처녀"]
relation = {
    "허생원-동이": "부자 관계 가능성 암시",
    "허생원-성서방네 처녀": "과거의 인연",
}
for k, v in relation.items():
    print(k, ":", v)
  1. people: 주요 인물을 리스트로 정리합니다.
  2. relation: 인물 사이의 관계를 딕셔너리로 저장합니다.
  3. items(): 관계 이름과 설명을 동시에 꺼냅니다.
  4. print(): 관계망을 보기 쉽게 출력합니다.

3. 김유정, 「동백꽃」

1936농촌소설해학

줄거리

순박한 소년 ‘나’는 점순이가 자신에게 관심을 표현한다는 사실을 알아차리지 못한다. 점순이는 닭싸움과 짓궂은 행동으로 마음을 드러내고, 두 사람은 동백꽃 속에서 풋풋하고 해학적인 감정을 경험한다.

작가와 사상

김유정은 농촌 하층민의 삶을 비극적으로만 그리지 않고, 토속어와 웃음으로 인간의 생명력을 보여 준다. 그의 문학은 해학 속의 현실 인식, 토속성, 순박한 욕망이 핵심이다.

해학 효과: \[H = C - K\] \(C\): 독자가 아는 맥락, \(K\): 인물이 모르는 지식

파이썬 코드: 감정 변화 리스트

Python
events = ["감자 거절", "닭싸움", "동백꽃 장면"]
feelings = ["무심함", "당황", "풋사랑"]
for e, f in zip(events, feelings):
    print(f"{e} → {f}")
  1. events: 사건 순서를 저장합니다.
  2. feelings: 사건마다 달라지는 감정을 저장합니다.
  3. zip(): 사건과 감정을 한 쌍으로 묶습니다.
  4. f-string: 결과를 예쁜 화살표 형식으로 출력합니다.

4. 김유정, 「봄봄」

1935해학농촌 현실

줄거리

‘나’는 점순이와 혼인시켜 주겠다는 장인의 말을 믿고 머슴처럼 일한다. 그러나 장인은 점순이가 아직 키가 작다는 핑계로 혼인을 계속 미룬다. 순진한 ‘나’와 교활한 장인의 대립은 웃음을 만들지만, 그 속에는 농촌 노동 착취의 현실이 숨어 있다.

작가와 사상

김유정은 억압적 현실을 설교하지 않고 웃음으로 폭로한다. 「봄봄」은 해학적 인물, 반복되는 기다림, 착취 구조를 통해 농촌 사회의 모순을 보여 준다.

갈등 강도: \[G = W \times P\] \(W\): 기다린 시간, \(P\): 약속 불이행 횟수

파이썬 코드: 약속 지연 계산

Python
months_worked = 36
promises = 5
delay_score = months_worked * promises
print("갈등 점수:", delay_score)
  1. months_worked: 일한 기간을 월 단위로 둡니다.
  2. promises: 장인이 미룬 약속 횟수입니다.
  3. *: 두 값을 곱해 갈등 정도를 단순 모델링합니다.
  4. print(): 계산 결과를 출력합니다.

5. 박태원, 「소설가 구보 씨의 일일」

1934도시 모더니즘산책자

줄거리

구보는 하루 동안 경성 거리를 걸으며 사람, 전차, 카페, 거리의 풍경을 관찰한다. 뚜렷한 사건보다 도시를 배회하는 지식인의 의식과 시선이 작품의 중심이 된다.

작가와 사상

박태원은 1930년대 도시 모더니즘의 대표 작가로, 사건 중심 서사보다 관찰과 문체 실험을 중시했다. 그의 사상은 고현학, 도시 관찰, 예술가의 자의식으로 요약된다.

도시 관찰 모델: \[O = P + S + T\] \(P\): 사람, \(S\): 공간, \(T\): 생각

파이썬 코드: 산책 경로 출력

Python
route = ["집", "거리", "전차", "다방", "친구"]
for i, place in enumerate(route, 1):
    print(i, place)
  1. route: 구보의 하루 이동 경로를 리스트로 저장합니다.
  2. enumerate(route,1): 1번부터 번호를 붙입니다.
  3. for: 경로를 순서대로 반복합니다.
  4. print(): 번호와 장소를 함께 출력합니다.

6. 채만식, 「태평천하」

1938풍자소설가족사

줄거리

윤 직원 영감은 일제강점기의 현실을 자기에게 유리한 ‘태평천하’로 착각한다. 그의 탐욕과 시대 인식의 왜곡은 가족 갈등과 함께 드러난다. 작품 제목 자체가 반어로 작동하며, 몰락하는 지주 계층의 허위의식을 풍자한다.

작가와 사상

채만식은 식민지 현실을 풍자와 아이러니로 비판한 작가다. 「태평천하」는 반어, 세대 갈등, 계급 비판, 식민지 현실 인식이 선명하다.

풍자 강도: \[S = I + C\] \(I\): 반어, \(C\): 비판 의식

파이썬 코드: 반어 문장 판별 예시

Python
title = "태평천하"
reality = "식민지 현실과 가족 몰락"
if title != reality:
    print("제목과 현실의 충돌 → 반어 효과")
  1. title: 작품 제목을 저장합니다.
  2. reality: 실제 작품 현실을 저장합니다.
  3. !=: 두 내용이 다른지 비교합니다.
  4. if: 다르면 반어 효과를 출력합니다.

7. 채만식, 「탁류」

1937~1938세태소설비판적 사실주의

줄거리

군산을 배경으로 초봉과 그 주변 인물들이 돈, 욕망, 가부장적 질서, 식민지 자본의 흐름 속에서 고통받는다. 제목의 ‘탁류’는 혼탁한 강물처럼 개인을 휩쓸어 가는 시대 현실을 상징한다.

작가와 사상

채만식은 현실을 단순히 기록하지 않고 구조적 모순을 드러냈다. 「탁류」는 돈의 폭력, 여성 수난, 식민지 도시의 혼탁함을 통해 1930년대 사회를 비판한다.

사회 압력: \[P = M + V + C\] \(M\): 돈, \(V\): 폭력, \(C\): 관습

파이썬 코드: 사회 압력 점수

Python
pressures = {"돈": 4, "폭력": 5, "관습": 3}
total = sum(pressures.values())
print("사회 압력 합계:", total)
  1. pressures: 압력 요소와 강도를 딕셔너리로 둡니다.
  2. values(): 숫자 값만 꺼냅니다.
  3. sum(): 모든 압력 값을 더합니다.
  4. print(): 작품의 사회적 긴장을 수치화해 봅니다.

8. 주요섭, 「사랑손님과 어머니」

1935서정소설동심 시점

줄거리

어린 옥희의 시선을 통해 어머니와 사랑손님 사이의 조심스러운 감정이 드러난다. 어른들의 사랑은 직접 말해지지 않고, 아이의 순수한 관찰을 통해 은근하게 표현된다. 결국 두 사람의 사랑은 사회적 윤리와 관습 앞에서 이루어지지 못한다.

작가와 사상

주요섭은 사회비판적 문제의식도 지녔지만, 이 작품에서는 절제된 서정과 심리 묘사가 돋보인다. 핵심은 동심 시점, 절제미, 여성의 감정과 사회 규범의 충돌이다.

서술 거리: \[D = A - C\] \(A\): 어른의 실제 감정, \(C\): 아이가 이해한 감정

파이썬 코드: 시점 차이 표현

Python
adult_meaning = "사랑"
child_view = "친절한 손님"
print("어른의 의미:", adult_meaning)
print("아이의 이해:", child_view)
  1. adult_meaning: 어른 독자가 파악하는 감정입니다.
  2. child_view: 옥희가 이해하는 수준입니다.
  3. 두 출력의 차이가 작품의 서정성과 여백을 만듭니다.

9. 김동인, 「광염소나타」

1930유미주의액자소설

줄거리

천재 음악가 백성수는 강렬한 예술적 영감을 얻기 위해 점점 더 위험하고 비윤리적인 행위를 저지른다. 작품은 예술과 도덕의 관계를 극단적인 방식으로 질문한다.

작가와 사상

김동인은 예술의 독자성과 미적 가치를 강조한 작가로 평가된다. 「광염소나타」는 예술지상주의, 유미주의, 천재와 윤리의 충돌을 보여 준다.

예술-윤리 충돌: \[Q = E - R\] \(E\): 예술적 열정, \(R\): 윤리적 책임

파이썬 코드: 가치 충돌 모델

Python
art = 10
ethics = 9
conflict = abs(art - ethics)
print("가치 충돌 차이:", conflict)
  1. art: 예술적 열정의 크기입니다.
  2. ethics: 윤리적 책임의 크기입니다.
  3. abs(): 두 가치의 차이를 양수로 구합니다.
  4. 차이가 클수록 갈등이 커진다고 해석할 수 있습니다.

10. 이태준, 「복덕방」

1930년대단편소설몰락과 근대

줄거리

복덕방을 중심으로 노년 인물들의 삶과 욕망, 몰락감이 드러난다. 근대 도시의 변화 속에서 전통적 인간관계와 생계 방식은 흔들리고, 인물들은 작은 희망과 좌절 사이를 오간다.

작가와 사상

이태준은 문장미와 인물 심리 묘사에 뛰어난 작가로, 단편 형식의 완성도를 높인 인물로 평가된다. 그의 문학은 인간에 대한 연민, 세련된 문체, 근대의 그늘을 중시한다.

몰락감: \[F = A + L\] \(A\): 나이 듦, \(L\): 생활 기반 상실

파이썬 코드: 인물 상태 정리

Python
characters = [
    {"name": "노인 A", "hope": "돈", "mood": "불안"},
    {"name": "노인 B", "hope": "체면", "mood": "쓸쓸함"},
]
for c in characters:
    print(c["name"], c["hope"], c["mood"])
  1. characters: 인물 정보를 딕셔너리 리스트로 저장합니다.
  2. name, hope, mood: 이름·욕망·정서를 나타냅니다.
  3. for c in characters: 인물을 하나씩 꺼냅니다.
  4. c["name"]: 딕셔너리에서 필요한 값을 가져옵니다.

비교 정리표

모더니즘 계열: 「날개」, 「소설가 구보 씨의 일일」 — 내면, 도시, 의식의 흐름.

향토·해학 계열: 「메밀꽃 필 무렵」, 「동백꽃」, 「봄봄」 — 농촌, 자연, 인연, 웃음.

풍자·비판적 사실주의 계열: 「태평천하」, 「탁류」 — 식민지 현실, 돈, 계급, 가족 몰락.

서정·심리 계열: 「사랑손님과 어머니」, 「복덕방」 — 절제된 감정, 연민, 사회 규범.

유미주의 계열: 「광염소나타」 — 예술과 윤리의 극단적 충돌.
